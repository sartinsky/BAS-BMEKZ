Шкала ставок налогов

В регистре хранится информация о ставках налогов. Для каждого из налогов может быть предусмотрена либо одна ставка, либо шкала ставок.

При заполнении шкалы следует для одного и того же измерения Налог заполнить несколько порогов шкалы, то есть указать Номер порога и его значение (ресурс Порог). Номера порогов должны указываться последовательно ("Порог 1", "Порог 2", ... и т.д.) в порядке возрастания их значений. Ресурс Ставка должен быть заполнен ставкой налога, применяемой, если база налогообложения не превышает указанный порог.

При заполнении информации о налоге, у которого нет шкалы, а только одна ставка, следует в качестве номера порога установить "Порог 1", а в качестве порога установить любое заведомо большое число, например, 9 999 999 999,99.

Как правило, ставки одного и того же налога одинаковые для всех предприятий и организаций. Такие ставки налогов следует вносить в регистр, оставляя пустым значение измерения Организация. Если же для какого-то налога ставка может изменяться от одной организации к другой, то при заполнении регистра следует заполнить измерение Организация и указать ставку налога для этой конкретной организации.

Реквизит Статья налоговой декларации используется в том случае, если в регламентированной отчетности по этому налогу требуется выделять начисления по разным ставкам в отдельных статьях соответствующих налоговых деклараций.
