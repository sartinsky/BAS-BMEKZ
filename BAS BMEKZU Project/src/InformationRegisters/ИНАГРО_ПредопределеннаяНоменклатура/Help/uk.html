Регістр відомостей "Визначена ​​номенклатура"

Призначений для вказівки списку зумовлених номенклатур по якому будуть автоматично розраховуватися послуги.

Слід зазначити, що всі перераховані тут послуги можуть нараховуватися автоматично. У разі, якщо для якоїсь послуги не виставлено відповідність у вигляді вказівки найменування послуги, то така послуга не нараховується.
