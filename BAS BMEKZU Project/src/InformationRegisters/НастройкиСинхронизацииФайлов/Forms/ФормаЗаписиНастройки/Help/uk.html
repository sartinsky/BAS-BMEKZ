Призначена для настройки синхронізації файлів для кожного власника (документа, папки та інших об'єктів програми).

Відкривається зі списку Налаштування синхронізації файлів за відповідним посиланням.

  • По умовчанню увімкнено прапорець Синхронізувати файли з хмарним сервісом.
  • У полі нижче виберіть зі списку обліковий запис хмарного сервісу, з яким буде проводитися синхронізація файлів. Якщо в списку немає потрібного облікового запису, можна ввести новий.

Що синхронізувати

  • У розділі Що синхронізувати можна вибрати одне зі значень:
    • Всі файли, приєднані до (можна вибрати об'єкт програми) - в цьому випадку умови відбору не потрібні;
    • Файли (вказується обраний список) за заданими правилами синхронізації - вибрано по умовчанню:
      • заповнювати умови відбору файлів не потрібно, якщо необхідно синхронізувати всі файли.
      • При необхідності введіть новий відбір: натисніть кнопку Додати, виберіть Додати новий елемент. Також можна використовувати клавішу Ins.
        • У колонці Поле зі списку полів виберіть потрібне.
        • Виберіть Вид порівняння.
        • Виберіть або напишіть Значення.
  • Напишіть Короткий опис правила.

Умова за датою

  • Для швидкої настройки відборів передбачено стандартну умову з використанням будь-якої дати, пов'язаної з об'єктом-власником файлу.
  • Натисніть кнопку Додати, виберіть Додати умову за датою.
  • Проставте кількість днів щодо потрібної дати, після яких файли будуть очищатися.
  • Виберіть потрібну Дату зі списку (одну з дат, зазначену в об'єкті-власнику файлів), наприклад, дата відправки документа і т.і.

Групування відборів

  • Виділіть кілька відборів, натисніть Групувати умови. Виберіть умову групування - І, Або, Не.
  • По умовчанню кілька відборів групуються за І.
