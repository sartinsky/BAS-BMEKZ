Регістр відомостей "Способи відображення витрат по амортизації основних засобів (бухгалтерський облік)"

У регістрі зберігаються дані про відповідність способів відображення витрат по амортизації основним засобам організацій для бухгалтерського обліку.

Дані з регістру використовуються при розрахунку амортизації.

Регістр періодичний, змінюється документами:

  • "Введення початкових залишків ОЗ";
  • "Зміна способів відображення витрат по амортизації ОС";
  • "Введення в эксплуатцию ОС".
