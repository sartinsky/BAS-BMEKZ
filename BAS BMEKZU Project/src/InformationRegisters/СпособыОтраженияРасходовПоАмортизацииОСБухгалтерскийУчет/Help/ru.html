Регистр сведений "Способы отражения расходов по амортизации основных средств (бухгалтерский учет)"

В регистре хранятся данные о соответствии способов отражения расходов по амортизации основным средствам организаций для бухгалтерского учета.

Данные из регистра используются при расчете амортизации.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение способов отражения расходов по амортизации ОС";
  • "Ввод в эксплуатцию ОС".
