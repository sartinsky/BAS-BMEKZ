Регистр сведений "Первоначальные сведения о НМА (налоговый учет)"

Содержит первоначальные сведения о нематериальных активах организаций  для целей налогового учета:

  • Признак начисления амортизации;
  • Метод начисления амортизации;
  • Первоначальная стоимость (для целей налогового учета);
  • Срок полезного использования;
  • Учитывается как ОС (НУ);
  • Налоговая группа ОС (для учитывающихся как ОС );
  • Налоговое назначение. 

Данные из регистра используются для расчета амортизации объектов НМА и НИОКР. 

По данным регистра определяется организация, в которой учитывается объект, в определенный момент времени.

Регистр периодический, изменяется документом:

  • "Принятие к учету НМА".
