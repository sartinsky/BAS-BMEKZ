Регистр сведений "Дополнительные реквизиты номенклатуры"

Регистр служит для хранения дополнительных настроек справочника "Номенклатура":

  • содержит ссылку на элемент справочника "Вид культуры";
  • указывается, к какому виду ТМЦ принадлежит номенклатура (Перечисленные виды тмц: культура, побочный продукт, отходы I кат., отходы II кат., отходы III кат., тара, сырье, продукция, побочная продукция, прочие);
  • содержит ссылку на элемент справочника "Класификатор зерна" для правильного отображения номеклатуры при выгрузке складских квитанций в Гос реестратор;
  • признак является ли услуга производственной (Производственная услуга);
  • признак вести контроль по начислению, выставлению оплаты за оказаные предприятием услуги (Вести расчеты по услуге).
