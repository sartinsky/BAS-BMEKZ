Регістр відомостей "Додаткові реквізити номенклатури"

Регістр слугує для зберігання додаткових налаштувань довідника "Номенклатура":

  • містить посилання на елемент довідника "Вид культури";
  • зазначається, якому виду ТМЦ належить номенклатура (Перелічені види тмц: культура, побічний продукт, відходи I кат., відходи II кат., відходи III кат., тара, сировина, продукція, побічна продукція, інші);
  • містить посилання на елемент довідника "Класифікатор зерна" для вірного відображення номенклатури під час вивантаження складських квитанцій в Держреєстратор;
  • ознака чи є послуга виробничою (Виробнича послуга);
  • ознака вести контроль по нарахуванню оплати за надані підприємством послуги (Вести розрахунки по послузі).
