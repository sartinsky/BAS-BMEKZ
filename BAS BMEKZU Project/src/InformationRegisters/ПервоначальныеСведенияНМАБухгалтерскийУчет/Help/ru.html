Регистр сведений "Первоначальные сведения о НМА(бухгалтерский учет)"

Содержит первоначальные сведения о нематериальных активах  организаций  для целей бухгалтерского учета:

  • Признак начисления амортизации (погашения стоимости);
  • Способ начисления амортизации;
  • Первоначальная стоимость;
  • Объем продукции (работ) для начисления амортизации при использовании способа начисления "Пропорционально объему продукции (работ)";
  • Срок полезного использования;
  • Ликвидационная стоимость.

Данные из регистра используются для расчета амортизации объектов НМА. 

По данным регистра определяется организация, в которой учитывается объект, в определенный момент времени.

Регистр периодический, изменяется документом:

  • "Принятие к учету НМА".
