Регистр сведений "Первоначальные сведения ОС (налоговый учет)"

Содержит информацию о первоначальных сведениях об основных средствах организаций для налогового учета:

  • Метод начисления амортизации;
  • Первоначальная стоимость;
  • Налоговая группа ОС;
  • Учитывается как запас НУ.

Данные из регистра используются для расчета амортизации основных средств. 

По данным регистра определяется организация, в которой учитывается основное средство, в определенный момент времени.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Ввод в эксплуатацию ОС".
