Регистр сведений "События основных средств организации (регл. учет)"

Содержит информацию о событиях основных средств организаций  для бухгалтерского и налогового учета:

  • Название документа события;
  • Номер документа;
  • Сумма затрат по бухгалтерскому учету.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение графиков амортизации ОС";
  • "Изменение параметров начисления амортизации ОС";
  • "Изменение состояния ОС";
  • "Изменение способов отражения расходов по амортизации ОС";
  • "Модернизация ОС";
  • "Передача ОС";
  • "Перемещение ОС";
  • "Подготовка к передаче ОС";
  • "Ввод в эксплуатацию ОС";
  • "Списание ОС".
