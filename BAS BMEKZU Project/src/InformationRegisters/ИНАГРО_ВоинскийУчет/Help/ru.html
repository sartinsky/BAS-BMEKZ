Регистр сведений «Воинский учет»

Регистр предназначен для хранения данных о воинском учете физических лиц.

В регистр заносятся следующие реквизиты:

1.      Период.

2.      Физ. лицо.

3.      Отношение к воинской обязанности (военнообязанный, не военнообязанный, призывник), к воинскому учету (состоит на воинском учете, встает на воинский учет, не состоит на воинском учете, снят с воинского учета по возрасту, снят с воинского учета по состоянию здоровья).

4.      Звание (звание, группа, состав, категория).

5.      Данные по военкомату.

6.      Специальность.

7.      Спец. учет.
