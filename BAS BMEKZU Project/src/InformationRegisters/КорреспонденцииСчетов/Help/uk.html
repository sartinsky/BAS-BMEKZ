Довідник "Кореспонденції рахунків"

 

Довідник кореспонденцій рахунків бухгалтерського обліку представляє собою навігатор по конфігурації, який можна використовувати як при початковому освоєнні програми, так і в поточній роботі при реєстрації господарських операцій. Довідник кореспонденцій допоможе бухгалтеру за змістом господарської операції або по кореспонденції рахунків бухгалтерського обліку за дебетом і (або) кредиту рахунку зрозуміти, яким документом необхідно відобразити операцію в конфігурації.

Для введення господарської операції необхідно вибрати кореспонденцію рахунків. На підставі обраної кореспонденції рахунків можна ввести документ конфігурації одним з наступних способів:

  • по клавіші Enter
  • по кнопці Ввести операцію,
  • за гіперпосиланням документа обраної кореспонденції рахунків.

Відібрати потрібну кореспонденцію рахунків можна за рахунками дебету або кредиту, за змістом операції або по документу конфігурації. Для цього передбачені поля швидкого відбору:

  • Рахунок Дт - за рахунком (субрахунком) дебету кореспонденції рахунків,
  • Рахунок Кт - за рахунком (субрахунком) кредиту кореспонденції рахунків,
  • Зміст операції - за змістом операції, окремих слів або частин слів. У випадаючому списку можна подивитися приклад відбору за змістом операції,
  • Документ - по документу конфігурації.

Можна використовувати відбір відразу по декількох полях швидкого відбору.

Використовувані кореспонденції рахунків відображаються на окремій закладці Останні використані.
