"Экспорт данных"

Обработка предназначена для выгрузки данных в другие учетные системы. На данный момент доступна функция выгрузки информации об использовании бланков складских квитанций в специализированную систему "Держреестр".

Общие реквизиты

  • Каталог выгрузки данных - путь к папке, в которую будет проводиться выгрузка информации в виде файлов;
  • Укажите период анализа "с", "по" - период, за который будет проводиться выгрузка данных;
  • Показать процесс - включает дополнительную индикацию процесса подготовки и выгрузки данных обработкой;

Выгрузка данных запускается непосредственно нажатием кнопки "В государственный реестр".
