"Помічник Ж/Д  вагаря"

 

Обробка «Помічник Ж/Д  вагаря» призначена для автоматизації щоденної роботи вагаря. На формі розміщені списки введених вагарем ТТН, по яким не закінчене заповнення. Іншими словами списки містять перелік автомашин, що знаходяться на території елеватора. 

Загальні реквізити

Форма містить наступні реквізити:

·           «Вагар» - найменування поточного користувача (вагаря);

·           «Період з», «по» - інтерфейс введення періоду, за який аналізуються товарно-транспортні накладні;

·           «Показати ТТН по всім вагарям» - по замовчуванню форма «показує» список накладних, введених у програму поточним вагарем. По бажанню та при наявності певних повноважень можна продивитися накладні, введені іншими користувачами;

·           «Показати ТТН з фасуванням / декілька ТМЦ» - по технології проходження фасованих вантажів через ваги їх не обов'язково зважувати (маркування та вага вказуються на етикетках до мішків). При необхідності такі машини теж можна відобразити в переліку накладних;

·           Кнопка «Оновити» - виконує повторне заповнення списків ТТН (перезаповнення форми);

·           Кнопка «Нова машина» - призводить до виводу на екран відповідної форми нової накладної;

·            «Закрити» - закриває поточну форму.
