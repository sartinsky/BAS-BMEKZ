"Помощник Ж/Д  весовщика"

Обработка «Помощник Ж/Д  весовщика» предназначена для автоматизации каждодневной работы весовщика. На форме расположены списки введенных весовщиком ТТН, по которым не закончено заполнение. Другими словами списки содержат перечень автомашин, находящихся на территории элеватора. 

Общие реквизиты

Форма содержит следующие реквизиты:

·           «Весовщик» - наименование текущего пользователя (весовщика);

·           «Период с», «по» - интерфейс ввода периода, за который анализируются товарно-транспортные накладные;

·           «Показать ТТН по всем весовщикам» - по умолчанию форма «показывает» список накладных, введенных в программу текущим весовщиком. По желанию и при наличии определенных полномочий можно просмотреть накладные, введенными другими пользователями;

·           «Показывать накладные с фасовкой/несколько ТМЦ» - по технологии прохождения фасованных грузов через весы их не обязательно взвешивать (маркировка и вес указываются в этикетках на мешках). При необходимости такие машины тоже можно отразить в списке накладных;

·           Кнопка «Обновить» - выполняет повторное заполнение списков ТТН (перезаполнение формы);

·           Кнопка «Новая машина» - приводит к выводу на экран соответствующей формы новой накладной;

·            «Закрыть» - закрывает текущую форму.
