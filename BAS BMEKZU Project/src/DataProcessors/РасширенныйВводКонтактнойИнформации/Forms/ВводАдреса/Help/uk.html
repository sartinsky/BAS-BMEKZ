Призначена для введення адреси за шаблоном. При заповненні можливе використання адресного класифікатора.

Відкривається по кнопці Вибрати з поля Адреса в елементах списків.

Введення адреси

  • У полі Країна виберіть країну адреси. Країна "Україна" вибрана програмою по умовчанню, при цьому поле може бути недоступним в залежності від настройок. Для того, щоб змінити країну, натисніть Вибрати. Вибір здійснюється зі списку Статистичної класифікації країн світу.
  • Поле Індекс призначене для введення поштового індексу, відповідного даній адресі.
  • Поле Місто, населений пункт призначено для введення регіону, міста і т. п. При необхідності можна ввести частині адреси детально, натиснувши Вибрати. У відкритому шаблоні введіть дані населеного пункту.
  • Поле Вулиця призначене для вибору (натисніть кнопку Вибрати, в списку вулиць знайдіть потрібну) або введення вулиці адреси.
  • Поля Будинок, Корпус, Квартира дозволяють ввести відповідні частини адреси. Можна змінити ці варіанти, вибравши додаткові значення зі списку.
  • Також можна Додати додаткові поля для введення інших напередвизначених частин адреси, таких як Будова, Літера, Споруда, Ділянка, Офіс, Бокс, Приміщення, Кімната, А/с.
  • для введення номера будинку натисніть кнопку в вікні Вибір будинку виводиться список всіх можливих будинків і будівель для введеної адреси. Додаткові частини адреси, наприклад, корпус, також вибираються зі списку.
  • Поле Представлення адреси при заповненні адреси за шаблоном недоступне, заповнюється програмою автоматично в міру заповнення полів адреси.

Приклади заповнення адреси

  • Для введення адреси "Київ, Комарова проспект, будинок 1, корпус 2, літера Щ, офіс 12345" необхідно:
    • У полі Місто, населений пункт вказати "Київ м".
    • У полі Вулиця вказати "Комарова пр-кт".
    • В полі Будинок вказати "2".
    • В полі Корпус вказати "2".
    • Для введення літери необхідно Додати поле, вибрати з випадаючого списку значення Літера і проставити в поле значення "Щ".
    • Замість поля Квартира вибрати зі списку поле Офіс і ввести значення "12345".

Історія змін

  • У деяких списках можна переглянути історію зміни адреси за допомогою команди Ще - Історія змін.

Адреса на Google Maps

  • За допомогою цих команд будь-яку адресу, введену в програмі, можна подивитися на картах.

Введення адреси у вільній формі

  • Увімкніть прапорець Адреса у вільній формі меню Ще, для того щоб при введенні адреси за шаблоном ввести свідомо неправильну адресу (наприклад, з товаром на накладній неправильну адресу постачальника, його треба тимчасово зафіксувати для друку, але надалі його використовувати не можна). Така адреса використовується для зберігання, тому можна рекомендувати використовувати його тільки для одиничних випадків.
  • Напишіть адресу. Перевірка заповнення адреси в даному випадку відключена.
  • Для підтвердження адреси натисніть ОК. Програмою буде зафіксовано, що адреса заповнена у вільній формі і на перегляд адреса буде виводитися в такому вигляді.

Очищення адреси

  • З допомогою команди меню Ще - Очистити адресу можна стерти неправильно введену адресу.
