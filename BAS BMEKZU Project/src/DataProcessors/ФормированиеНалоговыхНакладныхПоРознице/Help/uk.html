Обробка "Формування податкових документів по роздробі"

Обробка "Формування податкових документів по роздробі" призначена для заповнення податкових накладних з роздрібних продажів.

Для заповнення використовується інформація про прийнятмй роздрібний виторг за зазначену дату, на суму якої не були виписані податкові накладні.
