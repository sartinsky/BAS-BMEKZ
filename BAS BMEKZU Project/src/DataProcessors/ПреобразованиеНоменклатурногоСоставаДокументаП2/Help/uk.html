Обробка перетворює Додатки 2, виписані в ІБ, в т.ч. автоматично, відповідно до чинного Порядку виписки ПН.

В Д2 здійснюється ідентифікація коригувань наступних видів:

- повне повернення по рядку ПН: вказується Причина коригування «Повернення товару або авансових платежів», код причини 103. Номери груп коригування присвоюються автоматично;

- часткове повернення (зменшення кількості з іншої причини без зміни ціни) по рядку ПН: формується два рядки, в першому сторнується вся кількість по рядку ПН, в другому вказується кількість, що залишилася в ПН після зменшення; вказується Причина коригування «Зміна кількості», код причини 102. Номери груп коригування присвоюються автоматично;

- зміна ціни без зміни кількості по рядку ПН: формується два рядки, в першому сторнується стара ціна для усієї кількості по рядку ПН, в другому вказується вся кількість по рядку ПН за новою ціною; вказується Причина коригування «Зміна ціни», код причини 101. Номери груп коригування присвоюються автоматично;

- зміна номенклатури, або зміна ціни і кількості одночасно: користувач самостійно формує один сторнуючий рядок і один або кілька нових рядків-замін і вказує для них загальний номер групи. Обробка при необхідності перетворює сторнуючий рядок в два рядки, за правилами, описаними для часткового повернення, і вказує для всіх рядків Причину коригування «Зміна номенклатури», код причини 104;

- умовний продаж: формується два рядки, в першому сторнується вся сума по рядку ПН, в другому вказується база і сума ПДВ, що залишилися в ПН після зменшення; вказується Причина коригування «Умовний продажів», код причини 201. Номери груп коригування присвоюються автоматично;

- інші випадки: користувач може самостійно сформувати рядки, вказати для них коди причин і номера груп коригування.

У випадках, коли рядок вихідного Д2 розбивається на два рядки, нові рядки з позитивною кількістю нумеруються починаючи з номера, вказаного користувачем. За замовчуванням нумерація починається з номера, наступного за номером останнього рядка ПН.

Важливо! Перед випискою Д2 до податкової накладної, виписаної до 01.12.2018, потрібно в цій ПН номера рядків привести у відповідність з номерами рядків, зареєстрованих в ЄРПН. У загальному випадку для цього достатньо записати ПН. Однак якщо в процесі вивантаження ПН змінювалася, потрібно вручну проставити в ній номера рядків.

- проведені
