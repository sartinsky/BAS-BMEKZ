Обработка преобразует Приложения 2, выписанные в ИБ, в т.ч. автоматически, в соответствии с действующим Порядком выписки НН.

В П2 осуществляется идентификация корректировок следующих видов:

- полный возврат по строке НН: указывается Причина корректировки «Повернення товару або авансових платежів», код причины 103. Номера групп корректировки присваиваются автоматически;

- частичный возврат (уменьшение количества по другой причине без изменения цены) по строке НН: формируется две строки, в первой сторнируется все количество по строке НН, во второй указывается количество, оставшееся в НН после уменьшения; указывается Причина корректировки «Зміна кількості», код причины 102. Номера групп корректировки присваиваются автоматически;

- изменение цены без изменения количества по строке НН: формируется две строки, в первой сторнируется старая цена для всего количества по строке НН, во второй указывается все количество по строке НН по новой цене; указывается Причина корректировки «Зміна ціни», код причины 101. Номера групп корректировки присваиваются автоматически

- изменение номенклатуры, либо изменение цены и количества одновременно: пользователь самостоятельно формирует одну сторнирующую строку и одну или несколько новых строк-замен и указывает для них общий номер группы. Обработка при необходимости преобразует сторнирующую строку в две строки, по правилам, описанным для частичного возврата, и указывает для всех строк Причину корректировки «Зміна номенклатури», код причины 104;

- условная продажа: формируется две строки, в первой сторнируется вся сумма по строке НН, во второй указывается база и сумма НДС, оставшиеся в НН после уменьшения; указывается Причина корректировки «Умовний продаж», код причины 201. Номера групп корректировки присваиваются автоматически;

- прочие случаи: пользователь может самостоятельно сформировать строки, указать для них коды причины и номера групп корректировки.

В случаях, когда строка исходного П2 разбивается на две строки, новые строки с положительным количеством нумеруются начиная с номера, указанного пользователем. По умолчанию нумерация начинается с номера, следующего за номером последней строки НН.

Важно! Перед выпиской П2 к НН, выписанной до 01.12.2018, нужно в этой НН номера строк привести в соответствие с номерами строк, зарегистрированными в ЕРНН. В общем случае для этого достаточно записать НН. Однако если в процессе выгрузки НН изменялась, нужно вручную проставить в ней номера строк.
