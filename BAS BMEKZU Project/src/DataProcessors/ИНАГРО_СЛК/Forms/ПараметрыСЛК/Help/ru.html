Установка сервера СЛК 2.1

Для использования Системы Лицензирования Конфигураций (СЛК) 2.1 необходимо установить на любой компьютер в локальной сети сервер СЛК. Дистрибутив сервера СЛК поставляется с конфигурацией.

Убедиться в успешности установки сервера СЛК 2.1 и посмотреть доступные ключи защиты можно, открыв окно консоли сервера СЛК. Для установки программного ключа используется команда «Установить программный ключ» из окна консоли сервера СЛК. Аппаратные ключи СЛК доступны для использования без дополнительных настроек.

Подробная инструкция доступна по ссылке http://licencecenter.ru/downloads/doc/activation.pdf.

Настройка параметров сервера СЛК 2.1

Для настройки параметров подключения к серверу СЛК 2.1 необходимо в верхней табличной части указать путь к серверу СЛК и сетевой порт сервера СЛК, по умолчанию 9099. Сервера СЛК используются в очередности указания в табличной части. Для пользователей можно указать приоритетный сервер СЛК.

Для отключения связи с сервером СЛК 2.1 используется кнопка "Демо режим".

Настройка пользователей

Настройки доступа пользователей к подсистемам указываются в нижней табличной части, там же указывается приоритетный сервер СЛК (номер строки сервера, нумерация с нуля). При наличии ведущих подсистем (шапка колонки выделена жирный) необходимо использовать хотя бы одну из них. При отключении всех ведущих подсистем работоспособность не гарантируется. 

Для указания настройки для всех пользователей поле "Пользователь" необходимо оставить пустым.
