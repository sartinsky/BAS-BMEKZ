В этой форме можно посмотреть различные свойства фонового задания.

1. Группа свойств "Основное".

Наименование свойства Назначение свойства
Наименование

Пустое, если запускалось вручную либо это основное фоновое задание, запускаемое регламентным заданием. Содержит УИД регламентированного задания если это создаваемое фоновое задание в рамках другого фонового задания.

Состояние

Возможны значения: "Выполнено", "Завершено с ошибками", "Отменено пользователем", "Выполняется".

Дата начала Дата и время начала выполнения фонового задания.
Дата окончания Дата и время окончания выполнения фонового задания (если задание еще выполняется, тогда пустая дата и время: <>.
Место выполнения Имя компьютера сети без домена, на котором выполнялось фоновое задание. Это либо имя сервера, либо имя компьютера пользователя, если регламентные задания обрабатывались в файловой информационной базе.

 

2. Группа свойств "Дополнительно".

Наименование свойства Назначение свойства
Идентификатор

Нужен только для продвинутого администратора, знающего внутренний язык 1С. Требуется для исследования нестандартных проблем в работе фоновых заданий, если таковые есть.

Ключ

Значение свойства Ключ, указанного в регламентном задании (на момент создания фонового задания) или другое, указанное явно, если фоновое задание создано программно, а не планировщиком сервера.

Имя метода Имя процедуры в конфигруции, которая выполнятся или выполнялась в фоновом задании. Обычно соответствует свойству "имя метода" соответствующего регламентного задания.

 

3. Сообщения пользователю и описание информации об ошибке.

Поле содержит текс сообщений и текст сведений об ошибке, если сообщения и/или ошибки произошли при выполнении фонового задания.
