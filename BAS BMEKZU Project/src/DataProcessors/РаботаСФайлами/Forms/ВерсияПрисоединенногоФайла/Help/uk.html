Перегляд відомостей про версію файлу.

Після закінчення редагування файлу (по команді Завершити редагування у списку файлів) програма рекомендує зберегти версію файлу і ввести інформацію про дану версію. По умовчанню цю можливість включено (рекомендується використовувати збереження версій, так як в разі серйозних помилок, які призвели до псування файлу, можна повернутися до попередньої, незіпсованої версії). Згодом можна відредагувати коментар і при необхідності змінити автора версії.

Відомості про версію файлу

  • Власник - найменування файла-власника, автоматично заповнюється при додаванні версії, не підлягає редагуванню;
  • Найменування - найменування версії, автоматично заповнюється при додаванні версії, не підлягає редагуванню;
  • Батьківська версія - для того щоб підпорядкувати одну версію іншої, виберіть зі списку потрібну версію. Історія версій в програмі для наочності представляється у вигляді дерева;
  • Розмір (байт) - розмір даних версії файлу в байтах.
  • Розширення - розширення (формат) версії файлу. Поруч розташовується картинка, яка також показує формат файлу, визначається програмою автоматично.
  • Коментар до версії - введіть додаткові відомості про версію файлу (також можна ввести при збереженні версії по закінченні редагування файлу).
  • Автор - користувач, який додав версію файлу. При необхідності можна вибрати іншого автора зі списку користувачів.
  • Дата створення - дата і час додавання версії файлу в базу.
  • Номер версії - нумерація версій ведеться програмою автоматично при змінах файлу, при цьому максимальний номер має остання збережена версія.
  • Дата зміни - дата зміни даних версії файлу (місцевий час).
  • Кодування - автоматично вибирається значення По умовчанню.

Перегляд версії файлу

  • Натисніть Переглянути, для того щоб відкрити вибрану версію файлу відповідним додатком для перегляду або редагування (також можна відкрити файл подвійним натисканням кнопки миші по потрібному рядку або за допомогою відповідної команди меню Ще).

Збереження версії файлу на диск

  • Для того щоб зберегти файл на диск або інший пристрій, натисніть Зберегти як. Виберіть каталог для збереження файлу.
