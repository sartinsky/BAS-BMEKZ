Призначений для роботи з файлами, які приєднані до об'єкта програми.

Відкривається по команді Приєднані файли панелі навігації об'єкта програми.

У списку виводиться:

  • - наявність ЕП або шифрування.
  • Найменування - повне ім'я файлу. Значок поруч з найменуванням вказує тип файлу. При наявності в колонці виводиться також і опис - докладні відомості про файл.
  • Автор - користувач, який додав файл;
  • Дата зміни - дата зміни файлу даних (місцевий час);

За допомогою команди Ще - Додатково - Змінити форму можна додати відомості:

  • Дата створення - дата і час введення файлу в програму;
  • Редагує - користувач, який зайняв файл для редагування і т.д.

Додавання нового файлу

  • Натисніть Додати. Виберіть спосіб введення файлу з випадаючого меню:
    • Файл з диска;
      • вкажіть шлях до файлу на комп'ютері.
    • За шаблоном;
      • виберіть шаблон із запропонованого списку.
    • зі сканера.
      • відкривається вікно для роботи зі скануванням. Дотримуйтесь вказівок програми.
  • Після додавання файлу автоматично відкривається оновлений список файлів.
  • Файл зберігається в спеціальному каталозі. Для кожного об'єкта програми формується окрема папка.
  • У деяких списках програми елементи, до яких приєднані файли, позначаються значком .

Перегляд зображень у списку

  • Увімкніть прапорець Ще - Попередній перегляд, для того щоб можна було переглядати файли зображень прямо в списку в спеціальному вікні.

Перегляд картки файлу

  • Натисніть Ще - Відкрити картку (натисніть клавішу F2), для того щоб переглянути властивості виділеного в списку файлу, відкривши вікно Приєднаний файл, також можна скористатися відповідною командою контекстного меню.

Редагування приєднаних файлів

  • У тонкому клієнті редагування файлів табличних документів (формату .mxl) здійснюється у вбудованій формі редактор табличних документів. У веб-клієнті необхідно завантажити прикладну програму для редагування цих файлів. Детальніше про особливості редагування табличних документів у веб-клієнті див. тут.
  • Для редагування файлів інших форматів на комп'ютері повинні бути встановлені відповідні прикладні програми, наприклад, для редагування файлів у форматі Microsoft Word має бути встановлена ця програма.
  • Команда Переглянути відкриває файл зовнішньою програмою для перегляду або редагування, залежно від того, позначено чи ні файл як зайнятий іншим користувачем для редагування.
  • Команда Редагувати відкриває файл у режимі редагування, а також виконує захоплення файлу в програмі. Захоплений файл буде недоступний для редагування іншим користувачам програми і позначається в списку зеленим кольором.
  • Після закінчення редагування збережіть файл, закрийте зовнішній додаток. Натисніть Зберегти і звільнити файл в інформаційній базі, ця дія звільняє файл, зберігає його в програмі і знімає позначку зайнятості для редагування.
  • Команда Ще - Скасувати редагування звільняє файл, не зберігаючи вироблені в ньому зміни в програмі, і знімає позначку зайнятості для редагування. Команда може бути корисна, якщо необхідно відмовитися від результатів редагування файлу (наприклад, при редагуванні в файл були внесені небажані зміни або помилки).

Друк файлу

  • Натисніть кнопку Друк.

Відправка файлу по електронній пошті

  • Натисніть кнопку Надіслати. Програма формує поштове повідомлення, заповніть необхідні поля. Приєднаний файл автоматично буде додатком даного листа.

Збереження файлу

  • За допомогою команди Ще - Зберегти як можна зберегти файл в зазначений каталог на жорсткому або мережевому диску.
  • Також можна скористатися відповідною командою контекстного меню по правій кнопці миші.
  • Відповідною командою меню Ще або контекстного меню по правій кнопці миші можна Відкрити каталог файлу в Провіднику Windows.

Оновлення файлу

  • З допомогою команди Ще - Оновити з файлу на диску можна оновити файл з оригінального файлу на диску (з якого спочатку був створений файл у програмі).
  • Також можна скористатися відповідною командою контекстного меню по правій кнопці миші.

Порівняння файлів

  • Виберіть у списку файли для порівняння.
  • Виконайте команду Ще - Порівняти файли. При першому порівнянні файлів програма пропонує вибрати, за допомогою якої офісної програми виконати порівняння. Після цього відкриваються файли для порівняння в обраній офісній програмі. Програма для порівняння повинна бути встановлена на комп'ютері.

Електронний підпис і шифрування

У випадаючому меню по кнопці Електронний підпис і шифрування доступні команди для роботи з електронним підписом файлів:

  • Підписати - створює ЕП для файлу. Перед створенням ЕП програма пропонує вибрати сертифікат ЕП з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний файл стає недоступним для редагування і відображається у списках зі значком .
  • Зберегти разом з електронним підписом - зберігає підпис у файл на диску. Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.
  • Додати електронний підпис з файлу - додає підпис до файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) був отриманий пізніше, ніж підписаний файл.
  • Зашифрувати - по команді відбувається шифрування файлу. На відміну від підписання можуть використовуватися кілька сертифікатів, наприклад, якщо необхідно зашифрувати файл для декількох користувачів. Зашифрований файл у списках відображається зі значком .
  • Розшифрувати - по команді відбувається розшифровка файлу. Для розшифровки потрібно ввести пароль до закритого ключа.

Друк табличного документа зі штампом електронного підпису

  • Для друку підписаного електронним підписом табличного документа зі штампами візуалізації електронного підпису у списку приєднаних файлів передбачена команда Друк - Зі штампом элоектронной підпису.
  • Попередньо, табличний документ повинен бути підписаний одним або декількома електронними підписами відповідно до того, як це передбачено в документі.
