Призначена для вибору режиму відкриття файлу.

Для відкриття файлу в списку файлів використовуйте:

  • клавішу Enter потрібно вибрати режим відкриття файлу, якщо не було інших настройок;
  • подвійне натиснення лівої кнопки миші - так само, як і в попередньому випадку;
  • Кнопку Відкрити файл;
  • Команду Редагувати (F4).

Варіанти відкриття файлу

  • За допомогою перемикача виберіть режим відкриття файлу:
    • Тільки для перегляду - файл не буде помічений, як зайнятий Вами для редагування і відкриється зовнішнім додатком в режимі "тільки перегляд".
    • Для редагування - файл буде помічений, як зайнятий Вами для редагування і відкриється зовнішнім додатком з можливістю внесення і збереження змін.
  • Для підтвердження вибору натисніть Відкрити файл.

Відключення вибору режиму відкриття

  • Для того щоб більше не проводити вибір режиму відкриття, після вибору варіанту відкриття файлу увімкніть прапорець Більше не питати.
  • Якщо для текстових файлів у персональних настройках програми встановлений спосіб відкриття У вбудованому редакторі, то вибір режиму відкриття файлу не потрібно, редактор відкривається одразу і перемикання між режимами відкриття проводиться вже в самому редакторі.
  • Якщо файл був зайнятий для редагування, то вибір режиму відкриття файлу не потрібно, файл відкривається в режимі редагування.
  • Включити вибір режиму можна у персональних настройках програми на закладці Робота з файлами.
  • У персональних настройках програми (закладка Робота з файлами) можна вибрати Дію при виборі файлу:
    • Відкривати файл (установка по умовчанню);
      • Увімкніть прапорець Вибір режиму відкриття при виборі файлу. По умовчанню прапорець включений.
    • Відкривати картку файлу - в цьому випадку по клавіші Enter або подвійним клацанням миші відкривається картка файлу, вибір режиму відкриття файлу не проводиться.
