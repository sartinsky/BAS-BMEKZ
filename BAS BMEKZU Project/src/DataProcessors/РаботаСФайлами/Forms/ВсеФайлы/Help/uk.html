Список призначений для управління файлами програми.

У списку виводяться відомості:

  • - наявність електронного підпису та шифрування;
  • Найменування - повне найменування і опис файлу. Значок поруч з найменуванням вказує тип файлу;
  • Розміщення - папка списку файлів або об'єкт програми, до якого приєднано файл;
  • Автор - користувач, який додав файл;
  • Дата створення - час і дата додавання файлу в базу;
  • Розмір (Кб) - розмір файлу;
  • Розширення - розширення файлу;
  • Редагує - користувач, який зайняв файл для редагування;
  • Версія - номер версії файлу;
  • Відредагував - користувач, який додав версію файлу.
  • Том - том на диску, в якому зберігаються файли;
  • Шлях до файлу на томі.

Доступні дії:

  • Переглянути - Відкрити файл зовнішнім додатком для перегляду;
  • Скасувати редагування - зняти з файлу позначку зайнятості для редагування;
  • Відкрити картку файлу можна за допомогою відповідної команди меню Ще;
  • Зберегти файл у вказаний каталог на жорсткому або мережевому диску можна за допомогою команди Ще - Зберегти як.

Див. також:

  • Загальні відомості про файли;
  • Робота з формами.
