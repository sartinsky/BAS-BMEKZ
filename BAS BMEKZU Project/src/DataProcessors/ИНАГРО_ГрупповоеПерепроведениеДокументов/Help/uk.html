Групове перепроведення документів

 

Обробка призначена для групового перепроведения документів. Наприклад, для відновлення правильної послідовності проведення документів.

Перепровести документи можна по всім організаціям або по обраній організації.

При виконанні можна зупиняти перепроведення при виявленні помилки, для цього потрібно встановити відповідний прапорець.

Перепроведення виконується по кнопці Виконати.
