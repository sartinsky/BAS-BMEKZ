Адміністратора програми часто потрібно з'ясувати, які події відбувалися в програмі в певний момент часу або які дії виконував той чи інший користувач. Для цих цілей призначений журнал реєстрації.

У журналі реєстрації можуть фіксуватися основні дії, що виконуються по модифікації даних, виконання регламентних операцій, підключення і відключення від програми і т. д. Настройка журналу проводиться в Конфігураторі.

Відкрити журнал реєстрації можна за відповідною командою розділу Адміністрування - Обслуговування.

При роботі з програмою можуть виникати події наступних видів, кожен з яких відображається в журналі реєстрації спеціальним значком:

- Помилка;
- Попередження;
- Інформація;
- Інформація по недоступних полях;
- Примітка.
Кожна подія фіксується в окремому рядку журналу.

У списку виводиться:

  • Дата, час - крім дати і часу проставляється значок, що відображає вид події;
  • Користувач - повне ім'я користувача;
  • Комп'ютер - мережеве ім'я комп'ютера;
  • Додаток - наприклад, Тонкий клієнт;
  • Сеанс - номер сеансу;
  • Подія - тип події;
  • Дані, коментар - вказівка об'єкта програми: вказується конкретний елемент списку, документ і т.і. ; в коментарі - короткий опис події;
  • Метадані - тип об'єкта програми і його найменування: список, форма документа і т.і.

Для того щоб настроїти кількість, видимість і взаємне розташування колонок журналу реєстрації, натисніть Ще - Змінити форму.

Збереження журналу реєстрації на комп'ютер

  • Натисніть Вивантажити в файл, вкажіть шлях для збереження журналу на комп'ютер. Журнал зберігається у вигляді файлу XML, переглянути його можна, наприклад, в програмі MS Excel.

Оновлення списку подій

  • Список подій не є динамічним списком. Для того щоб побачити події, які потрапили в журнал реєстрації після його відкриття, натисніть Оновити поточний список або клавішу F5.

Кількість подій

  • Журнал відображає фіксовану кількість записів, яке вказується в полі Показувати (По умовчанню дорівнює 200). Крім напередвизначених значень (200, 500, 1000 і 10000 подій) можна ввести довільну кількість подій, яке буде збережено.

Відбори

За допомогою відборів можна відфільтрувати події в журналі реєстрації.

  • У полі швидкого відбору Критичність виберіть значення Помилки або Попередження, для того щоб побачити певні види подій. По умовчанню в журналі показані Всі події.
  • Для швидкого встановлення періоду перегляду подій журналу натисніть Інтервал. Також до встановлення інтервалів дат можна перейти з допомогою подвійного натискання по вмісту колонки Дата, Час.
  • Виділіть дані в одній з колонок журналу, натисніть Встановити відбір за значенням в поточній колонці. Будуть відібрані події журналу, що містять ці дані. Також можна знайти цю команду в контекстному меню по правій кнопці миші.
  • Для того щоб відібрати події за одним або кількома критеріями, натисніть Установити відбір. Також можна пройти за посиланням в поле Відбір. Якщо задані будь-які відбори (в тому числі і швидкі), посилання приймає значення критеріїв відбору, якщо ніяких відборів не встановлено, то приймає значення Не встановлений. Проводиться установка фільтрів відбору:
    • В полі Інтервал виберіть період, вкажіть Початок і Закінчення періоду;
    • У списку Важливість за допомогою прапорців виберіть один або кілька видів подій (Помилка, Попередження, Інформація, Примітка), за якими потрібно зробити відбір. По умовчанню встановлені прапорці;
    • В полі Події виберіть значення зі списку за допомогою прапорців (по умовчанню встановлені всі прапорці);
    • У полі Користувачі виберіть значення зі списку за допомогою прапорців (у цьому списку вказуються тільки користувачі, відображені в журналі реєстрації. По умовчанню прапорці знято);
    • У полі Програми виберіть значення зі списку за допомогою прапорців (Конфігуратор, Тонкий клієнт, Фонове завдання. По умовчанню прапорці знято);
    • У полі Комп'ютери виберіть потрібні значення зі списку за допомогою прапорців (по умовчанню прапорці зняті);
    • У полі Сеанси перерахуйте через ";" потрібні номери сеансів;
    • У групі полів Дані можна вибрати для пошуку один з видів відбору:
      • У полі Метадані виберіть потрібні значення зі списку (список упорядкований за типом даних) для відбору за допомогою прапорців: виберіть потрібне найменування списку, документа і т. д. наприклад, "довідник Класифікатор банків";
      • У полі Дані можна зробити вибір елементів для пошуку зі списків програми:
        • виберіть Тип даних, наприклад, "Банк";
        • З відповідного списку програми, наприклад, "Класифікатора банків", виберіть конкретний об'єкт програми, наприклад, "Ощадбанк";
    • в полі Представлення даних наберіть дані для пошуку вручну або скопіюйте зі списку;
    • У групі полів Транзакція виберіть зі списку за допомогою прапорців Статус транзакції (Немає транзакції, Зафіксована, Не завершена, Скасована). По умовчанню встановлені всі прапорці. Також можливий відбір за полем Ідентифікатор транзакції.
    • У групі полів Інше можна вибрати за допомогою прапорців для відбору Робочі сервери, Основні IP-порти, Допоміжні IP-порти, також можна провести відбір за полем Коментар.
  • Після зазначення полів для відбору натисніть ОК. Після цього події будуть відібрані і посилання прийме вид перерахування критеріїв відбору.
  • Відображаються записи, що задовольняють заданим критеріям відбору у кількості, не більшій, ніж значення поля Показувати.
  • Для того щоб відключити всі відбори (в тому числі і швидкі), натисніть Відключити відбір або виконайте команду Ще - відключити відбір. Цю команду також можна знайти в контекстному меню по правій кнопці миші.

Перегляд події

  • Для перегляду детальної інформації про подію виділіть її в списку, натисніть Відкрити. Можна знайти цю команду в контекстному меню по правій кнопці миші.
  • Також можна переглянути подію за допомогою подвійного натиснення кнопки миші по потрібному рядку.

Перегляд об'єкта програми

Якщо з подією пов'язаний деякий об'єкт програми, то цей об'єкт можна відкрити для перегляду.

  • Виділіть дані в колонці Дані, Коментар відповідної події і натисніть Відкрити дані для перегляду. Також можна знайти цю команду в контекстному меню по правій кнопці миші.
  • Відкрити дані для перегляду можна також подвійним натисненням кнопки миші по вмісту колонки Дані, Коментар.
