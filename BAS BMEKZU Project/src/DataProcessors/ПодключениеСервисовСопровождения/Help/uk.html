Призначена для підключення тестових періодів сервісів супроводу.

Підключення тестового періоду відображається для не підключених сервісів, якщо є можливість активації тестового періоду. Обробка дозволяє вибрати тестовий період зі списку доступних користувачеві і виконати підключення.
