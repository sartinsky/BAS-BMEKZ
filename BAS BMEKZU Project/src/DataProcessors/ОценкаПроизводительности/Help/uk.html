У програмі передбачені вбудовані засоби для збору і аналізу даних про продуктивність роботи за APDEX, яка є міжнародним стандартом оцінки продуктивності інформаційних систем.

Для того щоб оцінювати продуктивність, необхідно на початку роботи програми заповнити список Ключові операції.

Оцінка продуктивності програми за методикою APDEX складається з наступних основних етапів:

  • заповнення профілів ключових операцій (в профілі заповнюються пріоритет і цільове значення часу для кожної ключової операції, передбаченої в програмі);
  • збір інформації про час виконання кожної ключової операції;
  • на підставі зібраних даних - формування звіту Оцінки продуктивності за ключовими операціями.

До перегляду звіту Оцінка продуктивності за ключовими операціями можна перейти за відповідним посиланням розділу Адміністрування - Обслуговування - Звіти та обробки - Звіти адміністратора.

Для того щоб виробляти оцінку продуктивності, увімкніть прапорець Оцінка продуктивності. Після цього в програмі будуть проводитися виміри продуктивності під час виконання зазначених ключових операцій, наприклад, при проведенні документа.

  • Результати вимірів будуть оброблені в звіті Оцінка продуктивності.

Експорт вимірів продуктивності

  • Отримані виміри продуктивності програма дозволяє автоматично в фоні вивантажувати в зовнішні файли, наприклад, для виконання аналізу продуктивності в іншій програмі.
  • Для того щоб настроїти вивантаження цих даних, перейдіть за посиланням Настройки розділу Адміністрування - Обслуговування - Оцінка продуктивності.
  • Сформовані зовнішні файли, а також файли, що містять настройки, при яких були отримані ці виміри, автоматично упаковуються в ZIP-архів.
  • Для того щоб виконати експорт вимірів продуктивності вручну, натисніть Експорт вимірів. у розділі Адміністрування - Обслуговування - Оцінка продуктивності. Файл із вимірами продуктивності можна відкрити або зберегти. Для збереження файлу виберіть шлях до папки на комп'ютері.

Імпорт вимірів продуктивності

  • Можна проаналізувати виміри продуктивності іншої програми за допомогою імпорту з відповідного файлу.
  • Для імпорту скористайтеся посиланням Імпорт вимірів в розділі Адміністрування - Обслуговування - Оцінка продуктивності.
  • У полі Файл імпорту за допомогою кнопки вкажіть шлях до файлу на комп'ютері, натисніть кнопку Імпорт.
