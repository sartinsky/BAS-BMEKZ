Отчет "Данные по качеству зерна"

Отчет предназначен для анализа качества зерна, остатка зерна, выставленых услуг, исходные документы "Реестры", "Акт очистки/сушки", "Акт расчета", "Переоформление", "Переработка".

 

Измерения накапливаются в регистре свединий "Данные по качеству партий зерна" согласно настроек в регистре сведений "Настройка параметров учета элеватора"
