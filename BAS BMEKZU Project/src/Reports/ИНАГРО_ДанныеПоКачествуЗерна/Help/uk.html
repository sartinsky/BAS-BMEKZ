Отчет "Данні по якості зерна"

Звіт призначений для аналізу якості зерна, залишку зерна, виставлених послуг, вихідні документи "Реєстри", "Акт очистки/сушки", "Акт розрахунку", "Переоформлення", "Переробка".

 

Виміри накопичуються в регістрі відомостей "Дані по якості партій зерна" згідно налаштувань в регістрі відомостей "Налаштування параметрів обліку елеватора"
