Отчет «Журнал складских квитанций»

Данный отчет предназначен для формирования перечня складских квитанций. Для вывода данного отчета следует выбрать пункт «Журнал складских квитанций» в меню «Элеватор/Отчеты» главного меню программы. При запуске данного отчета на экран будет выдана диалоговая форма запроса параметров работы.

В данной форме можно указать следующие параметры отбора информации:

• «Период» отчета. Устанавливает временные границы для формирования отчета;

• «Организация». Устанавливается фирма, по которой необходимо сформировать отчет. Если же фирма не указана, то отчет будет сформирован по всем фирмам;

• «Тип отчета» – определяет какой вид складских квитанций нужно отобрать;

• «Владелец». Включает возможность отбора данных по конкретному владельцу зерна. Если не указывать, то отчет будет сформирован по всем владельцам зерна;

• «Культура» - Реквизит определяет культуру или группу культур, по которым будет сформирован журнал;

• «Урожай». Выбираем год урожая, по которому будет сформирован журнал;

• «Вид хранения» - при выборе конкретного значения в отчет попадет информация только о нем, иначе по всем видам хранения;

• «Склад» – если реквизит выбран, то в отчет будут включены данные, имеющие отношение к выбранному складу. Если склад не указан – отчет формируется по всем складам
