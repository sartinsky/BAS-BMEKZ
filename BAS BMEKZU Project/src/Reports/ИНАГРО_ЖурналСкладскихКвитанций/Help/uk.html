Звіт «Журнал складських квитанцій»

Даний звіт призначений для формування переліку складських квитанцій. Для виведення даного звіту слід вибрати пункт «Журнал складських квитанцій» в меню «Елеватор / Звіти» головного меню програми. При запуску даного звіту на екран буде видана діалогова форма запиту параметрів роботи.

В даній формі можна вказати наступні параметри відбору інформації:

• «Період» звіту. Встановлює часові межі для формування звіту;

• «Організація». Встановлюється фірма, по якій необхідно сформувати звіт. Якщо ж фірма не вказана, то звіт буде сформований по всіх фірмах;

• «Тип звіту» - визначає який вид складських квитанцій потрібно відібрати;

«Власник». Включає можливість відбору даних по конкретному власнику зерна. Якщо не вказувати, то звіт буде сформований за всіма власникам зерна;

• «Культура» - Реквізит визначає культуру або групу культур, за якими буде сформовано журнал;

• «Урожай». Вибираємо рік врожаю, за яким буде сформований журнал;

• «Вид зберігання» - при виборі конкретного значення в звіт потрапить інформація тільки про нього, інакше за всіма видами зберігання;

• «Склад» - якщо реквізит обраний, то до звіту будуть включені дані, що мають відношення до заданої складу. Якщо склад не вказано - звіт формується по всіх складах
