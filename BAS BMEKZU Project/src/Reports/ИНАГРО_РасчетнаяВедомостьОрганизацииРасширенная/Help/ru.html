Отчет "Расчетные ведомости организаций"

Отчет предоставляет возможность сформировать произвольную расчетную ведомость предприятия, отобранную и сгруппированную по различным критериям.

Отчет по умолчанию настроен так, чтобы в результате получилась ведомость в разрезе организаций и подразделений.

Возможные показатели отчета

  • Сумма начисления или удержания
  • Отработано дней
  • Отработано часов
  • Оплачено дней
  • Оплачено часов

Возможные группировки отчета

  • Головная организация
  • Организация
  • Подразделение организации
  • Должности организации
  • Месяц начисления
  • Работник

Для того чтоб получить отчет только по работникам-пенсионерам надо в настройках отчета, на закладке "Отбор" поставить галочку напротив поля - "только пенсионеры." Аналогичные действия надо сделать, если необходим только по работникам-инвалидам.

В отчете есть возможность получать вывод ФИО ответственных лиц организации (директор, гл. бухгалтра). Для того чтоб включить эту возможность, требуется в настройках пользователя поставить галочку, напротив поля "Выводить подписи ответственных лиц организации". Вывод подписей будет только в том случае, если установлен быстрый отбор по организации.
