Звіт «Типова форма П5»

Звіт «Типова форма П-5» призначений для складання організаціями табель обліку робочого часу. Звіт дозволяє сформувати табелі як по всіх організаціях відразу, так і по конкретних організації або підрозділу або по співробітнику працівників. Засоби відбору даних для звіту показуються / ховаються в діалозі звіту за допомогою кнопки «Швидкі відбори». Кнопка «Сформувати» дозволяє отримати Табель, заповнений за наявною в ІБ інформації про використання робочого часу. Кнопка «Бланк» дозволяє отримати порожні бланки табелів зі списком працівників, але без даних про використання робочого часу. Звіт дозволяє виводити табель кожного підрозділу організації як окремий документ, умовні позначення, примітка.
