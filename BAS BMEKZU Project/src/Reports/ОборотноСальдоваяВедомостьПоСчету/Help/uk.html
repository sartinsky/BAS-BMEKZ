Звіт "Оборотно-сальдова відомість по рахунку"

Звіт призначений для формування оборотно-сальдової відомості по вибраному рахунку за певний період часу.

За складом вихідної інформації звіт нагадує фрагмент звіту Оборотно-сальдова відомість.

Звіт можна сформувати з деталізацією по субрахунках або по об'єктах аналітичного обліку (субконто). 

Дані можна вивести з додатковою розбивкою за періодами часу: місяць, рік і т. д.

У звіті можна вивести розгорнуте сальдо. При цьому розгорнуте сальдо розраховується для кожного рівня груповання і по рахунку в цілому.
