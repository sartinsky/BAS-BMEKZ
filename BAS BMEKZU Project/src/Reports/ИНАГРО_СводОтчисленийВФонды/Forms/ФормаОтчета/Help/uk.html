Для перегляду звіту необхідно натиснути кнопку Сформувати, попередньо задавши параметри (якщо вони передбачені). У клієнт-серверному варіанті роботи програми передбачено формування звітів у фоновому режимі.

Перегляд звіту

  • По умовчанню форма звіту складається з двох частин: в лівій частині відображаються відомості звіту, в правій частині розташовуються швидкі налаштування. 
  • Для зручності перегляду звіту передбачена кнопка Настройки. Натисніть кнопку для того, щоб приховати швидкі настройки і побачити весь звіт. Повторне натискання кнопки повертає настройки.
  • В програмі може бути передбачено кілька варіантів звітів, які виконують схожий аналіз. У цьому випадку у програмі передбачені:
    • для настройки звітів - вкладка Настройки,
    • для перегляду списку інших варіантів звітів - вкладка Інші звіти. Кнопка Настройки набуває вигляду Настройки та інші звіти, натиснувши її, можна показати або приховати обидві вкладки.

Настройки

  • Основні параметри звіту виведені на вкладку Настройки, наприклад, Період. Період можна вибрати із списку:
    • рік (місяць, день) - в цьому випадку можна вибрати потрібне значення із списку (Фіксований) або вибрати Відносний і значення вказати Минулий, Цей і т.п.
    • довільний період, у цьому випадку вкажіть дату початку і закінчення періоду. 
  • Для відбору даних за потрібними параметрами увімкніть відповідні прапорці.
  • Настройки зберігаються.
  • По команді Всі дії - Всі настройки можна перейти до настройки всіх передбачених параметрів звіту. 
  • Передбачена настройка відборів:
    • одиничних (використовуються операції порівняння Дорівнює, Не дорівнює, В групі, Не в групі). Після вибору операції вкажіть значення для відбору.
    •  множинних (операції порівняння В списку, Не в списку, В групі зі списку, Не в групі зі списку). Для множинних відборів натисніть кнопку Підбір, виберіть із запропонованого списку.
  • Вкажіть потрібні параметри, натисніть кнопку Завершити редагування. 
  • Натисніть Всі дії - Зберегти настройки, для того щоб записати поточні параметри звіту.
  • Натисніть Всі дії - Завантажити настройки, для того щоб скористатися раніше збереженими настройками звіту.

Інші звіти

На вкладці Інші звіти розміщений список схожих за змістом звітів. 

  • В списку виводяться найменування та опису звітів.
  • Список звітів згрупований. Група Варіанти звіту призначена для швидкого вибору інших варіантів цього звіту.
  • Можуть бути інші групи, в яких виводяться схожі за змістом інші звіти тієї ж підсистеми. 
  • Для переходу до іншого варіанту звіту натисніть на заголовок лівою кнопкою миші.

Друк звіту

  • Сформований звіт можна роздрукувати, натиснувши кнопку Друк.
  • Для того щоб перевірити звіт і змінити параметри сторінки, натисніть кнопку Попередній перегляд.
  • З допомогою команди Всі дії - Параметри сторінки можна настроїти друк.

Надсилання звіту електронною поштою

З допомогою команди Відправити звіт можна:

  • Надіслати зараз, для цього:
    • за допомогою прапорців виберіть один або кілька форматів, в яких необхідно відправити звіт.
    • при необхідності увімкніть прапорець Додатково запакувати в архів (.zip).
    • Натисніть Продовжити. Звіт буде прикріплений до електронного листа в якості вкладення.
  • Створити розсилку звітів;
  • Включити в існуючу розсилку звітів. 

Вибір варіанту звіту

  • Для завантаження збереженого раніше варіанта звіту натисніть кнопку Вибрати варіант, або скористайтеся командою Всі дії - Вибрати варіант.
  • Натисніть кнопку Вибрати для переходу до потрібного варіанту звіту.

Зміна варіанту звіту

  • Натисніть Всі дії - змінити варіант звіту. Можна додати поля, змінити параметри та настройки звіту.
  • Внесені зміни можна зберегти у інший варіант звіту.

Збереження варіанта звіту

  • Натисніть Всі дії - Зберегти варіант, вкажіть Найменування та інші параметри.

Скидання настройок

Натисніть Всі дії - Скинути настройки, для того щоб скинути настройки.

Інші доступні дії

  • Для пошуку в звіті скористайтеся командою Знайти.
  • За допомогою кнопокЗгорнути всі групи або Розгорнути всі групи можна приховати або показати групування звіту.
  • Для збереження результату звіту у файл, натисніть .
  • Натисніть для того щоб підрахувати суму чисел у виділених клітинках.
