Отчет "Журнал лабораторных анализов"

Данные количественно качественных показателей Реестров ТТН по каждой культуре регистрируются в журнале лабораторных анализов.

Данный отчет является основным отчетом для ВТЛ  он показывает итоги по лабораторным анализам за день и за весь период анализа.

Для вывода данного отчета следует выбрать пункт «Журнал лабораторных анализов» в меню «Элеватор/Лаборатория» главного меню программы. 

Отбор

На закладке отбор можно задать всевозможные параметры для отбора информации: период формирования отчета; организацию, по которой необходимо сформировать отчет, владелец – элемент справочника «Контрагенты», вид культуры – элемент справочника «Виды культур», культура – элемент справочника «Номенклатура». По умолчанию отчет отображает виды характеристик, у которых установлен признак «Показывать в журнале лабораторных анализов», для того чтобы отчет отобразил весь перечень характеристик необходимо установить флажок «Показать все виды характеристик». Отчет имеет несколько форм отображения информации: «Журнал лабораторных анализов (форма 49)» и «Журнал лабораторных анализов с ц./% (форма 49», форма отчета выбирается в реквизите «Форма».
