Звіт "Відвантаження по одержувачах (аналіз відвантажень)"

призначений для аналізу зерна, що вивозиться за межі підприємства, продуктів виробництва або непридатних відходів.
Має зручний набір налаштувань для конфігурації вмісту, що виводиться.
Перед формуванням звіту користувачеві необхідно заповнити форму введення параметрів звіту:
 1. Період формування;
 2. Необхідні відбори.
