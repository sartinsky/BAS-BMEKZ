Звіт "Продажі по місяцях"


Інформація для керівника


У звіті Продажі по місяцях відображається сума продажів за основними видами діяльності організації по місяцях за різні роки. Для аналізу даних по окремій номенклатурній групі або контрагенту можна встановити відповідний відбір.

В суму продажу включається виторг за відвантажені товари чи надані послуги, незалежно від того, надійшла оплата від покупців, чи ні.

Візуально звіт представлений у вигляді графіка.

Інформація для бухгалтера


В суму продажу включаються тільки продажі за основними видами діяльності (оборот по кредиту рахунку 70), а інші доходи (оборот за кредитом рахунку 71) не включаються.

Показники звіту можуть бути розшифровані звітом Продажі в розрізі номенклатурних груп.
