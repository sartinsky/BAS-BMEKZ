Звіт "Аналіз по ТМЦ"

Звіт призначений для аналізу ТМЦ організації за конкретний період, вказаний в шапці, або за весь період роботи організації, якщо період не вказаний. По кнопці «Відбір» можливо відібрати значення по номенклатурі, рахунку обліку та складу. В звіті відображаються залишки по ТМЦ на початок та кінець періоду, а також надходження і видаток ТМЦ за вказаний період. По кнопці «Сформувати» виконується побудова звіту згідно поточних налаштувань.
