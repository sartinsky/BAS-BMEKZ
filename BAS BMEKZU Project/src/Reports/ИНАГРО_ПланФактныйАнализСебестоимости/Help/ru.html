Отчет "План-фактный анализ себестоимости"

Отчет предназначен для план-фактного анализа себестоимости продукции в разрезе номенклатур и статей затрат. Предусмотрена возможность установить отбор по периоду, организации, номенклатуре и статье затрат. По кнопке «Настройки» можно указать дополнительные параметры для формирования отчета.
