Звіт "Форма 36"

Даний звіт призначений для формування журналу кількісно-якісного обліку хлібопродуктів за формою № 36. Журнал кількісно-якісного обліку ведеться за кожним місцем зберігання, по власнику зерна та по культурі, з виділенням партії сортового насіння, сортів пивоварного ячменю, природних сумішей зерна, а по борошну і крупі - по кожному сорту.

Для формування даного звіту слід вибрати пункт «Форма 36» в меню «Елеватор/Звіти» головного меню програми.

Звіт формує 36 форму як по кожному власнику, так і загальну форму (по складу), вид форми визначається прапорцями: «Загальна» чи «По власнику». 

Відбір

У даній формі можна вказати наступні параметри відбору інформації:

·           «Період» звіту. Встановлюються часові межі для формування звіту;

·           «Організація». Встановлюється фірма, по якій необхідно сформувати звіт. Якщо ж фірма не вказана, то звіт буде сформований по всіх фірмах;

·            «Власник». Включає можливість відбору даних по конкретному власнику зерна. Якщо не вказувати, то звіт буде сформований по всім власникам зерна;

·           «Культура» - Реквізит визначає культуру або групу культур, по яких буде сформовано журнал;

·           «Урожай». Вибираємо рік урожаю, по якому буде сформований журнал;

·           «Вид зберігання» - при виборі конкретного значення в звіт потрапить інформація тільки про нього, інакше - за всіма видами зберігання;

·           «Склад» – якщо реквізит обраний, то в звіт будуть включені дані, що мають відношення до обраному складу. Якщо склад не зазначений - звіт формується по всіх складах.

Формування звіту, після вибору необхідних параметрів виведення, відбувається натисненням на кнопку «Сформувати».

У сформованому журналі кількісно-якісного обліку за формою № 36 вказуються реквізити документів руху зерна, показники якості, розраховується центнеро-відсоток по приходу і витратах, ведеться контроль залишків.

У звіт включаються підсумки за кожен день, по кожній культурі, власнику і року урожаю.

При розшифровці рядків таблиць буде запропоновано відкрити форму відповідного документа.
