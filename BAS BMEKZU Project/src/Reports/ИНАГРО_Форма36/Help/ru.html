Отчет "Форма 36"

Данный отчет предназначен для формирования журнала количественно-качественного учета хлебопродуктов по форме №36. Журнал количественно-качественного учета ведется по каждому месту хранения, владельцу зерна, культуре, с выделением партии сортовых семян, сортов пивоваренного ячменя, естественных смесей зерна, а по муке и крупе - по каждому сорту.

Для вывода данного отчета следует выбрать пункт «Форма 36» в меню «Элеватор/Отчеты» главного меню программы.

Отчет формирует 36 форму как по каждому владельцу, так и общую форму (по складу), вид формы указывается флажками: «Общая», «По владельцу». 

Отбор

В данной форме можно указать следующие параметры отбора информации:

·           «Период» отчета. Устанавливает временные границы для формирования отчета;

·           «Организация». Устанавливается фирма, по которой необходимо сформировать отчет. Если же фирма не указана, то отчет будет сформирован по всем фирмам;

·            «Владелец». Включает возможность отбора данных по конкретному владельцу зерна. Если не указывать, то отчет будет сформирован по всем владельцам зерна;

·           «Культура» - Реквизит определяет культуру или группу культур, по которым будет сформирован журнал;

·           «Урожай». Выбираем год урожая, по которому будет сформирован журнал;

·           «Вид хранения» - при выборе конкретного значения в отчет попадет информация только о нем, иначе по всем видам хранения;

·           «Склад» – если реквизит выбран, то в отчет будут включены данные, имеющие отношение к выбранному складу. Если склад не указан – отчет формируется по всем складам.

Формирование отчета, после выбора необходимых параметров вывода, происходит нажатием на кнопку «Сформировать».

В сформированном журнале количественно-качественного учета по форме №36 указываются реквизиты документов движения зерна, показатели качества, рассчитывается центнеро-процент по приходу и расходу, ведется контроль остатков.

В отчет включаются итоги за каждый день, по каждой культуре, владельцу и году урожая.

При расшифровке строк таблиц будет предложено открыть форму соответствующего документа.
