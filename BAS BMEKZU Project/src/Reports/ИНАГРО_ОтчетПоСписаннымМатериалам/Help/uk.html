Звіт "Звіт по списаним матеріалам"

Звіт призначений для аналізу списаних матеріалів організації на певний основний засіб за конкретний період, вказаний в шапці документа, або за весь період роботи організації, якщо період не заданий. В шапці звіту можливо відібрати значення по організації, основному засобу та номенклатурі. В звіті відображаються кількість та сума списаних ТМЦ за вказаний період. Кнопка «Налаштування» дозволяє вказати додаткові параметри для формування звіту. По кнопці «Сформувати» виконується побудова звіту згідно поточних налаштувань.
