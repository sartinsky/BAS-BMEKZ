Отчет "Отчет по списанным материалам"

Отчет предназначен для анализа списанных материалов организации на определенное основное средство за конкретный период, указанный в шапке, или за весь период работы организации, если период не указан. В шапке отчета можно отобрать значения по организации, основному средству и номенклатуре. В отчете отображаются количество и сумма списанных ТМЦ за указанный период. Кнопка «Настройка» позволяет указать дополнительные параметры для формирования отчета. По кнопке «Сформировать» выполняется построение отчета в соответствии с текущей настройкой.
