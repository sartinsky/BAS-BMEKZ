Отчет "Журнал по направлению зерна"

  Данный отчет предназначен для анализа введенных в программу документов ТТН ввоз   в зависимости от направления на место хранения и присвоенному номеру пробы(среднесуточного анализа)

 

  С помощью системы отбора журнал можно сформировать  по конкретному владельцу или культуре.

  Данный журнал служит также для проверки корректности присвоения номера пробы
