Звіт "Аналіз по закупці"

Звіт призначений для аналізу закупок організації. Передбачена можливість встановлення відбору по періоду, організації, контрагенту та складу, на який відбувалось придбання. По кнопці «Налаштування» можна вказати додаткові параметри для виведення звіту. Для формування необхідно натиснути кнопку «Сформувати».
