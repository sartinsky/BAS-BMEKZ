Отчет "Анализ по закупкам"

Отчет предназначен для анализа закупок организации. Предусмотрена возможность установить отбор по периоду, организации, контрагенту и складу, на который осуществлялась покупка. По кнопке «Настройки» можно указать дополнительные параметры для вывода отчета. Для формирования  необходимо нажать кнопку «Сформировать».
