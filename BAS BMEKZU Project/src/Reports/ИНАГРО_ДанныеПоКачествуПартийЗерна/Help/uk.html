Отчет "Данные по качеству партий зерна"

Отчет предназначен для анализа партий, качества зерна, остатка зерна, выставленных услуг, исходные документы "Реестры", "Акт очистки/сушки", "Акт расчета", "Переоформление", "Переработка".

 

Измерения  накапливаются в регистре сведений "Данные по качеству партий зерна" согласно настроек в регистре сведений "Настройка параметров учета элеватора", для учета партий и выставлений услуг по факту.
