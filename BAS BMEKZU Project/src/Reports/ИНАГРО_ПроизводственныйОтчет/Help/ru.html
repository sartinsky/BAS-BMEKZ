Отчет «Производственный отчет»

Данный отчет предназначен для анализа документов переработки. С его помощью можно проанализировать состав и количество сырья, пошедшего на производство готовой продукции для любого владельца. Кроме того отчет содержит информацию о процентном содержании каждого вида сырья в составе партии готовой продукции.
