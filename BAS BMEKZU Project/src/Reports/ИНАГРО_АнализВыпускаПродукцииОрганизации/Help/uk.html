Звіт «Аналіз випуску продукції організації»

Звіт призначений для аналізу випуску продукції організації. Передбачена можливість встановити відбір по періоду, організації, підрозділу і номенклатурній групі. По кнопці «Налаштування» можливо вказати додаткові параметри для формування зовнішнього виду звіту.
