Отчет "Товарный отчет"

Отчет предназначен для формирования товарного отчета.

Отчет формирует вывод данных : остатки на начало периода, суммы поступления товара в разрезе приходных документов, суммы расхода товара в разрезе расходных документов, остатки товара на конец периода

В форме диалога отчета задаются следующие настройки:

  • Даты начала и конца периода , за который требуется сформировать отчет
  • Номер отчета. Номера товарных отчетов по складу должны нумероваться последовательно.
  • Организация, для которой формируется отчет.
  • Розничный склад, по которому формируется отчет
