Отчет «Расчетные листы организаций»

Отчет предназначен для получения краткой информации по всем начислениям и удержаниям по сотруднику на конкретный расчетный период. С помощью кнопки «Отбор» можно отобрать значения по организации, подразделению или работнику. По кнопке «Сформировать» выполняется построение отчета в соответствии с текущей настройкой.
