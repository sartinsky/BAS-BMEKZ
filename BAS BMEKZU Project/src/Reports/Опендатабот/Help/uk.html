Опендатабот. Ризики

Звіт дозволяє отримати і відобразити інформацію про дані ЄДР, дані податкової служби, реєстраційні дані, судові провадження і т.д. для організацій і ФОП, інформацію про участь фізичної особи в судових процесах, наявність в базі розшуку МВС, базі боргів по аліментах.

При відкритті звіту рядок пошуку заповнюється значенням поля «Код за ЄДРПОУ» довідника «Контрагенти» або «Код за ДРФО» довідника «Фізичні особи» поточного елемента.

Пошук інформації здійснюється за:

  • Код за ЄДРПОУ - для отримання даних по організаціям;
  • Код за ДРФО - для отримання даних по ФОП;
  • ПІБ - для отримання даних по фізичним особам.

При пошуку інформації по ПІБ, можна групувати інформацію по регіону. Для цього, в додаткових параметрах звіту, необхідно вибрати пріоритетний регіон.

Дані, які вже були отримані за допомогою звіту, зберігаються і можуть бути використані користувачем протягом 24 годин, без використання платної перевірки. Якщо необхідно отримати актуальні дані, необхідно встановити ознаку «Оновлювати дані, збережені в програмі» в групі «Додаткові параметри» форми звіту.
