Опендатабот. Ризики

Отчет позволяет получить и отобразить информацию о данных ЕГР, данных налоговой службы, регистрационных данных, судебных производствах и т.д. для организаций и ФЛП, информацию об участии физического лица в судебных процессах, наличие в базе розыска МВД, базе долгов по алиментам.

При открытии отчета строка поиска заполняется значением поля «Код по ЕДРПОУ» справочника «Контрагенты» или «Код по ДРФО» справочника «Физические лица» текущего элемента.

Поиск информации осуществляется по:

  • Код по ЕДРПОУ – для получения данных по организациям;
  • Код по ДРФО – для получения данных по ФЛП;
  • ФИО – для получения данных по физическим лицам.

При поиске информации по ФИО, можно группировать информацию по региону. Для этого, в дополнительных параметрах отчета, необходимо выбрать приоритетный регион.

При поиске информации по коду ЕДРПОУ или коду ДРФО, выполняется проверка существования контрагента в справочнике. Если элемент справочника существует, то будет показана ссылка на карточку Контрагента. Если в справочнике отсутствует запись с заданным кодом, то контрагента можно будет автоматически создать нажав кнопку "Создать контрагента". При этом будет выполнено заполнение необходимых полей (наименование, полное наименование, ИНН плательщика НДС, схема налогообложения, юридический адрес, номер телефона, электронная почта, контактное лицо). 

Заполнить информацию о контрагенте из сервиса "Опендатабот. Ризики" можно из карточки контрагента, заполнив поле "Код по ЕДРПОУ" и нажав кнопку "Заполнить по ЕДРПОУ". 

Данные, которые уже были получены с помощью отчета, сохраняются и могут быть использованы пользователем в течении 24 часов, без использования платной проверки. Если необходимо получить актуальные данные, необходимо установить признак «Обновлять данные, сохраненные в программе» в группе «Дополнительные параметры» формы отчета.
