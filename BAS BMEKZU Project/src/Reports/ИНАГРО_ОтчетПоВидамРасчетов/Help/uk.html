Звіт "Звіт по видах розрахунків"

За допомогою цього звіту можна отримати найрізноманітніші вибірки по видах розрахунку за довільний період по працівниках організації. Рамки такого періоду встановлюються для реквізиту Період реєстрації. Записи, що включаються в звіт, можуть бути відібрані як по періоду дії, так і по періоду реєстрації. Вибірку записів можна також обмежити записами по співробітниках, вказавши відповідні налаштування для реквізиту Співробітник.

Можливі показники звіту

  • Сума;
  • Отр. днів;
  • Отр. годинника;
  • Норма днів;
  • Норма годин.

 

Можливі угрупування звіту

  • Організація;
  • Вид розрахунку;
  • Співробітник;
  • Період дії;
  • Реєстратор.
