Отчет "Отчет по видам расчетов"

С помощью данного отчета можно получить самые разнообразные выборки по видам расчета за произвольный период по работникам организации.

Рамки такого периода устанавливаются для реквизита Период регистрации.

Записи, включаемые в отчет, могут быть отобраны как по периоду действия (флажок «Период действия»), так и по периоду регистрации (флажок "Период регистрации").

Выборку записей можно также ограничить записями по сотрудникам, указав соответствующие настройки для реквизита Сотрудник.

Возможные показатели отчета

  • Сумма
  • Отработано дней
  • Отработано часов
  • Норма дней
  • Норма часов

Возможные группировки отчета

  • Организация
  • Вид расчета
  • Сотрудник
  • Период действия
  • Регистратор
