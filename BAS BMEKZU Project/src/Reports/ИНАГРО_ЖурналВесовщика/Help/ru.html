Отчет "Журнал весовщика"

Отчет «Журнал весовщика» является отчетом весовщика. Отчет дает возможность провести анализ всего сырья, что ввозится, вывозится и перемещается внутри предприятия.

Отчет предназначен для анализа движения ТМЦ через весы предприятия. Имеет несколько форм отображения информации (форма ЗХС-28, произвольная форма), группировки по владельцам, итоги по взвешиваниям за день и за весь период анализа. Вид печатных форм указывается в реквизите «Форма». 

Отбор

На закладке отбор можно задать всевозможные параметры для отбора информации: период анализа документов; организацию по которой необходимо сформировать отчет, если же фирма не указана, то отчет будет сформирован по всем фирмам. В поле журнал весовщика

 указывется весовой журнал по которому необходимо получить данные.
