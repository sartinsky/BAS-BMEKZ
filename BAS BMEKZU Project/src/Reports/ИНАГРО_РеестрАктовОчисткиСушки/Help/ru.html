Отчет "Реестр актов очистки/сушки"

Данный отчет предназначен для анализа введенных в программу документов «Форма 34 сводная». Отчет отображается в разрезе документов.

Для вывода данного отчета следует выбрать пункт «Реестр актов очистки сушки» в меню «Элеватор/Отчеты» главного меню программы. 

Отбор

В этой форме можно указать следующие параметры для отбора информации:

·           «Период» отчета. Устанавливает временные границы для формирования отчета;

·           «Организация». Устанавливается фирма, по которой необходимо сформировать отчет. Если же фирма не указана, то отчет будет сформирован по всем фирмам;

·           «Культура». Реквизит определяет культуру или группу культур, по которым будет сформирован отчет;

·           «Вид культуры». Реквизит определяет вид культуры, по которым будет сформирован отчет;

·           «Владелец». Включает возможность расчета данных по конкретному владельцу зерна. Если не указывать, то будет сформирован отчет по всем владельцам зерна;

·           «Договор». Реквизит дает возможность отбора данных для отчета по выбранному договору;

·           «Склад» – если реквизит выбран, то в отчет будут включены данные, имеющие отношение к выбранному складу. Если склад не указан – отчет формируется по всем складам;

·           «Вид хранения» - при выборе конкретного значения в отчет попадет информация только о нем, иначе по всем видам хранения;

·           «Вид урожая». Выбираем год урожая, по которому будет сформирован журнал;

Возможны варианта сортировки отчета:

·           дате;

·           владельцу;

·           культуре.

Формирование отчета, после выбора необходимых параметров вывода, происходит нажатием на кнопку «Сформировать».

При расшифровке строк таблиц Вам будет предложено открыть форму соответствующего документа
