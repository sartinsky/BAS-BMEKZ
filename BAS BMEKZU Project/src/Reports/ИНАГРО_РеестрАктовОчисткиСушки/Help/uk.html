Звіт "Реєстр актів очистки/сушки"

Даний звіт призначений для аналізу введених в програму документів «Форма 34 зведена». Звіт відображається в розрізі документів.

Для виведення даного звіту слід вибрати пункт «Реєстр актів очистки/сушки» в меню «Елеватор/Звіти» головного меню програми. 

Відбір

У цій формі можна вказати наступні параметри для відбору інформації:

·           «Період» звіту. Встановлює часові межі для формування звіту;

·           «Організація». Встановлюється підприємство, по якому необхідно сформувати звіт. Якщо ж фірма не вказана, то звіт буде сформований по всіх підприємствах;

·           «Культура». Реквізит визначає культуру або групу культур, по яких буде сформований звіт;

·           «Вид культури». Реквізит визначає вид культури, за якими буде сформований звіт;

·           «Власник». Включає можливість розрахунку даних по конкретному власнику зерна. Якщо не вказувати - буде сформовано звіт за всіма власникам зерна;

·           «Договір». Реквізит дає можливість відбору даних для звіту за обраним договором;

·           «Склад» – якщо реквізит обраний, то в звіт будуть включені дані, що мають відношення до обраному складу. Якщо склад не зазначений - звіт формується по всіх складах;

·           «Вид зберігання» - при виборі конкретного значення в звіт потрапить інформація тільки про даний вид зберігання, інакше - за всіма видами зберігання;

·           «Вид врожаю». Вибираємо рік урожаю, по якому буде сформований журнал;

Можливі варіанти сортування звіту:

·           даті;

·           власнику;

·           культурі.

Формування звіту, після вибору необхідних параметрів виведення, відбувається натисненням на кнопку «Сформувати».

При розшифровці рядків таблиць Вам буде запропоновано відкрити форму відповідного документа.
