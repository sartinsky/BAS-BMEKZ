Звіт "Динаміка заборгованості постачальникам"


Інформація для керівника


Звіт Динаміка заборгованості постачальникам відображає зміни заборгованості постачальникам по організації за період. Дані наводяться без деталізації по контрагентам.

Інформація для бухгалтера


Заборгованість постачальникам визначається як залишок за кредитом рахунків обліку розрахунків за договорами з видом договору З постачальником, З комітентом і З комісіонером.

Строк оплати за договорами враховується двома способами:

  • Якщо передбачена єдина політика для взаєморозрахунків з усіма постачальниками, достатньо вказати в настройках параметрів обліку строк оплати постачальникам. Строк оплати за договорами з постачальниками встановлюється в настройках параметрів обліку на закладці Розрахунки з контрагентами.
  • Якщо передбачені особливі умови розрахунків з окремими постачальниками, то необхідно в довіднику Договори для конкретного договору встановити прапорець Встановлений строк оплати за договором, днів і вказати строк оплати.

Для визначення заборгованості постачальникам може бути змінено порядок розрахунку: можна виключити окремі рахунки з аналізу. Змінений порядок розрахунку буде застосовуватися для всіх звітів розділу "Розрахунки з постачальниками".

Показники звіту можуть бути розшифровані звітом Заборгованість постачальникам по строках боргу.
