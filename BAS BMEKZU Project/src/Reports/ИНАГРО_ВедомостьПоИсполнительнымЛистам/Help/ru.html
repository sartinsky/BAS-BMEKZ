Отчет "Ведомость по исполнительным листам"

Отчет предоставляет возможность сформировать свод удержанных по исполнительным листам сумм за произвольный период.

Отчет по умолчанию настроен так, чтобы в результате получился список удержанных по исполнительным листам сумм, сгруппированный по организациям, месяцам и работникам.

Возможные показатели отчета

  • Размер удержания
  • Размер почтового сбора

Возможные группировки отчета

  • Организация
  • Месяц удержания
  • Работник
  • Исполнительный документ и его параметры
