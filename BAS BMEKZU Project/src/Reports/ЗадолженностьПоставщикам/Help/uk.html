Звіт "Заборгованість постачальникам"


Інформація для керівника


У звіті Заборгованість постачальникам відображається заборгованість постачальникам і сума авансу детально по постачальникам на початок та кінець обраного періоду, а також рух за період – збільшення і погашення боргу. Постачальники в звіті відсортовані за спаданням величини боргу на кінець періоду.

Інформація для бухгалтера


Заборгованість постачальникам визначається як залишок за кредитом рахунків обліку розрахунків за договорами з видом договору З постачальником, З комітентом. Залишок по дебету (аванс) відображається в окремій графі. Зміна заборгованості визначається як обороти за рахунками обліку розрахунків.

Дані звіту аналогічні даними стандартного звіту Анализ субконто по субконто Договір і відбором по виду договору З постачальником, З комітентом, але з додатковим сортуванням за зменшенням величини боргу на кінець періоду.

Для визначення заборгованості постачальникам може бути змінено порядок розрахунку: можна виключити окремі рахунки з аналізу. Змінений порядок розрахунку буде застосовуватися для всіх звітів розділу "Розрахунки з постачальниками".
