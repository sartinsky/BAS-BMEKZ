Звіт "Заборгованість покупців по строках боргу"


Інформація для керівника


Звіт Заборгованість покупців по строках боргу дозволяє розділити дебіторську заборгованість за періодами виникнення, а також вивести суму простроченої заборгованості. Кількість і тривалість періодів (інтервалів) можна настроїти самостійно.

Покупці в звіті відсортовані за спаданням величини боргу на дату звіту.

Інформація для бухгалтера


Заборгованість покупців визначається як залишок по дебету рахунків обліку розрахунків за договорами з видом договору З покупцем, З комісіонером і З комітентом. Сума заборгованості за кожним контрагентом розбивається по інтервалам в залежності від дати виникнення боргу – дати відвантаження покупцю товарів, надання послуг.

Строк оплати за договорами враховується двома способами:

  • Якщо передбачена єдина політика для взаєморозрахунків з усіма покупцями, достатньо вказати в настройках параметрів обліку строк оплати для покупців. Строк оплати за договорами з покупцями встановлюється в настройках параметрів обліку на закладці Розрахунки з контрагентами.
  • Якщо передбачені особливі умови розрахунків з окремими покупцями, то необхідно в довіднику Договори для конкретного договору встановити прапорець Встановлений строк оплати за договором, днів і вказати строк оплати.

Для визначення заборгованості покупців може бути змінено порядок розрахунку: можна виключити окремі рахунки з аналізу. Змінений порядок розрахунку буде застосовуватися для всіх звітів розділу "Розрахунки з покупцями.
