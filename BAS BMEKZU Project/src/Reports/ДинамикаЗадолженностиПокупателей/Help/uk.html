Звіт "Динаміка заборгованості покупців"


Інформація для керівника


Звіт Динаміка заборгованості покупців показує зміну заборгованості покупців по організації за період. Дані наводяться без деталізації по контрагентам.

Інформація для бухгалтера


Заборгованість покупців визначається як залишок по дебету рахунків обліку розрахунків за договорами з видом договору З покупцем, З комісіонером і З комітентом.

Для визначення заборгованості покупців може бути змінено порядок розрахунку: можна виключити окремі рахунки з аналізу. Змінений порядок розрахунку буде застосовуватися для всіх звітів розділу "Розрахунки з покупцями", а також для показників звіту Оборотні кошти.

Показники звіту можуть бути розшифровані.
