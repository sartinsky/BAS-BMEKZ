Отчет "Ведомость зарплаты по путевым листам"

Отчет предназначен для получения данных по отработанному времени, сумме основной зарплаты, дополнительной зарплаты и общей сумме зарплате,  начисленной по путевым листам. Возможна детализация по организациям, месяцу начисления, сотрудникам, транспортным средствам, датам начисления и документам.
