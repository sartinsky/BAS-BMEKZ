Звіт "Відомість зарплати по подорожніх  листах"

Звіт призначений для отримання  даних по відпрацьованому часу, сумі основної зарплати, додаткової зарплати та загальній сумі нарахованої зарплати  за шляховими листами. Можлива деталізація по організаціям, місяцю нарахування, співробітникам, транспортним засобам, датам нарахування і документам.
