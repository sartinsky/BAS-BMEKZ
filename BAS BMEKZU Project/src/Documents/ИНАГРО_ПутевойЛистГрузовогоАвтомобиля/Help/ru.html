Документ "Путевой лист грузового автомобиля"

Документ предназначен для отражения следующих операций:

  • заправка транспорта ГСМ (горюче-смазочным материалом)

  • списание ГСМ по норме и сверх нормы по каждому рейсу

  • выпуск готовой продукции (тонно-километры или километры)

  • расчет заработной платы водителя

Заполнение формы документа следует начать с выбора автомобиля.

При выборе автомобиля автоматически в реквизиты «Водитель» подставляется водитель.

Если документ заполняется не за один день, то необходимо установить флаг «Сводный путевой лист». При этом в табличной части документа появится дополнительная колонка, в которой нужно указать дату первичного документа.

Далее выберите из списка необходимую технологическую операцию. По необходимости укажите марку прицепа. Это необходимо для дополнительного расчета списания топлива.

Укажите счет куда списываются затраты и его аналитику. Статья затрат должна отображать списание топлива.

Перенесите с бумажного носителя всю необходимую информацию:

  • Количество ходок

  • Тоннаж

  • Расстояние

  • Пробег с грузом

  • Отработано часов

Сумма зарплаты рассчитывается автоматически. Топливо, которое было израсходовано по факту записываем в нижней таблице.

При выполнении хозяйственных операций возможна ситуация когда происходит перерасход топлива. Отобразить перерасход можно на отдельный счет. Для этого необходимо установить флажок «Выделить перерасход топлива» и указать счет списания затрат. Ввести в процентном соотношении любую надбавку по топливу.
