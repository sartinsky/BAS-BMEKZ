Документ "Подорожній лист вантажного автомобіля"

Документ призначений для відображення слідуючих операцій:

  • заправка транспорту ПММ (паливно-мастильним матеріалом)

  • списання ПММ по нормі та понад норму по кожному рейсу

  • випуск готової продукції (тонно-кілометри або кілометри)

  • розрахунок заробітної плати водія

Заповнення форми документу слід починати з вибору автомобіля.

Під час вибору автомобіля автоматично в реквізити «Водій» підставляється водій.

Якщо документ заповнюється не за один день, то необхідно встановити прапорець «Зведений подорожній лист». При цьому в табличній частині документу з'явиться додаткова колонка, в якій необхідно вказати дату первинного документу.

Далі виберіть із списку необхідну технологічну операцію. По необхідності вкажіть марку прицепу. Це необхідно для додаткового розрахунку списання палива.

Вкажіть рахунок, на який будуть списуватися затрати та його аналітику. Стаття затрат повинна відображувати списання палива.

Перенесіть з паперового носія всю необхідну інформацію:

  • Кількість ходок

  • Тоннаж

  • Відстань

  • Пробіг с вантажем

  • Відпрацьовано годин

Сума зарплати розраховується автоматично. Паливо, що було витрачено по факту, записуємо у нижній таблиці.

Під час виконання господарських операцій можлива ситуація, коли відбувається перевитрата палива. Відобразити перевитрату можна на окремий рахунок. Для цього необхідно встановити прапорець «Виділити перевитрату палива» та вказати рахунок списання витрат. Ввести у процентному відношенні будь-яку надбавку по паливу.
