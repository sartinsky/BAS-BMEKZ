Документ "Коригування списком"

Документ «Коригування списком» призначений для виконання довільних операцій по коригуванню оперативних даних у підсистемі елеватор. Його функція аналогічна документу «Коригування» тільки лише за тим винятком, що документ «Коригування списком» пропонує можливість введення необхідних даних у вигляді списку (табличної частини), що дозволяє його використовувати, наприклад, для групового введення залишків. На відміну від документа «Коригування» в документі є можливість ввести вид фасування, за умови що реквізит «Фасоване» має значення «істина».

Документ застосовується для введення початкових залишків або внесення змін безпосередньо в регістри зберігання інформації.

Назви і призначення реквізитів документа аналогічні документу «Коригування».

При проведенні документа формуються рухи по регістрах оперативного обліку.
