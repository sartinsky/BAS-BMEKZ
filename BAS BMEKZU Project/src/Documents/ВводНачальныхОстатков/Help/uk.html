Документ «Введення початкових залишків»

Документ призначений для введення вхідних залишків у програму.

Введення залишків здійснюється в розрізі різних розділів обліку.

Розділ обліку вибирається в реквізиті "Розділ обліку".

Розділи обліку згруповані у відповідності з планом рахунків бухгалтерського обліку.

Для введення даних, для яких не виділено окремий розділ обліку, слід вибрати розділ "Інші рахунки бухгалтерського обліку".

В залежності від обраного розділу буде змінюватися склад даних, які необхідно ввести і форма документа.

Після закінчення введення вхідних залишків рекомендується провести контроль введених даних, сформувавши звіт по вхідних залишках.

Для введення залишків рекомендується використовувати спеціалізовану обробку "Введення початкових залишків".
