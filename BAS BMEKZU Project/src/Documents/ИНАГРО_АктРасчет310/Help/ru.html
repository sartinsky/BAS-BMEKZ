Документ "Акт расчет(№310)"

Для анализа изменений состояния партии зерна по конкретному Владельцу разработан документ «Акт-расчет». Документ относится к обязательным документам ХПП. В акте производится расчет объема и основных показателей зерна возвращаемого владельцу, и обоснования в уменьшении веса. Документ применяется для закрытия лицевого счета по клиенту и культуре.

 

Общие реквизиты документа

 Для этого необходимо заполнить следующие реквизиты:

·           «Владелец» - определяет тип справочника «Контрагенты»;

·           «Договор» - необходимо указать документ договор по которому осуществляется хранение зерна;

·           «Культура» - элемент справочника «Номенклатура» определяет культуру по которой будет выполнятся расчет;

·           «Вид хранения» - вид хранения, на котором хранится зерно;

·           «Урожай» - имеет вид года сбора урожая, данной зерновой продукции;

·           «Склад» - внутренний склад элеватора, на котором хранится зерно;

·           «За период» - указать период, за который производится расчет.

Группы реквизитов «Принятое зерно, средние показатели качества», «Подлежит возврату качество при отпуске» и «Расход осн. зерна, побочных продуктов, отходов» заполняются автоматически при нажатии кнопки «Рассчитать». Перед расчетом пользователь должен указать качество зерна, которое осталось у владельца на остатке.

На закладке «отклонения» также автоматически будет рассчитан средний срок хранения партии а также недостача, которая оправдывается снижением влажности, сорной примеси естественной убылью при хранении зерна.

 

В табличной части «Движение» пользователь имеет возможность просмотреть список документов, вошедших в расчет.

Документ может выполнять расчет остатка зерна как по факту, так и предварительно. Для того чтобы указать способ расчета документ имеет два вида операции: «предварительный расчет», «расчет по факту». При предварительном расчете с помощью флажков «очищали», «сушили», которые находятся на закладке «Основные», необходимо указать будит ли рассчитаны снижение сорной примеси или снижение влажности.  Вид операции «предварительный расчет» следует использовать для того если необходимо получить предварительный остаток зерна у владельца, а вид операции «расчет по факту» следует использовать при закрытии партии зерна владельца. Если необходимо спрогнозировать вес побочного продукта или отходов 3 категории при предварительном расчете необходимо на закладке «Прогноз побочных продуктов и отходов» для каждого вида отходов указать  принцип расчета.

Для печати документа нажмите на кнопку «Печать». У документа есть несколько печатных форм: «Акт расчет и №-310», «Акт расчет» (для вывода на печать предварительного расчета), «Акт расчет и №-661».

Особенности проведения документа

При проведении документа с видом операции «расчет по факту» формируются движения по регистрам оперативного учета и проводки по счету 023 «Материальные ценности на ответственном хранении». Документ с видом расчета «Предварительный расчет» никаких движение и проводок не делает.
