Документ "Акт розрахунок(№310)"

Для аналізу змін стану партії зерна по конкретному Власнику розроблений документ «Акт-розрахунок». Документ належить до обов'язкових документів ХПП. В акті проводиться розрахунок обсягу та основних показників зерна повертаючого власнику, і обгрунтування по зменшенню ваги. Документ застосовується для закриття особового рахунку по клієнту та культурі.

Загальні реквізити документа

 Для цього необхідно заповнити такі реквізити:

·           «Власник» - визначає тип довідника «Контрагенти»;

·           «Договір» - необхідно вказати документ договір за яким здійснюється зберігання зерна;

·           «Культура» - елемент довідника «Номенклатура» визначає культуру по якій буде виконуватися розрахунок;

·           «Вид зберігання» - вид зберігання, на якому зберігається зерно;

·           «Урожай» - має вигляд року збору врожаю, даної зернової продукції;

·           «Склад» - внутрішній склад елеватора, на якому зберігається зерно;

·           «За період» - вказати період, за який проводиться розрахунок.

Групи реквізитів «Прийняте зерно, середні показники якості», «Підлягає поверненню якість при відпустці» і «Витрата осн. зерна, побічних продуктів, відходів» заповнюються автоматично при натисканні кнопки «Розрахувати». Перед розрахунком користувач повинен вказати якість зерна, яке залишилося у власника на залишку.

На закладці «відхилення» також автоматично буде розрахований середній термін зберігання партії а також нестача, яка виправдовується зниженням вологості, смітної домішки природним спадом при зберіганні зерна.

 

В табличній частині «Рух» користувач має можливість переглянути список документів, які увійшли в розрахунок.

Документ може виконувати розрахунок залишку зерна як по факту, так і попередньо. Для того щоб вказати спосіб розрахунку документ має два види операції: «попередній розрахунок», «розрахунок по факту». При попередньому розрахунку за допомогою прапорців «очищали», «сушили», які знаходяться на закладці «Основні», необхідно вказати чи будуть розраховані зниження смітної домішки або зниження вологості.  Вид операції «попередній розрахунок» слід використовувати для того якщо необхідно отримати попередній залишок зерна у власника, а вид операції «розрахунок по факту» слід використовувати при закритті партії зерна власника.

Для друку документа натисніть на кнопку «Друк». У документа є кілька друкованих форм: «Акт розрахунку і №-310», «Акт розрахунок» (для виводу на друк попереднього розрахунку).

Особливості проведення документа

При проведенні документа з видом операції «розрахунок по факту» формуються рухи по регістрах оперативного обліку та проводки по рахунку 023 «Матеріальні цінності на відповідальному зберіганні». Документ з видом розрахунку «Попередній розрахунок» ніяких рухів і проводок не робить.
