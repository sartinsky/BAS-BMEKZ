Документ «Приход из производства»

 

Документ предназначен для оприходования из производства готовой продукции. Документ выполняет движение по регистрам оперативного учета подсистемы элеватор.

В шапке документа необходимо заполнить следующие реквизиты:

·           «Склад» - место хранения, на которое будет выполнено оприходование товароматериальных ценностей;

·            «Владелец» - владелец продукции;

·           «Договор» - документ-основание для взаиморасчетов с клиентом;

·            «Урожай» и «Вид хранения» - оприходовать на определенный урожай и  вида хранения.

В табличной части необходимо указать перечень наименований готовой продукции, их количество и качество.

·           «Рецепт» - указывается согласно какого рецепта производится выпуск продукции.

·           «Продукция», «Ед» - приходуемое наименование продукции, единица измерения;

·            «Количество, шт.», «Вес, кг» - указывается количество мешков, и общий вес продукции;

·           «Лабораторный анализ», «№ анализа», «Влажность», «Сорная примесь» - реквизиты лабораторного анализа, показатели качества;

Для печати документа нажмите кнопку «Печать». Документ имеет несколько печатных форм: «Рапорт»; «Накладная (форма 112)»

При проведении документа формируются движения по регистрам оперативного учета и проводки по счету 023 «Материальные ценности на ответственном хранении».
