Документ "Встановлення порядку підрозділів для закриття рахунків"

Документ призначений для завдання послідовності підрозділів для закриття рахунків при розрахунку собівартості готової продукції і напівфабрикатів.

У табличній частині "Порядок закриття" зазначаються наступні реквізити:

  • Підрозділ. Значення реквізиту вибирається з довідника "Підрозділи організацій".

Передбачено заповнення даної табличної частини по кнопці "Заповнити". У табличну частину потрапляють підрозділу в залежності від організації.
