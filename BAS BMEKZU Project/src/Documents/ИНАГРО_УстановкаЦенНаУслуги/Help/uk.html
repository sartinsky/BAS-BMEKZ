Документ "Установка цін на послуги"

Даний документ призначений для введення в програму розцінок на послуги ХПП.  

Загальні реквізити документа

У програмі можуть бути розцінки, які застосовуються для переважної більшості клієнтів, в такому випадку в документі реквізити «Контрагент» та «Договір» потрібно залишити порожніми. Для внесення персональних розцінок реквізити «Контрагент» та «Договір» необхідно заповнити.

Документ містить табличну частину, в якій зазначаються: вид культури - елемент довідника «Види культур» для розрахунку вартості послуг; найменування послуги - елемент довідника «Номенклатура», для якої вводиться ціна. Автоматично показується «стара ціна», що діяла на передодні зміни. Нова ціна вводиться в однойменному полі табличної частини.У нижній частині форми розташовані кнопка «Заповнити» - заповнює документ «старими цінами».

Особливості проведення документа

Після запису документ робить записи в регістрі відомостей «Ціни на послуги».
