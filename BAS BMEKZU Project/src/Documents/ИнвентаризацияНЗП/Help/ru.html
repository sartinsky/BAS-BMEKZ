Документ "Инвентаризация незавершенного производства"

Документ заполняется по результатам проведенной инвентаризации остатков незавершенного производства или по расчетным данным.

Документ необходимо ввести по каждому производственному подразделению, в котором есть остатки НЗП на конец месяца. Остатки НЗП указываются только в суммовом выражении с точностью до подразделения и номенклатурной группы производства.

Документ не формирует движений. Данные, указанные в нем, учитываются при расчете себестоимости: в себестоимость продукции (работ, услуг), выпущенных подразделением по номенклатурной группе, включаются все производственные затраты, кроме оставленных в НЗП.
