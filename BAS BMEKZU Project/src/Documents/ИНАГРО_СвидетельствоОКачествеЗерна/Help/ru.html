Документ "Свидетельство о качестве зерна"

Документ предназначен для регистрации в системе выписанных свидетельствах о качестве зерна  (форма № 41).

 

Общие реквизиты документа

В шапке документа, кроме «Организации», заполните следующие реквизиты:

·           «Владелец», «Договор» - укажите владельца анализируемой ТМЦ и договор, по которому осуществляется анализ;

·           «Склад» - укажите склад предприятия, на который оприходовано зерно;

·           «Вид хранения» - вид хранения, на который оприходована данная номенклатура;

·           «ТМЦ» - принимает значение справочника «Номенклатура»;

·           «Урожай» - имеет вид года сбора урожая данной зерновой продукции, элемент справочника «Виды урожая».

В реквизите «Лабораторный анализ» соответствующей закладки указывается соответствующий лабораторный анализ из списка, из анализа автоматически подставляется номер анализа в реквизит «№ анализа» и качественные показатели «Влажность», «Сор. Примесь», «Зерн. Примесь», эти реквизиты можно указать и вручную.

На закладке «Дополнительно» указываются другие реквизиты, необходимые для корректного формирования печатной формы документа.

Чтобы вывести печатную форму (№ 41) на печать воспользуйтесь кнопкой «Печать».

Особенности проведения документа

Документ не формирует никаких движений.
