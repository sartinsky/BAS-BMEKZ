Документ "Переміщення ОЗ"

Документ "Переміщення ОЗ" призначений для встановлення нового значення матеріально-відповідальної особи і підрозділу для переліку основних засобів з бухгалтерського обліку.

Реквізити

  • Підрозділ. Підрозділ вибирається з довідника "Підрозділи організацій". Це підрозділ, в якому буде обліковуватися основний засіб в бухгалтерському обліку. При початку вибору підрозділу відкривається список підрозділів тільки тієї організації, яка вказана в документі. Якщо основний засіб може використовуватися в різних підрозділах і амортизація по ньому може нараховуватися на витрати цих підрозділів. У цьому випадку підрозділи, на які будуть відноситися витрати по амортизації, будуть вказуватися при описі способів відображення витрат по амортизації в довіднику "Способи відображення витрат по амортизації". А в поточному реквізиті слід вказати підрозділ, за яким основний засіб буде обліковуватися.
  • МВО. Матеріально-відповідальна особа вибирається зі списку працівників обраної організації. При цьому відкривається форма для вибору працівників, які на момент оформлення документа прийняті на роботу в організацію. Якщо підрозділ заповнений, то у формі вибору працівника показується список працівників тільки вибраного підрозділу організації.

При проведенні документа дані записуються в регістр відомостей "Місцезнаходження ОС Бухгалтерський Облік". При цьому обов'язково заповнення поля "Організація". Проводки в бухгалтерському обліку не формуються.

Для швидкого заповнення табличної частини однотипними об'єктами основних засобів, що мають однакові найменування, потрібно ввести в табличну частину хоча б один такий об'єкт. Потім слід скористатися кнопкою Заповнити у командній панелі табличної частини. При виборі в меню пункту По найменуванню таблична частина буде заповнена об'єктами основних засобів, мають таке ж найменування, як у введеного спочатку.
