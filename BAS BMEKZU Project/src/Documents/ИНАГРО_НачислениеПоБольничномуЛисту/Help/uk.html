Документ "Нарахування по лікарняному листу"

Документ "Нарахування по лікарняному листу" реєструє заміну планових нарахувань нарахуванням оплати по середньому заробітку з варіантом розрахунку "Для лікарняних листів" вказаному в документі робітнику на вказаний період.

В шапці документа вказуються інші необхідні реквізити, зокрема, причина, відсоток оплати по лікарняному (який залежить від тривалості безперервного трудового стажу працівника).

Сума нарахування розраховується автоматично натисненням кнопки "Розрахувати". Результати розрахунку відображаються в табличній частині документа на закладці "Нарахування", На закладці "Розрахунок середнього заробітку" відображаються результати розрахунку середнього заробітку по варіанту "Для лікарняних листів".

Записи по складових частинах заробітку, що враховується, заповнюємо згідно з кількістю місяців, які беруть участь в розрахунку середнього. Це пов'язано з необхідністю враховувати межу, з якої платилися внески до фонду соціального страхування безробіття. Для визначення середньоденного (середньогодинного) заробітку підсумовуємо розрахований раніше заробіток, що враховується, і ділимо на відпрацьований час за розрахунковий період з ресурсу "Відпрацьовано днів" ("Відпрацьовано годин") по регістру "Основні нарахування працівників організації".

Якщо в базі відсутні дані для розрахунку середньою, їх можна ввести уручну і зафіксувати результат, встановивши прапор "Ручне виправлення розрахунку середнього". Аналогічним чином, встановивши прапор "Ручне виправлення % оплати", можна зафіксувати значення реквізиту "Відсоток оплати".

Важливо! У разі, коли працівник прийнятий на внутрішнє сумісницю в документі лікарняний повинні виникнути 2 записи по кожному виду розрахунку але з різними наказами про прийом (основному і по всіх внутрішніх поєднаннях). Це необхідно для забезпечення роботи механізму витіснень. Суми будуть розраховані тільки в записах по основному наказу,
решта записів повинна бути присутньою в документі формально.
