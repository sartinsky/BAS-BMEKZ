Документ "Начисление по больничному листу"

Документ "Начисление по больничному листу" регистрирует замену плановых начислений начислением оплаты по среднему заработку с вариантом расчета "Для больничных листов" указанному в документе работнику на указанный период.

В шапке документа указываются другие необходимые реквизиты, в частности, причина, процент оплаты по больничному (который зависит от длительности непрерывного трудового стажа работника).

Сумма начисления рассчитывается автоматически нажатием кнопки "Рассчитать". Результаты расчета отображаются в табличной части документа на закладке "Начисления", На закладке "Расчет среднего заработка" отображаются результаты расчета среднего заработка по варианту "Для больничных листов".

 Записи по составным частям учитываемого заработка заполняем согласно с количеством месяцев, которые участвуют в расчете среднего. Это связано с необходимостью учитывать предел, с которого платились взносы в фонд социального страхования безработицы. Для определения среднедневного (среднечасового) заработка суммируем рассчитанный ранее учитываемый заработок и делим на отработанное время за расчетный период из ресурса "Отработано дней" ("Отработано часов") по регистру "Основные начисления работников организации".

Если в базе отсутствуют данные для расчета средней, их можно ввести вручную и зафиксировать результат, установив флаг "Ручное исправление расчета среднего". Аналогичным образом, установив флаг "Ручное исправление % оплаты", можно зафиксировать значение реквизита "Процент оплаты".

Важно! В случае, когда работник принят на внутреннее совместительство в документе больничный должны возникнуть 2 записи по каждому виду расчета но с различными приказами о приеме (основному и по всем внутренним совмещениям). Это необходимо для обеспечения работы механизма вытеснений. Суммы будут рассчитаны только в записях по основному приказу, остальные записи должны присутствовать в документе формально.
