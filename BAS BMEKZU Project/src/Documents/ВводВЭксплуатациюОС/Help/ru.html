Документ "Ввод в эксплуатацию основных средств"

При вводе документа в шапке указываются следующие реквизиты:

  • Организация - организация, принимающая к учету ОС.
  • Событие - событие основного средства, которое отражается документом.
  • Подразделение - подразделение организации, в котором будут эксплуатироваться ОС.

На закладке Основные средства заполняется список принимаемых к учету основных средств (табличное поле Основные средства).

  • Для каждого основного средства в колонке Инв. № указывается инвентарный номер. При выборе основного средства инвентарный номер заполняется автоматически кодом основного средства из справочника.
  • Для быстрого заполнения документа группой однотипных основных средств нужно ввести в табличное поле хотя бы один такой объект и автоматически заполнить табличное поле основными средствами с таким же наименованием по кнопке Заполнить - По наименованию.

На закладке Учетные данные вводится информация, используемая для учета ОС и начисления их амортизации:

  • МОЛ - материально ответственное лицо.
  • Счет учета ОС - субсчет 10-го или 11-го счета.
  • Счет начисления амортизации - субсчет 13-го счета.
  • Начислять амортизацию - флаг заполняется для ОС, стоимость которых амортизируется; не заполняется для земельных участков.
  • Способ отражения расходов по амортизации - определяет один или несколько счетов затрат при амортизации ОС, а также аналитику затрат: подразделение, статью затрат и пр.
  • Метод начисления амортизации (БУ) - один из методов, предусмотренных НП(С)БУ «Основные средства». В зависимости от выбранного метода, вводится информация, необходимая для начисления амортизации:
    • Срок полезного использования ОС.
    • Ликвидационная стоимость ОС.
    • Предполагаемый объем продукции, вырабатываемой с использованием ОС (для производственного метода амортизации).
    • Параметр выработки - единица измерения предполагаемого объема продукции;
    • График амортизации ОС - определяет коэффициенты распределения амортизации объекта по месяцам в течение года; применяется для ОС, используемых в сезонном производстве.
  • Налоговое назначение ОС - указывается организациями-плательщиками НДС и определяет, используется ли ОС в облагаемых или необлагаемых НДС операциях.

Организации-плательщики налога на прибыль дополнительно указывают для производственных (подлежащих амортизации в НУ) ОС Налоговую группу ОС, Метод начисления амортизации (НУ) и Срок полезного использования (НУ).

Организации-плательщики налога на прибыль дополнительно указывают для непроизводственных ОС Налоговое назначение "необл. НДС, нехоз.". Таким образом определяется, что ОС используется в нехозяйственной деятельности и не амортизируется в НУ.

Принятие к учету оборудования (основных средств, не требующих монтажа)

На закладке Основные средства заполняются следующие реквизиты:

  • Оборудование - наименование номенклатурной карточки, на которой оборудование учитывается на складе.
  • Склад - место хранения оборудования.
  • Счет - счет учета оборудования на складе: 1521 или 1531.

Принятие к учету объектов строительства (монтажа)

На закладке Основные средства заполняются следующие реквизиты:

  • Объект - объект строительства, на котором аккумулировались затраты по строительству (монтажу) основных средств.
  • Счет - счет учета затрат на строительство основных средств: обычно указывается счет 151 при строительстве ОС, счет 1522 или 1532 при монтаже ОС.
  • Стоимость (БУ) - сумма затрат, понесенных при строительстве (монтаже). Это поле заполняется автоматически при нажатии кнопки Рассчитать сумму.

Для организаций-плательщиков налога на прибыль сумма затрат по данным бухгалтерского учета формирует также амортизируемую стоимость ОС в НУ.

Принятие к учету основных средств по результатам инвентаризации

На закладке Основные средства заполняются следующие реквизиты:

  • Счет и Статья доходов - статья прочих доходов, по которой будут отражаться доходы от безвозмездного получения выявленных в результате инвентаризации основных средств.
  • Стоимость (БУ) - стоимость выявленных в результате инвентаризации основных средств.
  • Стоимость (НУ) - стоимость, подлежащая амортизации в налоговом учете.
