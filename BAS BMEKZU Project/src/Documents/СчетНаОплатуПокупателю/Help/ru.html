Документ "Счет на оплату покупателю"

 

Счет на оплату можно распечатать в виде стандартной формы счета и отправить пользователю по факсу.

Для оформления оплаты по счету, на основании документа "Счет на оплату покупателю" можно выписать документ "Приходный кассовый ордер" в случае наличной оплаты или зафиксировать безналичную оплату с помощью документа Поступление на банковский счет.

Отгрузку товаров по счету можно оформить с помощью документа "Реализация товаров и услуг". При оформлении  документа реализации на основании счета, документ реализации заполняется всеми товарами, выписанными в счете, без учета ранее оформленных документов реализации.
