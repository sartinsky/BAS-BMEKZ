Документ "Изменение налогового назначения запасов"

Документ «Изменение налогового назначения запасов» служит для отражения операции изменения целевого назначения запасов для целей НДС. Эту операцию необходимо осуществлять перед использованием запасов по определенному налоговому назначению, когда при приходовании таких запасов было указано другое (целевое) налоговое назначение.

В некоторых случаях документ изменяет также и бухгалтерскую себестоимость запасов.

Документ проводит корректировки сумм налоговых обязательств по НДС в случаях, когда изменение целевого назначения запаса влияет на начисление налоговых обязательств по условной продаже.

Заполнять документ рекомендуется следующим образом. В табличную часть внести требуемую номенклатуру, указать количество и новое целевое назначение.

Дальше можно автоматически подобрать партии товаров. Это действие осуществляется по нажатию на кнопку "Заполнить по партиям". Далее необходимо проверить метод корректировки налогового кредита и сумму НДС.

Кнопка "Обновить суммы" позволяет перезаполнить суммы в строках табличной части.
