Документ "Реестр договоров контрагентов"

Данный документ предназначен для сверки договоров.  

 

Общие реквизиты документа

 

Программа ведется в разрезе договоров, применяемые для клиентов. Для сверки договоров по клиенту в документ нужно ввести реквизит «Контрагент».

Документ содержит табличную часть, в которой указываются: список договоров – элемент справочника "Договора Контрагентов".
