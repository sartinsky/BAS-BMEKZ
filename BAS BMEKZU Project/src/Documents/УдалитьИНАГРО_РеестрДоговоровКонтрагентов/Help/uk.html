Документ "Установка цен на услуги"

Данный документ предназначен для ввода в программу расценок на услуги ХПП.  

Общие реквизиты документа

В программе могут быть расценки, применяемые для преобладающего большинства клиентов, в таком случае в документе реквизиты «Контрагент» и «Договор» нужно оставить пустыми. Для внесения персональных расценок реквизиты «Контрагент» и «Договор» необходимо заполнить.

Документ содержит табличную часть, в которой указываются: вид культуры – элемент справочника «Виды культуры» для расчет стоимости услуг»; наименование услуги – элемент справочника «Номенклатура», для которой вводится цена. Автоматически показывается «старая цена», действовавшая на кануне изменения. Новая цена вводится в одноименном поле табличной части.

В нижней части формы расположены кнопка «Заполнить» - заполняет документ «старыми ценами».

Особенности проведения документа

После записи документ делает записи в регистре сведений «Цены на услуги».
