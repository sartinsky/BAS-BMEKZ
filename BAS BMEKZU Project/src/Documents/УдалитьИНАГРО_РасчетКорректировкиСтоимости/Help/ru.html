Документ «Расчет корректировки стоимости»

Документ «Расчет корректировки стоимости» предназначен для отражения корректировочных операций по поступлению и реализации товаров (биологических активов).

Вид операции, которая будет произведена в результате проведения документа, определяется непосредственно в самой форме документа при помощи  кнопки "Операция".

С видом операции «Входящие документы» производиться корректировка стоимости приобретенных товаров (биологических активов). Закладка «состав корректировки» заполняется автоматически номенклатурным составом из указанного документа для корректировки. В табличной части документа на закладке «состав корректировки» указываем новую цену, автоматически рассчитывается сумма корректировки. Закладка «данные распределения» автоматически заполняется движениями номенклатуры, по которой была изменена сумма стоимости. При проведении документа корректируются взаиморасчеты с контрагентами, остатки номенклатуры (биологических активов), движения по списанию.

С видом операции «Исходящие документы» производится корректировка количества и стоимости реализованных товаров (биологических активов). Закладка «состав корректировки» заполняется автоматически номенклатурным составом из указанного документа для корректировки. Указав признак «V» в поле  «расхождение количества», количество списанной с остатков номенклатуры не будет изменяться, а сумма взаиморасчетов с контрагентами изменится. С этим видом операции документ имеет печатную форму.
