Документ "Приказ на перемещение" 

 

  Документ «Приказ на перемещение» предназначен для оформления перемещения зерна между удалёнными складами элеватора с использованием автотранспорта.

  Заполнение экранной формы документа следует начать с выбора «Владельца» зерна, по которому оформляется перемещение, «Год урожая».

Реквизиты «со склада» и «на склад» - содержат в себе значения складов между которыми происходит перемещение В табличной части заполните следующие реквизиты:

«Наименование» - культура, которую вывозим;

  В автоматическом режиме заполняем данные соответствующего лабораторного анализа из списка документов «Лабораторный анализ»;

Для получения печатной формы приказа или пропуска нажмите кнопку «Печать».
