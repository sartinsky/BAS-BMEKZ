Документ "Корректировка услуг"

Документ «Корректировка услуг» предназначен для выполнения произвольных операций по корректировке выставленных услуг в подсистеме элеватор. Документ применяется для внесения изменений непосредственно в регистры хранения информации.

Особенности проведения документа

При проведении документа формируются движения по регистру "Расчеты по услугам".
