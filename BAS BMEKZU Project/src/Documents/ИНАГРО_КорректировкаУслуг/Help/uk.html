Документ "Коригування послуг"

Документ «Коригування послуг» призначений для виконання довільних операцій по коригуванню виставлених послуг у підсистемі елеватор. Документ застосовується для внесення змін безпосередньо в регістри зберігання інформації.

Особливості проведення документа:

При проведенні документа формуються руху по регістру "Розрахунки по послугах".
