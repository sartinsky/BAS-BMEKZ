Документ "Зважування"

Якщо на підприємстві є електронні чи механічні ваги, то є можливість надання додаткових послуг стороннім організаціям по зважуванню транспорту. Для оформлення разового зважування зерна та ведення обліку по наданню послуг стороннім особам в конфігурації існує документ «Квитанція на зважування».

Загальні реквізити документу

В документі необхідно вказати замовника, водія та транспорт, на якому був ввезений вантаж. В реквізиті «ТМЦ» зазначте найменування вантажу.

«Показання ваг» - дані вагової, а саме «Брутто», «Тара» для автомобіля - для причепу значення «Нетто»  та  «Всього» буде розраховано автоматично.

Особливості проведення документу

Не проводиться у регістри.
