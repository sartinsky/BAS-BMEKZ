Документ «Реализация коммунальных услуг»

Документ "Реализация коммунальных услуг" предназначен для фиксации факта реализации коммунальных услуг.

На закладке «Счета учета расчетов» в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений «Контрагенты организаций» значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.
