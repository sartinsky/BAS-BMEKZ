Документ "Акт фасовки"

Документ предназначен для отражения в программе операций фасовки готовой продукции. Особенностью документа является то, с его помощью можно расфасовать любое ТМЦ. Данный документ используется в комбикормовом производстве.

 

На закладке «Основные» расположены реквизиты:

·           «Склад» - место хранения, с которого будет выполнено списание товароматериальных ценностей;

·            «Владелец» - владелец списываемого сырья;

·           «Договор» - документ-основание для взаиморасчетов с клиентом;

·            «Урожай» и «Вид хранения» - списать с определенного урожая и  вида хранения;

Документ имеет две закладки: «Сырье» и «Продукция».

На закладке «Сырье» в табличной части необходимо указать перечень товароматериальных ценностей списываемых в производство, их количество и качество.

·           «Номенклатура», «Ед» - списываемая номенклатура, единица измерения;

·            «Количество, шт», «Вес, кг» - укажите количество мешков, вес сырья;

·           «Лабораторный анализ», «№ анализа», «Влажность», «Сорная примесь» - реквизиты лабораторного анализа, показатели качества;

·           «Зачетный вес» - зачетный вес партии сырья.

На закладке «Продукция» в табличной части В табличной части необходимо указать перечень наименований фасованной продукции, их количество и качество.

·           «Продукция», «Ед» - фасованная номенклатура, единица измерения;

·           «Фасовка» - указывается вид фасовки;

·            «К-во, шт», «Вес, кг» - укажите количество мешков, вес сырья;

·           «Лабораторный анализ», «№ анализа», «Влажность», «Сорная примесь» - реквизиты лабораторного анализа, показатели качества;

Для печати документа нажмите кнопку «Печать». Документ имеет несколько печатных форм: «Требование» (форма №119); «Накладная (форма №112)».

При проведении документа формируются движения по регистрам оперативного учета и проводки по счету 023 «Материальные ценности на ответственном хранении».
