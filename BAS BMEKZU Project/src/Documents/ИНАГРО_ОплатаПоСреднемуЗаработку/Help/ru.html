Документ «Оплата по среднему заработку»

Документ «Оплата по среднему заработку» регистрирует замену плановых начислений работника на определенный период начислением оплаты по среднему заработку с «обычным» вариантом расчета.

Внешний вид и порядок расчета определяется значение реквизита "Вид расчета": для видов расчета "Командировочные" и "Оплата по среднему" доступны три страницы "Начисления", "Расчет среднего заработка", "Удостоверение"; а для вида расчета "Доплата до среднего заработка" доступна только страница "Расчет среднего заработка".

Страница "Расчет среднего заработка"

Страница содержит табличную часть и дополнительные параметры, которые определяют порядок заполнения и расчета табличной части.

Важно! Если в регистре сведений "Параметры расчета заработной платы организации" для организации включен флаг "Есть годовая премия", то в группе "Дополнительные параметры расчета средней" станут видимыми и доступными реквизиты "Расчет премии за год", а при заполнении табличной части в нее будут добавлены записи с видом расчета "По годовой премии (для отпуска и средней)".
Если в том же регистре сведений включен флаг "Есть премия", в табличную часть будут также добавлены виды расчета "По премиям пропорционально (для средней)" и "По премиям (для средней)".

По результатам заполнения и расчета табличной части "Расчет среднего" рассчитывается "Сумма средней". Порядок расчета зависит от значения реквизита "Вид учета времени для средней":

  • По рабочим дням;
  • По рабочим часам;
  • По календарным дням.

    Страница "Начисления"

    После ручного расчета средней можно перезаполнить и рассчитать табличную часть "Начисления" (Меню "Заполнить", "Рассчитать" таблицы "Начисления"). 

    Для видов расчета "Командировочные" и видов расчета со способом расчета "По среднему заработку" в табличной части "Начисления" будут автоматически заполнены параметры "Средняя" и "Вид учета времени для средней", они заполняются данными со страницы "Расчет среднего заработка".

    Важно! В случае, когда сотрудник принят на внутреннее совместительство в документе для предопределенного вида расчета "Командировка" и прочих непредопределенных видов расчета со способом расчета "По среднему заработку" в таблице Начисления должны возникнуть 2 записи по каждому виду расчета, но с различными назначениями (основному и по всем внутренним совмещениям). Это необходимо для обеспечения работы механизма вытеснений. Суммы будут рассчитаны только в записях по основному назначению, остальные записи должны присутствовать в документе формально. А для вида расчета "Доплата до среднего" в шапке документа будет доступен еще один реквизит "Назначение", который определяет по какому назначению должна быть рассчитана средняя.

    Документ "Оплата по среднему заработку" может быть введен на основании документа "Замены организаций". В этом случае порядок работы следующий:

    1. Создается документ  "Замены организаций";
    2. На основании  "Замены организаций" вводится документ "Оплата по среднему заработку";
    3. В документе "Начисление зарплаты работникам организаций" при заполнении табличной части "Начисления" должна появиться запись с видом расчета "Доплата до среднего". В качестве параметров в этой записи должна заполниться средняя (рассчитанная на шаге 2).
