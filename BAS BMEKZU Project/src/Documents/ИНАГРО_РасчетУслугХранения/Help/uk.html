Документ "Розрахунок послуг зберігання"


Для нарахування кількості послуг зберігання по всім контрагентам, культурам і складам необхідно використовувати документ «Розрахунок послуг зберігання». Документ створюється окремо на кожен місяць.

Для розрахунку документа необхідно використовувати кнопку «Розрахувати».
