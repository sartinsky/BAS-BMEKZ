Документ "Лабораторный анализ"

Документ предназначен для регистрации в системе результатов работы лаборатории предприятия.  

Общие реквизиты документа

С помощью данного документа вводятся данные, полученные лабораторией при проведении анализа качественных показателей зерна. Для этого необходимо заполнить следующие реквизиты:

·           «Дата» - дата проведения лабораторного анализа;

·           «Владелец» - владелец зерновой продукции;

·           «Склад» - место хранения зерновой продукции;

·           «Номенклатура» - вид зерновой продукции, взятой для проведения анализа;

·           «Урожай» - имеет вид года сбора урожая данной зерновой продукции;

·           «Вид хранения» - вид хранения, на который поступает данная партия зерна;

·            «Вес образца» - вес образца, который был взят для проведения лабораторного анализа;

·            «Метод определения стекловидности», «Происхождение», «Клас», «Сорт», «Склад/Силос» - данные этих реквизитов переносятся в печатную форму документа.

В реквизите «Журнал» - можно определить какому журналу будет принадлежать лабораторный анализ. Список лабораторных журналов создается в справочнике «Лабораторные журналы». В таком случае нумерация лабораторных анализов для каждого журнала будет своя.

В табличной части документа укажите перечень исследуемых характеристик зерна с полученными значениями, например: «Влажность», «Сорность», «Зерновая примесь» и др.

С помощью кнопки «Заполнить» пользователь может автоматически вывести в табличную часть документа виды характеристик зерна, которые были указаны в карточке соответствующему виду культуры.

В документе необходимо определить будет ли проводиться очистка или сушка, установив соответствующие флажки. Флажок «Сушить активным вентилированием» служит уточнением метода сушки.

Формирование печатной формы, после ввода необходимых параметров, происходит нажатием на кнопку «Печать». Пользователю будет предложено две печатные формы: «Карточка» и «Каточка анализа зерна (форма №47)»:.

 

Особенности проведения документа

При проведении документа формируются движения по регистрам оперативного учета.
