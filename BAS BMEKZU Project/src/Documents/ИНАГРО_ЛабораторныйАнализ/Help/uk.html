Документ "Лабораторний аналіз"

Документ призначений для реєстрації в системі результатів роботи лабораторії підприємства.  

Загальні реквізити документу

За допомогою даного документу вводяться дані, отримані лабораторією при проведенні аналізу якісних показників зерна. Для цього необхідно заповнити наступні реквізити:

·           «Дата» - дата проведення лабораторного аналізу;

·           «Власник» - власник зернової продукції;

·           «Склад» - місце зберігання зернової продукції;

·           «Номенклатура» - вид зернової продукції, взятої для проведення аналізу;

·           «Врожай» - моє вид року збору врожаю даної зернової продукції;

·           «Вид зберігання» - вид зберігання, на який надходить дана партія зерна;

·            «Вага зразка» - вага зразка, взятого для проведення лабораторного аналізу;

·            «Метод визначення скловидності», «Походження», «Клас», «Сорт», «Склад/Силос» - дані цих реквізитів переносяться в друковану форму документу.

В реквізиті «Журнал» - можна визначити, якому журналу буде належати лабораторний аналіз. Список лабораторних журналів створюється в довіднику «Лабораторні журнали». У такому випадку нумерація лабораторних аналізів для кожного журналу буде своя.

В табличній частині документу вкажіть перелік досліджуваних характеристик зерна з отриманими значеннями, наприклад: «Вологість», «Засміченість», «Зернова домішка» та ін.

За допомогою кнопки «Заповнити» користувач може автоматично вивести в табличну частину документу види характеристик зерна, які були вказані у картці відповідного виду культури.

У документі необхідно визначити, чи буде проводитися очищення чи сушіння, встановивши відповідні прапорці. Прапорець «Сушити активним вентилюванням» слугує для уточнення методу сушки.

Формування друкованої форми після вводу необхідних параметрів відбувається натисканням на кнопку «Друк». Користувачу буде запропоновано дві друковані форми: «Картка» та «Картка аналізу зерна (форма №47)».

 

Особливості проведення документу

При проведенні документу формуються рухи по регістрах оперативного обліку.
