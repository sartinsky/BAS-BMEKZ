Документ "Акт распределения отходов"

Документ позволяет распределить полученные в результате работы с зерном аспирационные отходы, отходы

3-ей категории между владельцами.

В табличной части документа указывается перечень культур, владельцев, между которыми необходимо распределить отходы. При помощи кнопки "Заполнить" документ можно заполнить автоматически данными количества ввезенного, вывезенного зерна, количества просушенного зерна.

 

Общие реквизиты документа

 Для этого необходимо заполнить следующие реквизиты:

·           «Склад» - внутренний склад элеватора, на котором хранится зерно;

·           «Вид хранения» - вид хранения, на котором хранится зерно;

·           «Урожай» - имеет вид года сбора урожая данной зерновой продукции;

·           «За период» - указать период, задаётся период, за который происходило накопление отходов.

 

На закладке «Расчёты по владельцам» указывается вид отходов, их вес (полученный в результате взвешивания) и показатели качества.

Флаг «списать зачётный вес» - служит для того чтобы на вес распределённых отходов не насчитывалась оплата за хранение.

 

На закладке «Дополнительно» необходимо заполнить реквизиты касательно корректного вывода печатных форм
