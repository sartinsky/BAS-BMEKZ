Документ «Зміна графіків амортизації ОЗ»

Документ "Зміна графіків амортизації ОЗ" використовується для зміни річного графіка амортизації для основних засобів, для яких для нарахування амортизації використовується лінійний спосіб.

Об'єкти, для яких встановлюється графік амортизації, вибираються з довідника "Основні засоби".
Для більш швидкого заповнення табличної частини документа необхідно скористатися кнопкою "Підбір".

При проведенні документа ці відомості записуються в регістр і використовуються документом "Закрытие месяца" при нарахуванні амортизації у місяці, наступному за місяцем зміни графіка розподілу амортизації.

Даний документ не формує ніяких проводок, він лише змінює значення відповідних ресурсів регістра відомостей ("Графіки амортизації бухгалтерський облік").

Для зміни інших характеристик обліку основних засобів (наприклад - способу відображення витрат по амортизації, графіків нарахування амортизації тощо) використовуються інші документи - "Зміна параметрів нарахування амортизації ОЗ", "Зміна способів відображення витрат по амортизації" ...

Для швидкого заповнення табличної частини однотипними об'єктами основних засобів, що мають однакові найменування, потрібно ввести в табличну частину хоча б один такий об'єкт. Потім слід скористатися кнопкою Заповнити у командній панелі табличної частини. При виборі в меню пункту По найменуванню таблична частина буде заповнена об'єктами основних засобів, мають таке ж найменування, як у введеного спочатку.
