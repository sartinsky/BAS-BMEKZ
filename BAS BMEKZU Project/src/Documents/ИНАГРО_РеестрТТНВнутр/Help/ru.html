Документ "Реестр ТТН (внутренний)"

Основанием для формирования документа  «Реестр ТТН внутренний» являются введенные в систему документы «ТТН внутренние». Документ предназначен для ввода в программу данных о перемещении зерна между складами предприятия.

Общие реквизиты документа

Реестры группируются по признакам: склад, на котором хранилось зерно, склад на который перемещаем, культура, дата, вид урожая, вид хранения.

Флажок «перевес» необходимо установить в случае когда проводится инвентаризация по предприятию и необходимо сделать перевес остатков.

На закладке «Основные» указываются параметры перемещаемой партии: склады между которыми осуществляется перемещение, культура, урожай, вид хранения, при автоматическом формировании эти реквизиты будут заполнены. В реквизите «Вес нетто» указывается общий вес по накладным.

На закладке «Список ТТН» отображается перечень товарно-транспортных накладных которые вошли в этот реестр. В реквизите «Анализ» необходимо указать соответствующий лабораторный анализ из списка, из анализа автоматически подставляется номер анализа в реквизит «№ анализа» и качественные показатели «Влажность», «Сор. Примесь», «Зерн. Примесь», эти реквизиты можно указать и вручную. При автоматическом заполнении данные лабораторного анализа переносятся с товарно-транспортных накладных.

На закладке «Дополнительно» добавлены реквизиты Члены комиссии, которые будут выводится в печатную форму.

При нажатии на кнопку «Печать» Вы получите несколько печатных форм: Приказ на перемещение, Накладная (ф.19), Ведомость взвешиваний (ф.171-б).

Особенности проведения документа

При проведении документа в бухгалтерском учете будут сформированы проводки по 023 счету «Материальные ценности на ответственном хранении». В оперативном учете записывается движение по регистрам.
