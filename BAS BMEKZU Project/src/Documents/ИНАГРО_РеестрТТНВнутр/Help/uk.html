Документ "Реєстр ТТН (внутрішній)"

Підставою для формування документу  «Реєстр ТТН внутрішній» являються введені в систему документи «ТТН внутрішні». Документ призначений для введення в програму даних про переміщення зерна між складами підприємства.

Загальні реквізити документу

Реєстри групуються по ознакам: склад, на якому зберігалося зерно, склад на який переміщуємо, культура, дата, вид врожаю, вид зберігання.

Прапорець «перезважування» необхідно встановлювати у тому випадку, коли проводиться інвентаризація по підприємству і необхідно зробити перезважування залишків.

На закладці «Основні» зазначаються параметри партії, що переміщується: склади, між якими здійснюється переміщення, культура, врожай, вид зберігання, при автоматичному формуванні ці реквізити будуть заповнені. В реквізиті «Вага нетто» вказується загальна вага по накладним.

На закладці «Список ТТН» відображається перелік товарно-транспортних накладних, які увійшли в щей реєстр. В реквізиті «Аналіз» зазначається відповідний лабораторний аналіз із списку, із аналізу автоматично підставляється номер аналізу в реквізит «№ аналізу» та якісні показники «Вологість», «Смітна домішка», «Зернова домішка», ці реквізити можна вказати і вручну.

При автоматичному заповненні дані лабораторного аналізу переносяться з товарно-транспортних накладних.

На закладці «Додатково» додані реквізити Члени комісії, які будуть виводитися в друковану форму.

При натисканні на кнопку «Друк» Ви отримаєте декілька друкованих форм: Наказ на переміщення, Накладна (ф.19), Відомість зважувань(ф.171-б).

Особливості проведення документу

При проведенні документу в бухгалтерському обліку будут сформовані проводки по 023 рахунку «Матеріальні цінності на відповідальному зберіганні». В оперативному обліку записуються рухи документу по регістрах.
