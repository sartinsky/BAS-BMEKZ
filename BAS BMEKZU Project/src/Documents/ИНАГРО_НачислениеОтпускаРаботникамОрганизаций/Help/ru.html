Документ "Начисление отпуска работникам организаций"

Документ "Начисление отпуска работникам организации"  регистрирует замену плановых начислений определенного работника на определенный период начислением оплаты по среднему заработку с вариантом расчета "Для оплаты отпусков". Документ также можно использовать для расчета компенсации отпуска.

Сведения, необходимые для расчета начислений, указываются в шапке документа. Сумма начисления определяется автоматически и отображается в табличной части "Начисления". Промежуточные данные расчета отображаются на закладке "Расчет среднего заработка" — в зависимости от порядка расчета.

Некоторые особенности расчета среднего заработка для оплаты отпуска:-

- Записи по составным частям учитываемого заработка, кроме годовой премии, вводим помесячно, проставляя им в качестве базового месяцы расчетного периода с учетом периодов индексации заработка. При расчете записей по годовой премии учитываются только те записи о премиях из регистра "Дополнительные начисления работникам организации", базовый период которых соответствует календарному году записи регистра среднего заработка;

- Запись по годовой премии вводим с базовым периодом, совпадающим с предшествующим началу события календарным годом;

- Результаты записей, составляющих учитываемый заработок, рассчитываются с учетом коэффициентов увеличения окладов. При расчете записей по премиям учитывается количество месяцев, по итогам работы за которые была начислена каждая конкретная премия.

- Для определения среднедневного заработка суммируем рассчитанный ранее учитываемый заработок и делим на подсчитанное ранее "Календарные дни" время. Внимание для правильной работы необходимо, чтобы был рассчитан регистр "Регламентированный производственный календарь", после чего необходимо рассчитать графики. Необходимо убедится, что "Календарные дни" в графиках заполнены;

- Количество дней отпуска определяем по указанному графику отпуска по "календарным дням". В документе может быть указан процент оплаты от среднего заработка.
