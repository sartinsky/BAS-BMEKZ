Документ "Нарахування відпустки робітникам організацій"

Документ "Нарахування відпустки робітникам організації"  реєструє заміну планових нарахувань певного робітника на певний період нарахуванням оплати по середньому заробітку з варіантом розрахунку "Для оплати відпусток". Документ також можна використовувати для розрахунку компенсації відпустки.

Відомості, необхідні для розрахунку нарахувань, вказуються в шапці документа. Сума нарахування визначається автоматично і відображається в табличній частині "Нарахування". Проміжні дані розрахунку відображаються на закладці "Розрахунок середнього заробітку" — залежно від порядку розрахунку.

Деякі особливості розрахунку середнього заробітку для оплати отпуска:

  • Записи по складових частинах заробітку, що враховується, окрім річної премії, вводимо щомісячно, проставляючи їм в якості базового місяці розрахункового періоду з урахуванням періодів індексації заробітку. При розрахунку записів по річній премії враховуються тільки ті записи про премії з регістра "Додаткові нарахування працівникам організації", базовий період яких відповідає календарному року запису регістра середнього заробітку;

  • Запис по річній премії вводимо з базовим періодом, співпадаючим з  совпадающим с передуючим початку події календарним роком;

  • Результати записів, що складають враховуємий заробіток, що враховується, розраховуються з урахуванням коефіцієнтів збільшення окладів. При розрахунку записів по преміях враховується кількість місяців, за підсумками роботи за які була нарахована кожна конкретна премія.

  • Для визначення середньоденного заробітку підсумовуємо розрахований раніше заробіток, що враховується, і ділимо на підраховане раніше "Календарні дні". Увага! Для правильної роботи необхідно, щоб був розрахований регістр "Регламентований виробничий календар", після чого необхідно розрахувати графіки. Необхідно переконатися, що "Календарні дні" в графіках заповнені;

  • Кількість днів відпустки визначаємо по вказаному графіку відпустки по "календарних днях". У документі може бути вказаний відсоток оплати від середнього заробітку.
