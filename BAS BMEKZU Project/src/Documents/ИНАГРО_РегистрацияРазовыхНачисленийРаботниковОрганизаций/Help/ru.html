Документ «Регистрация разовых начислений работников организаций»

Документ позволяет регистрировать разовые начисления, которые вводятся фиксированной суммой.

Для упрощения и ускорения ввода данных о начислениях рекомендуется заполнить документ списком работников с необходимыми отборами (кнопка «Заполнить»). При этом табличная часть начислений будет автоматически заполнена списком работников, указанным периодом, базовым периодом, видом расчета и суммой.
