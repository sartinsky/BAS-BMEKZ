Документ "Переробка"

Документ служить для одночасного списання сировини та оприбуткування з виробництва готової продукції. Особливістю документа є те, що довільне число найменувань ТМЦ  може як списуватися (йти в переробку) так і оприбутковуватися Даний документ активно використовується в борошномельному виробництві.

Загальні реквізити документа

У шапці документа необхідно заповнити такі реквізити:

·           «Склад» - місце зберігання, з якого буде виконано списання товарно-матеріальних цінностей;

·            «Власник» - власник сировини, яка списується;

·           «Договір» - документ-підстава для взаєморозрахунків з клієнтом;

·            «Урожай» і «Вид Зберігання» - списати з певного урожаю і виду зберігання;

Документ включає функціонал попередніх документів «Акт списання», «Прихід з виробництва». Документ має дві закладки: «Сировина» й «Продукція», у яких необхідно заповнювати відповідно сировину, яка надходить у переробку, і продукцію переробки.

Для того щоб заповнити документ за допомогою підбору натисніть кнопку «Підбір». Документ можна заповнити автоматично за даними рецепта. Для цього необхідно натиснути кнопку «Заповнити/Заповнити за рецептами». Для того щоб заповнити документ по залишках сировини натисніть кнопку «Заповнити/Заповнити залишками».

Для друку документа натисніть кнопку «Друк». Документ має декілька друкованих форм: «Вимога» (форма №119); "Накладна (форма №110)"; «Розпорядження (форма №109)»; «Забірна карта» (форма №111); «Рапорт»; «Накладна (форма №112)»; «Накладна форма №115».

Особливості проведення документа

При проведенні документа формуються рухи по регістрах оперативного обліку та проводки по рахунку 023 «Матеріальні цінності на відповідальному зберіганні».
