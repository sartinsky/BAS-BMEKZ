Документ «Переработка»

Документ служит для одновременного списания сырья и оприходования из производства готовой продукции. Особенностью документа является то, что произвольное число наименований ТМЦ  может как списываться (идти в переработки)  так и оприходоваться. Данный документ активно используется в мукомольном производстве.

 

Документ включает функционал предыдущих документов «Акт списания», «Приход из производства». Документ имеет две закладки: «Сырье» и «Продукция», в которых необходимо заполнять соответственно сырье, которое поступает в переработку, и продукцию переработки.

Для того чтобы заполнить документ с помощью подбора нажмите кнопку «Подбор». Документ можно заполнить автоматически по данным рецепта. Для этого необходимо нажать кнопку «Заполнить/Заполнить по рецепту». Для того чтобы заполнить документ по остаткам сырья нажмите кнопку «Заполнить/Заполнить по остаткам».

  

Для печати документа нажмите кнопку «Печать». Документ имеет несколько печатных форм: «Требование» (форма №119); «Накладная» (форма №110); «Распоряжение (форма №109)»; «Заборная карта» (форма №111); «Рапорт»; «Накладная (форма 112)»; «Накладная форма115».

При проведении документа формируются движения по регистрам оперативного учета и проводки по счету 023 «Материальные ценности на ответственном хранении».
