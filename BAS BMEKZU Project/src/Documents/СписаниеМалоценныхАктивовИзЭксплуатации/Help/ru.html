Документ "Списание малоценных активов из эксплуатации"

Документ предназначен для отражения в учете операций списания из эксплуатации малоценки.

При вводе документа в шапке необходимо указать следующие реквизиты:

  • Подразделение. Подразделение организации, из которого списывается из эксплуатации малоценка.

В табличной части необходимо указать список списываемых активов и количество. При этом заполняются следующие реквизиты:

  • Малоценный актив. Списываемая малоценка. Значение реквизита выбирается из справочника "Номенклатура".

  • Назначение использования. Определяет счета учета малоценного актива в эксплуатации.

  • МОЛ. Материально ответственное лицо, с которого списывается малоценка.Прдусмотрено два режима выбора МОЛа, в зависимости от настройки параметров учета Учет расчетов по заработной плате и кадровый учет ведутся (закладка Сотрудники и зарплата):

    • Если зарплата и кадровый учет ведутся В этой программе, выбор МОЛа производится из сотрудников организации, по которой оформляется передача в эксплуатацию.

    • Если зарплата и кадровый учет ведутся Во внешней программе, выбор МОЛа производится из общего списка физических лиц.

  • Количество. Количество списываемой малоценки.

    Документ может быть автоматически заполнен:

    • По остаткам - заполняется всеми малоценными активами, находящимися в эксплуатации в подразделении.
    • По остаткам с истекшим сроком - заполняется всеми малоценными активами, находящимися в эксплуатации в подразделении, срок полезного использования которых истек.
    • По остаткам у МОЛ - заполняется всеми малоценными активами, находящимися в подотчете у выбранного материально ответственного лица.

    Если списание производится ранее истечения срока полезного использования, то для корректировки НДС можно ввести на основании данного документа налоговую накладную.

    Особенности проведения документа

    В бухгалтерском учете формируется проводка по списанию со счета МЦ.

    Для необоротных активов списывается первоначальная стоимость и накопленная амортизация.
