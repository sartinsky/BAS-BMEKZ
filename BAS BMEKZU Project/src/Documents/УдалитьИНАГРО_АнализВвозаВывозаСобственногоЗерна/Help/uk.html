Документ "Аналіз ввезення та вивезення власного зерна"

 

Для аналізу ввезення (операція «Покупка») / вивезення (операція «Реалізація») власного зерна необхідно використовувати документ «Аналіз ввезення вивезення власного зерна».

У шапці документа необхідно заповнити такі реквізити:

·          «Організація», «Склад» - організація і склад, за якими будуть відібрані документи;

·          «Постачальник» / «Покупець», «Договір» - вказується контрагент і договір із ним.

 

На закладці «Покупка зерна» / «Продаж зерна» заповнюється (кнопка «Заповнити по залишках») перелік документів, за якими оприбутковуються / реалізуються ті чи інші культури із зазначенням їх ціни, фізичного та залікової ваги по кожній накладній.

 

На закладці «Додатково» вказується, чи слід відкривати форму надходження / реалізації і формувати реєстрацію вхідного податкового документу / податкову накладну.

На закладці «Підлеглі документи» вказується перелік документів надходження / реалізації, сформований при натисканні кнопки «Оформити надходження» / «Оформити реалізацію».
