Документ «Заявление - расчет с ФСС»

Документ предназначен для оформления заявки - расчета в ФСС (фонд социального страхования) для получения выплат за счет ФСС.

Заполнение полей документа. В соответствующих полях документа указываются:

  • Организация (по умолчанию это организация, указанная в настройках пользователя)
  • Ответственный (по умолчанию это ответственный, указанный в настройках пользователя)
  • Месяц начисления (по умолчанию это месяц рабочей даты)

В табличной части документа указываются:

  • документ (вводяться документы "Начисление по больничному листу" .)

Автоматическое заполнение. Меню «Заполнить по начислениям» позволяет автоматически занести в табличную часть документа документы"Начисление по больничному листу"  за выбранный период (Месяц начисления), по которым еще не было движений  в регистре "Взаиморасчеты по начислениям за счет ФСС".

Расчет сумм происходит автоматически при добавлении или изменении строки с табличной части "Начисления".

Проведение документа. Рассчитанные в документе суммы отражаются:

  • во взаиморасчетах по начислениям за счет ФСС, соответствующего дате документа
  • во взаиморасчетах с работником месяца, соответствующего дате документа

Ввод на основании. На основании документа можно ввести документ "Платежное поручение входящее".
