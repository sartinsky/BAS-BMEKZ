Документ "Зміна податкового призначення ТЗВ"

Документ «Зміна податкового призначення ТЗВ» служить для відображення факту зміни цільового призначення транспортно-заготівельних витрат в обліку ПДВ.

Документ проводить коригування сума податкових зобов'язань з ПДВ у випадках, коли цільове податкове призначення ТЗВ при вступі відрізнялося від фактичного податкового призначення при їх розподілі й така зміна впливає на можливість віднесення сум затарат на право включення сум сплаченого постачальнику ПДВ в податковий кредит.

Документ вводиться наприкінці періоду, після проведення документа "Закриття періоду".

Суми коригування необхідно вводити вручну.
