Документ "Передача товаров"

Документ предназначен для отражения операций по передаче собственных материалов стороннему переработчику для выпуска продукции и передачи заказчику продукции, выпущенной из давальческого сырья. 

В заголовке документа указывается Контрагент, которому передаются запасы, и Склад, с которого передаются запасы.

Если запасы передаются в возвратной таре, на закладке Возвратная тара указываются наименования и количество тары, а также счета расчетов по таре. Если запасы передаются без тары, документ не формирует расчетов с контрагентами.

Передача сырья в переработку

Для отражения передачи сырья в переработку вводится документ вида Передача сырья в переработку.

На закладке Товары указывается номенклатура и количество переданных материалов, а также счета учета

  • Счет учета - счет учета сырья на складе
  • Счет передачи - счет учета сырья в переработке. Как правило, указывается счет 206

Передача продукции заказчику

Для документального оформления передачи продукции заказчику, не совпадающей во времени с подписанием акта переработки вводится документ вида Передача продукции заказчику.

На закладке Товары указывается номенклатура и количество переданной продукции.

Документ не формирует бухгалтерских проводок и вводится только для формирования печатной формы. Кроме документа Передача товаров (или вместо него) должен быть введен документ Реализация услуг по переработке.
