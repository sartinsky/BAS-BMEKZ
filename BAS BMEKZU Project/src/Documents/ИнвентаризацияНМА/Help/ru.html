Документ "Инвентаризация НМА"

Документ предназначен для отражения результатов инвентаризации НМА и сравнения учетных данных и фактического наличия НМА.

Данные учета заполняются по данным информационной базы при помощи кнопки «Заполнить».

Выявленные в ходе инвентаризации объекты НМА можно добавить в табличную часть при помощи кнопки «Добавить» командной панели.

Для НМА, по которым не заполнены данные учета, можно ввести фактическую стоимость.

Фактическая стоимость НМА, находящихся на учете, равна учетной. Изменять ее нельзя.
