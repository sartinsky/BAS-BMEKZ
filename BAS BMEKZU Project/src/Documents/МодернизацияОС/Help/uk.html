Документ "Модернізація і ремонт ОЗ"

Документ "Модернізація і ремонт ОЗ" призначений для відображення в обліку модернізації (реконструкції, капітального ремонту) і поточного ремонту основних засобів для цілей бухгалтерського і податкового обліку.

У шапці документа вносяться відомості про об'єкт будівництва, з якого списуються витрати на поліпшення, рахунок обліку витрат, а також сума поліпшення.

Суму поліпшення можна отримати автоматично - при натисканні кнопки "Розрахувати суми" сума заповнюється автоматично за даними обліку на момент формування документа. При необхідності суму можна відкоригувати вручну.

В шапці вказується Вид поліпшення, Модернізація або Ремонт. Від Виду поліпшення залежить, чи буде сума витрат на поліпшення віднесена на вартість ОЗ або на витрати.

У табличній частині "Основні засоби" зазначаються об'єкти основних засобів, які модернізуються (реконструюються, ремонтуються).

Для більш швидкого заповнення табличної частини документа необхідно скористатися кнопкою "Підбір".

Після формування списку основних засобів необхідно скористатися кнопкою "Заповнити" для заповнення табличної частини інформацією про об'єкти основних засобів за даними обліку, що існують на момент введення документа, а також для розподілу сум модернізації.

Для швидкого заповнення табличної частини однотипними об'єктами основних засобів, що мають однакові найменування, потрібно ввести в табличну частину хоча б один такий об'єкт. Потім слід скористатися кнопкою Заповнити у командній панелі табличної частини. При виборі в меню пункту По найменуванню таблична частина буде заповнена об'єктами основних засобів, мають таке ж найменування, як у введеного спочатку.

При автоматичному заповненні суми поліпшення розподіляються рівними частками по всіх об'єктах, перерахованих у табличній частині.

Якщо модернізуються об'єкти, для яких амортизація нараховується виробничим методом, то необхідно вказати загальний обсяг випуску продукції (робіт, послуг) з урахуванням модернізації (реконструкції) об'єктів.

При проведенні документа ці відомості записуються в регістр і використовуються документом "Закриття місяця" при нарахуванні амортизації у місяці, наступному за місяцем відображення модернізації.
