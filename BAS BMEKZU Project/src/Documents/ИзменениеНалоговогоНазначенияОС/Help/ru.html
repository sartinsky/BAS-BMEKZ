Документ "Изменение налогового назначения ОС"

Документ "Изменение налогового назначения ОС" используется для указания нового налогового назначения по НДС, в  котором будет задействовано основное средство.

Обрабатываемые объекты выбираются из справочника "Основные средства".

Для каждого объекта основных средств указывается текущая балансовая стоимость в бухгалтерском и налоговом  учете (налога на прибыль)

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор".

После формирования списка основных средств необходимо воспользоваться кнопкой "Заполнить" для заполнения табличной части  информацией об объектах основных средств по данным учета.

 

По факту изменения налогового назначения ОС могут корректироваться налоговые обязательства.

Необходимость начисления налоговых обязательств (кредита) указывается в поле Метод корректировки, сумма НО (НК) - в поле Сумма НДС.
