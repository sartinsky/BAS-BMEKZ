Документ "Принятие к учету НМА"

Документ «Принятие к учету НМА» предназначен для отражения принятия нематериальных активов  к бухгалтерскому и налоговому учету.

Принятие НМА к учету означает, что формирование стоимости объекта окончено и начинается ее амортизация.

При вводе документа на закладке Нематериальный автив заполняется

  • Нематериальный актив, принимаемый к учету.
  • Способ отражения расходов по амортизации - определяет один или несколько счетов затрат при амортизации НМА, а также аналитику затрат: подразделение, статью затрат и пр.

На закладке Учетные данные вводится информация, используемая для учета НМА и начисления их амортизации:

  • МОЛ - материально ответственное лицо.
  • Счет учета  - субсчет 12-го счета.
  • Счет начисления амортизации - субсчет 13-го счета.
  • Начислять амортизацию - флаг заполняется для НМА, стоимость которых амортизируется.
  • Метод начисления амортизации - один из методов, предусмотренных НП(С)БУ «Нематериальные активы». В зависимости от выбранного метода, вводится информация, необходимая для начисления амортизации:
    • Срок полезного использования ОС.
    • Ликвидационная стоимость ОС.
    • Предполагаемый объем продукции, вырабатываемой с использованием ОС (для производственного метода амортизации).
    • Параметр выработки - единица измерения предполагаемого объема продукции;
  • Первоначальная стоимость - стоимость объекта, сформированная при приобретении (разработке) НМА. Стоимость объекта может быть определена автоматически при нажатии кнопки Рассчитать суммы.

Организации-плательщики налога на прибыль дополнительно указывают для производственных (подлежащих амортизации в НУ) НМА

  • Налоговую группу НМА.
  • Первоначальная стоимость (НУ) - стоимость НМА, подлежащая амортизации для целей налогового учета. Эта стоимость также может быть определена автоматически при нажатии кнопки Рассчитать суммы.
  • Срок полезного использования (НУ)
  • Способ амортизации (НУ), а также параметры начисления амортизации для целей налогового учета.

Принятие к учету приобретенного НМА

В заголовке документа выбирается Вид операции Необоротные активы.

На закладке Нематериальный актив указывается Счет учета инвестиций. Это тот же  счет, который был указан в документе Поступление НМА. Рекомендуется использовать счет 1541.

Принятие к учету НМА собственной разработки

В заголовке документа выбирается Вид операции Объекты строительства.

На закладке Нематериальный актив заполняются следующие реквизиты:

  • Объект строительств - объект, на котором аккумулировались затраты на разработку НМА.
  • Счет инвестиций - счет учета затрат на разработку: обычно указывается счет 1542.
