Документ "Встановлення параметрів видів культур"

Спеціальний документ призначений для введення в програму базисів для розрахунку залікової ваги (вологість, смітна домішка, зернова домішка) і вартості послуг (вологість для розрахунку вартості послуг, смітна домішка для розрахунку вартості послуг).

Загальні реквізити документа

Документ містить табличну частину, в якій зазначаються: вид культури – елемент довідника «Види культур», для якої вводяться базиси. По кнопці «Заповнити» таблична частина заповниться «старими показниками базисів».

Особливості проведення документа

Після запису, документ робить записи в регістрі відомостей «Значення базисів»
