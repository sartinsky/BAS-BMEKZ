Документ "Установка параметров видов культур

Специальный документ предназначен для ввода в программу базисов для расчета зачетного веса (влажность, сорная примесь, зерновая примесь) и стоимости услуг (влажность для расчета стоимости услуг, сорная примесь для расчета стоимости услуг).  

Общие реквизиты документа

Документ содержит табличную часть, в которой указываются: вид культуры – элемент справочника «Виды культур», для которой вводятся базисы. По кнопке «Заполнить» табличная часть заполнится «старыми показателями базисов».

Особенности проведения документа

После записи документ делает записи в регистре сведений «Значения базисов»
