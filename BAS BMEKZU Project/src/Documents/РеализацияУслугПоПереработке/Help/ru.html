Документ "Реализация услуг по переработке"

Документ предназначен для отражения операций по переработке давальческого сырья. Документ можно ввести на основании документов Отчет производства за смену, Требование-накладная.

Оказание услуг по переработке

Для отражения оказания услуг по переработке материалов заказчика нужно заполнить закладку Продукция (услуги по переработке). На этой закладке указывается:

  • Продукция - наименование выпущенной из сырья заказчика продукции (оказание услуг по переработке такого сырья признается этим документом).
  • Сумма (плановая) - плановая себестоимость выпущенной из сырья заказчика продукции.
  • Спецификация - список нормативов расходов, необходимых для выпуска продукции. Значение поля Спецификация будет использоваться при заполнении закладки Материалы заказчика.

На стоимость услуг формируется задолженность давальца. Счета учета расчетов с давальцем и налоговых обязательств указываются на закладке Расчеты.

Отчет об использованных материалах заказчика

Для отражения отчета перед заказчиком об использованных в переработке материалах нужно заполнить закладку Материалы заказчика. На этой закладке указывается:

  • Счет учета - счет учета материалов заказчика, списанных в производство. Указывается счет 0222.

Закладка Материалы заполняется автоматически

  • Заполнить - Заполнить по спецификации - по данным колонки Спецификации на закладке Продукция
  • Заполнить - Заполнить по остаткам счета 0222 - по остаткам счета 0222 по указанному контрагенту.
