Документ "Инвентаризация"

Документ содержит информацию о результатах проведенной на предприятии инвентаризации зерна в разрезе владельцев.

 

Общие реквизиты документа

С помощью данного документа вводятся данные, полученные лабораторией при проведении анализа качественных показателей зерна. Для этого необходимо заполнить следующие реквизиты:

  ·           «Склад» - место хранения зерновой продукции;

·           «Культура» - вид зерновой продукции, взятой для проведения анализа;

·           «Урожай» - имеет вид года сбора урожая данной зерновой продукции;

·           «Вид хранения» - вид хранения, на который поступает данная партия зерна;

·           «Дата начала, окончание инвентаризации» - указать период проведения инвентаризации.
