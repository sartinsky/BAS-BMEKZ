Документ "Додаток 2 до податкової накладної"

Для реєстрації зміни суми компенсації після постачання призначений документ «Додаток №2 до податкової накладної». Документ формує регламентовану друковану форму Розрахунок коригування, а також реєструється в ЄРПН.

У заголовку документа повинна бути вказана Податкова накладна, коригування за якою формується.

У документах автоматично заповнюється прапор Вимагає реєстрації в ЄРПН. Після реєстрації заповнюється вручну прапор Включено до ЄРПН. При використанні FREDO Звіт, прапор Включено до ЄРПН заповнюється автоматично по кнопці Вивантаження - Вивантажити і перевірити стан реєстрації в "FREDO Звіт".

Якщо Додаток 2 реєструється пізніше, ніж коригуються податкові зобов'язання, і повинен бути включений в уточнюючий розрахунок, в ньому заповнюється прапор Включається до уточн. розрахунку.

На закладках Товари, Послуги, Основні засоби, Нематеріальні активи вносяться коригування кількості та цін постачання. В одному рядку може бути відображено коригування або кількості, або ціни. Для кожної рядки заповнюються реквізити, які вказані в регламентованій формі документа, а також Стаття декларації - рядок Декларації з ПДВ, в якій відображаються коригування податкових зобов'язань, що відображені у Додатку 2.

 

Повернення по гуртових продажах

У разі повернення запасів, выдображеного документом Повернення товарів від покупця, документи рекомендується формувати автоматично обробкою Формування податкових накладних. У разі повернення послуг, ОЗ, НМА документ вводиться на підставі Податкової накладної.

Зазначається Вид операції, відповідний Виду операції Податкової накладної - підставі:

  • Операції, оподатковувані ПДВ (повернення) для Податкової накладної виду Операції, оподатковувані ПДВ
  • Операції, звільнені від ПДВ (повернення) для Податкової накладної виду Операції, звільнені від ПДВ
  • Операції, що не є об'єктом оподаткування (повернення) для Податкової накладної виду Операції, що не є об'єктом оподаткування

Вказується Контрагент - покупець і Договір з ним. Якщо покупець не є платником ПДВ, на закладці Додатково зазначається Тип причини невидачі покупцеві 02. Для покупця-платника ПДВ зазначається Тип причини невидачі покупцеві 0.

Коригування по гуртових продажах

Документ вводиться у разі, якщо номенклатура (ціни) фактичної поставки не відповідає виписаній на постачання Податковій накладній. Як правило, необхідність коригування виникає, якщо Податкова накладна не була виписана на передоплату. Можливі ситуації, коли Податкова накладна була виписана на відвантаження, але документ відвантаження після цього змінений.

Документ вводиться на підставі Податкової накладної. Зазначається Вид операції, відповідний Виду операції Податкової накладної - підставі:

  • Операції, оподатковувані ПДВ (повернення) для Податкової накладної виду Операції, оподатковувані ПДВ
  • Операції, звільнені від ПДВ (повернення) для Податкової накладної виду Операції, звільнені від ПДВ
  • Операції, що не є об'єктом оподаткування (повернення) для Податкової накладної виду Операції, що не є об'єктом оподаткування

Вказується Контрагент - покупець і Договір з ним. Якщо покупець не є платником ПДВ, на закладці Додатково зазначається Тип причини невидачі покупцеві 02. Для покупця-платника ПДВ зазначається Тип причини невидачі покупцеві 0.

Повернення по роздрібних продажах

Документи рекомендується формувати автоматично обробкою Формування податкових накладних по роздробі. Для цього повернення повинне бути відображене документом Звіт про роздрібні продажі.

Зазначається Вид операції, відповідний Виду операції Податкової накладної - підставі:

  • Підсумкова в роздробі по оподатковуваному ПДВ (повернення) для Податкової накладної виду Підсумкова в роздробі по оподатковуваному ПДВ
  • Підсумкова в роздробі по звільненому ПДВ (повернення) для Податкової накладної виду Підсумкова в роздробі по звільненому ПДВ

На закладці Додатково зазначається Тип причини невидачі покупцеві 11.

Повернення по роздрібних продажах платникам ПДВ

Документ вводиться на підставі Податкової накладної. Зазначається Вид операції, відповідний Виду операції Податкової накладної - підставі:

  • Повернення в роздріб платникові ПДВ по оподатковуваному ПДВ (повернення) для Податкової накладної виду Повернення в роздріб платникові ПДВ по оподатковуваному ПДВ
  • Повернення в роздріб платникові ПДВ по звільненому ПДВ (повернення) для Податкової накладної виду Повернення в роздріб платникові ПДВ по звільненому ПДВ

Вказується Контрагент - покупець і Договір з ним.

Коригування отримання робіт (послуг) від нерезидента

Документ вводиться на підставі Податкової накладної. Зазначається Вид операції Роботи від нерезидента коригування

Вказується Контрагент - постачальник робіт (послуг) та Договір (контракт) з ним. На закладці Додатково зазначається Тип причини невидачі покупцеві 14.

Повернення за умовним продажем

Документ вводиться на підставі Податкової накладної. Зазначається Вид операції Умовний продаж (повернення)

На закладці Додатково зазначається Тип причини невидачі покупцеві 07, 08 або 13.
