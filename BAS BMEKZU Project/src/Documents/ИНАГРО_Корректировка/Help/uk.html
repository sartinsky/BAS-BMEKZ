Документ "Коригування"

Документ «Коригування» призначений для виконання довільних операцій по коректуванню оперативних даних у підсистемі елеватор.

Документ застосовується для введення початкових залишків або внесення змін безпосередньо в регістри зберігання інформації.

Загальні реквізити документа

Документ містить дві закладки.

Закладка «Основні» пропонує введення наступних реквізитів:

·           «Правити залишки» - - установка даного флага призведе до зміни даних про оперативні залишках підсистеми елеватор;

·           «Правити форму 36» - установка даного флага призведе до зміни даних кількісно-якісного обліку («форма 36»);

·           «Зняти убуток при сушці з залишку» - установка даного прапора призведе до зміни даних про оперативні залишки на вагу зазначений в реквізиті «Спад ваги при суші, кг»;

·           «Власник» - власник зерна;

·           «Договір» - документ підстави для взаєморозрахунків;

·           «Номенклатура» - наименование культуры;

·           «Урожай» рік урожаю або партія готової продукції;

·           «Склад» - склад, на якому відбувається коригування;

·           «Вид зберігання» - вид зберігання зерна;

·           «Вид руху» - вкажіть, що будемо коригувати; приймає значення «прихід», «витрата»;

·           «Фізичний вага», «Кількість», «Залікова вага»- коригування кількісних реквізитів;

·           «Вологість», «Сміттєва домішка», «Зернова домішка» - призначені для введення якісних характеристик зерна;

·           «Ц /% по вологості», «Ц /% по смітної домішці», «Ц /% по зернової домішці» - призначені для введення кількісно-якісних характеристик зерна в центнеро-відсотках;

·           «Спад ваги при сушінні»- довідкове поле про коригування показника «убутку ваги при сушінні зерна».

На закладці «Додаткові» можна вказати необхідність виконання одночасно з основною додаткового коректування по іншій групі реквізитів «Власник». «Владелец».

Особливості проведення документа

При проведенні документа формуються рухи по регістрах оперативного обліку.
