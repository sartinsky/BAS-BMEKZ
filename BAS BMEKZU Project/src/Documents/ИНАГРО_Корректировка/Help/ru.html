Документ "Корректировка"

Документ «Корректировка» предназначен для выполнения произвольных операций по корректировке оперативных данных в подсистеме элеватор.

Документ применяется для ввода начальных остатков или внесения изменений непосредственно в регистры хранения информации.

Общие реквизиты документа

Документ содержит две закладки.

Закладка «Основные» предлагает ввод следующих реквизитов:

·           «Править остатки» - установка данного флага приведет к изменению данных об оперативных остатках подсистемы элеватор;

·           «Править форму 36» - установка данного флага приведет к изменению данных количественно-качественного учета («форма 36»);

·           «Снять убыль при сушки с остатка» - установка данного флага приведет к изменению данных об оперативных остатках на вес указанный в реквизите «Убыль веса при суше, кг»;

·           «Владелец» - владелец зерна;

·           «Договор» - документ основания для взаиморасчетов;

·           «Номенклатура» - наименование культуры;

·           «Урожай» - год урожая или партия готовой продукции;

·           «Склад» - склад, на котором производится корректировка;

·           «Вид хранения» - вид хранения зерна;

·           «Вид движения» - укажите, что будем корректировать; принимает значения «приход», «расход»;

·           «Физический вес», «Количество», «Зачетный вес»- корректировка количественных реквизитов;

·           «Влажность», «Сорная примесь», «Зерновая примесь» - предназначены для ввода качественных характеристик зерна;

·           «Ц/% по влажности», «Ц/% по сорной примеси», «Ц/% по зерновой примеси» - предназначены для ввода количественно-качественных характеристик зерна в центнеро-процентах;

·           «Убыль веса при сушке» - справочное поле о корректировке показателя «убыли веса при сушке зерна».

На закладке «Дополнительные» можно указать необходимость выполнения одновременно с основной дополнительной корректировки по другой группе реквизитов «Владелец».

Особенности проведения документа

При проведении документа формируются движения по регистрам оперативного учета.
