Документ "Списание с банковского счета"

Документ предназначен для учета списания безналичных денежных средств.

Этими документами отражаются следующие виды хозяйственных операций:

Оплата поставщику 
Возврат покупателю
Перечисление налогов/взносов по заработной плате
Расчеты по кредитам и займам
Прочие расчеты с контрагентами
Перевод на другой счет организации
Перечисление заработной платы

Покупка/продажа валюты
Перечисление подотчетному лицу
Прочее списание

При вводе документа в шапке нужно указать следующие реквизиты:

  • Банковский счет - текущий счет организации, с которого были списаны денежные средства.

  • Счет учета - счет учета денежных средств.

  • Получатель - контрагент, который является получателем денежных средств.

  • Сумма - сумма списанных с банковского счета денежных средств.

    Оплата поставщику

    Для отражения оплаты поставщику за товары, работы и услуги необходимо выбрать в поле Вид операции операцию Оплата поставщику.

    Одним документом можно отразить оплату поставщику по нескольким договорам, счетам, документам расчета. Детальная информация о расчетах с поставщиком вводится в табличной части:

    • Договор - договор с контрагентом. Должен иметь вид "С поставщиком", "С комиссионером" или "С комитентом".
    • Документ расчетов - указывается для договора, в котором настроен учет расчетов по документам расчетов. В этом случае при проведении будет погашена задолженность только по указанному документу расчетов (сформирована задолженность по счету на оплату)
    • Счет расчетов - счет учета, на котором при проведении будет погашаться остаток задолженности.
    • Счет авансов - счет учета, на который относится часть оплаты, оставшаяся нераспределенной после погашения задолженности контрагента.
    Возврат покупателю

    Для отражения возврата покупателю полученной ранее оплаты необходимо выбрать в поле Вид операции операцию Возврат покупателю. После выбора операции нужно заполнить реквизиты расшифровки платежа:

    • Договор - договор с контрагентом. Должен иметь вид "С покупателем", "С комиссионером" или "С комитентом".
    • Документ расчетов - указывается для договора, в котором настроен учет расчетов по документам расчетов. В этом случае при проведении будет погашена задолженность только по указанному документу расчетов (по счету на оплату).
    • Счет авансов - счет учета, на котором при проведении будет погашаться остаток задолженности по ранее полученным авансам.
    • Счет расчетов - счет учета, на который относится часть возвращаемой суммы, оставшаяся нераспределенной после погашения задолженности контрагента.
    Перечисление налогов/взносов по заработной плате

    Для отражения перечисления налога (сбора, взноса) в бюджет необходимо выбрать в поле Вид операции операцию Перечисление налога. После выбора операции нужно заполнить реквизиты расшифровки платежа:

    • Счет дебета - счет учета, на котором учитываются расчеты по данному налогу (сбору).

    Расчеты по кредитам и займам

    Для отражения погашения задолженности организации по полученным кредитам и займам, а также при выдаче займа другой организации необходимо выбрать в поле Вид операции операцию Расчеты по кредитам и займам. После выбора операции нужно заполнить реквизиты расшифровки платежа:

    • Договор - договор с контрагентом, которому возвращается (предоставляется) кредит или займ. Должен иметь вид "Прочее".
    • Счет расчетов - счет учета, на котором учитываются расчеты по кредиту (займу).

    Прочие расчеты с контрагентами

    Для отражения прочих операций, связанных с расчетами с прочими дебиторами и кредиторами, за исключением операций по оплате товаров, работ и услуг, а также за исключением операций по кредитам и займам, необходимо выбрать в поле Вид операции операцию Прочие расчеты с контрагентами. После выбора операции нужно заполнить реквизиты расшифровки платежа:

    • Договор - договор с контрагентом. Должен иметь вид "Прочее".
    • Счет расчетов - счет учета, на котором учитываются расчеты с контрагентом.

    Перевод на другой счет организации

    Для отражения перемещения денежных средств с одного банковского счета организации на другой необходимо выбрать в поле Вид операции операцию Перевод на другой счет организации.

    После выбора операции нужно заполнить реквизиты шапки:

    • Счет получателя - банковский счет организации, на который были переведены денежные средства.

    Также нужно заполнить реквизиты расшифровки платежа:

    • Счет дебета - счет учета денежных средств на банковском счете-получателе.

    Перечисление заработной платы

    Для отражения безналичного перечисления заработной платы на банковские счета сотрудников необходимо выбрать в поле Вид операции операцию Перечисление заработной платы.

    В табличной части указываются суммы, перечисленные сотрудникам. При этом заполняются реквизиты: 

    • Ведомость - ведомость на выплату заработной платы. В ведомости должен быть указан способ выплаты Через банк.
    • Сумма платежа - сумма перечисленной зарплаты.
    • Счет учета - бухгалтерский счет, на котором учитывается задолженность по заработной плате.

    Перечисление подотчетному лицу

    Для отражения перечисления средств на банковскую карту подотчетного лица необходимо выбрать в поле Вид операции операцию Перечисление подотчетному лицу. Указывается Физическое лицо - сотрудник организации, которому перечислены денежные средства под отчет.

    Покупка/продажа валюты

    Для перечисления иностранной валюты банку для продажи, а также для перечисления банку средств для покупки иностранной валюты необходимо выбрать в поле Вид операции операцию Покупка(продажа) валюты. Указывается Счет, на котором учитываются перечисленные денежные средства в пути. Рекомендуется указывать счет 333 при перечислении денежных средств в национальной валюте и счет 334 при перечеслении иностранной валюты. 
    Прочее списание

    Для отражения всех других операций, помимо перечисленных выше, необходимо выбрать в поле Вид операции операцию Прочее списание. После выбора операции нужно указать Счет дебета - счет учета для проводки в корреспонденции со счетом учета денежных средств, а также значения субконто этого счета.
