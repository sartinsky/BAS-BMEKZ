Документ "Списання з банківського рахунку"

Документ призначений для обліку списання безготівкових коштів.

Цими документами відображаються такі види господарських операцій:

Оплата постачальнику
Повернення покупцю
Перерахування податків/внесків із заробітної плати
Розрахунки за кредитами і займам
Інші розрахунки з контрагентами
Перерахування на інший рахунок організації
Перерахування заробітної плати

Купівля/продаж валюти
Перерахування підзвітній особі
Інше списання

При введенні документа в шапці потрібно вказати наступні реквізити:

  • Банківський рахунок - поточний рахунок організації, з якого були списані кошти.

  • Рахунок обліку - рахунок обліку коштів.

  • Одержувач - контрагент, який є одержувачем коштів.

  • Сума - сума списаних коштів з банківського рахунку.

    Оплата постачальнику

    Для відображення оплати постачальнику за товари, роботи і послуги необхідно вибрати в полі Вид операції операцію Оплата постачальнику.

    Одним документом можна відобразити оплату постачальнику за кількома договорами, рахунками, документами розрахунку. Детальна інформація про розрахунки з постачальником вводиться в табличній частині:

    • Договір - договір з контрагентом. Повинен мати вид "З постачальником", "З комісіонером" або "З комітентом".
    • Документ розрахунків - вказується для договору, в якому налаштований облік розрахунків за документами розрахунків. У цьому випадку при проведенні буде погашена заборгованість тільки за вказаним документом розрахунків (сформована заборгованість по рахунку на оплату)
    • Рахунок розрахунків - рахунок обліку, на якому при проведенні погашатиметься залишок заборгованості.
    • Рахунок авансів - рахунок обліку, на який відноситься частина оплати, що залишилася нерозподіленою після погашення заборгованості контрагента.
    Повернення покупцю

    Для відображення повернення покупцеві отриманої раніше оплати необхідно вибрати в полі Вид операції операцію Повернення покупцю. Після вибору операції потрібно заповнити реквізити розшифровки платежу:

    • Договір - договір з контрагентом. Повинен мати вид "З покупцем", "З комісіонером" або "З комітентом".
    • Документ розрахунків - вказується для договору, в якому налаштований облік розрахунків за документами розрахунків. У цьому випадку при проведенні буде погашена заборгованість тільки за вказаним документом розрахунків (по рахунку на оплату).
    • Рахунок авансів - рахунок обліку, на якому при проведенні погашатиметься залишок заборгованості за раніше отриманими авансами.
    • Рахунок розрахунків - рахунок обліку, на який відноситься частина повернутої суми, що залишилася нерозподіленою після погашення заборгованості контрагента.
    Перерахування податків/внесків із заробітної плати

    Для відображення перерахування податку (збору, внеску) у бюджет необхідно вибрати в полі Вид операції операцію Перерахування податку. Після вибору операції потрібно заповнити реквізити розшифровки платежу:

    • Рахунок дебету - рахунок обліку, на якому обліковуються розрахунки по даному податку (збору).

    Розрахунки за кредитами і позиками

    Для відображення погашення заборгованості організації за отриманими кредитами і позиками, а також при видачі позики іншої організації необхідно вибрати в полі Вид операції операцію Розрахунки за кредитами і позиками. Після вибору операції потрібно заповнити реквізити розшифровки платежу:

    • Договір - договір з контрагентом, якому повертається (надається) кредит або позику. Повинен мати вид "Інше".
    • Рахунок розрахунків - рахунок обліку, на якому обліковуються розрахунки за кредитом (позикою).

    Інші розрахунки з контрагентами

    Для відображення інших операцій, пов'язаних з розрахунками з іншими дебіторами і кредиторами, за винятком операцій з оплати товарів, робіт і послуг, а також за винятком операцій за кредитами та позиками, необхідно вибрати в полі Вид операції операцію Інші розрахунки з контрагентами. Після вибору операції потрібно заповнити реквізити розшифровки платежу:

    • Договір - договір з контрагентом. Повинен мати вид "Інше".
    • Рахунок розрахунків - рахунок обліку, на якому обліковуються розрахунки з контрагентом.

    Перерахування на інший рахунок організації

    Для відображення переміщення коштів з одного банківського рахунку організації на інший необхідно вибрати в полі Вид операції операцію Перерахування на інший рахунок організації.

    Після вибору операції потрібно заповнити реквізити шапки:

    • Рахунок отримувача - банківський рахунок організації, на який були переведені кошти.

    Також потрібно заповнити реквізити розшифровки платежу:

    • Рахунок дебету - рахунок обліку коштів на банківському рахунку-одержувача.

    Перерахування заробітної плати

    Для відображення безготівкового перерахування заробітної плати на банківські рахунки співробітників необхідно вибрати в полі Вид операції операцію Перерахування заробітної плати.

    У табличній частині зазначаються суми, перераховані співробітникам. При цьому заповнюються реквізити: 

    • Відомість - відомість на виплату заробітної плати. У відомості має бути вказаний спосіб виплати Через банк.
    • Сума платежу - сума перерахованої зарплати.
    • Рахунок обліку - бухгалтерський рахунок, на якому обліковується заборгованість по заробітній платі.

    Перерахування підзвітній особі

    Для відображення перерахування коштів на банківську карту підзвітної особи необхідно вибрати в полі Вид операції операцію Перерахування підзвітній особі. Зазначається Фізична особа - співробітник організації, яким перераховано кошти під звіт.

    Купівля/продаж валюти

    Для перерахування іноземної валюти банку для продажу, а також для перерахування банку коштів для купівлі іноземної валюти необхідно вибрати в полі Вид операції операцію Купівля(продаж) валюти. Зазначається Рахунок, на якому обліковуються перераховані кошти в дорозі. Рекомендується вказувати рахунок 333 при перерахуванні коштів у національній валюті і рахунок 334 при перечеслении іноземної валюти. 
    Інше списання

    Для відображення всіх інших операцій, крім перерахованих вище, необхідно вибрати в полі Вид операції операцію Інше списання. Після вибору операції потрібно вказати Рахунок дебету - рахунок обліку для проводки в кореспонденції з рахунком обліку коштів, а також значення субконто цього рахунку.
