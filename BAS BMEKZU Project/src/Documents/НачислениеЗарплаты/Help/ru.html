Начисление зарплаты

Документ производит начисление зарплаты и формирует проводки по начисленной зарплате.

Табличные части документа можно заполнять автоматически (кнопки «Заполнить» и «Подбор») либо вручную. Автоматически документ заполняется сотрудниками всей организации или указанного подразделения. После заполнения необходимо откорректировать отработанное время для сотрудников, которые работали не все дни месяца. Затем можно производить пересчет зависимых начислений, налогов и взносов (кнопка "Пересчитать").

Результаты расчета по каждому сотруднику можно проанализировать с помощью расчетного листка.
