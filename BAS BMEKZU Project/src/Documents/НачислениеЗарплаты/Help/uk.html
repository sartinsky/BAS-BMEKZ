Нарахування зарплати

Документ проводить нарахування зарплати і формує проводки по нарахованій зарплаті.

Табличні частини документа можна заповнювати автоматично (кнопки «Заповнити» та «Підбір») або вручну. Автоматично документ заповнюється співробітниками організації або зазначеного підрозділу. Після заповнення необхідно відкоригувати відпрацьований час для співробітників, які працювали не всі дні місяця. Потім можна проводити перерахунок залежних нарахувань, податків та внесків (кнопка "Перерахувати").

Результати розрахунку по кожному працівнику можна проаналізувати з допомогою розрахункового листка.
