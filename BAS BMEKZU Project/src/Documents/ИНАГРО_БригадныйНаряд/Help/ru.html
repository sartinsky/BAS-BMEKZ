Документ "Бригадный наряд"

Документ «Бригадный наряд» позволяет реализовать различные варианты бригадных нарядов. Известно, что исходными данными для расчета любого наряда являются сумма фонда заработной платы, выданной на наряд, отработанные каждым сотрудником бригады часы, размер почасового тарифа и коэффициент трудового участия каждого работника. Опции документа «Учитывать КТУ», «Учитывать часы», «Учитывать оклад» позволяют определить тот или иной способ распределения суммы наряда, определенной в реквизите «Сумма наряда».

В форме документа также следует выбрать из справочника «Подразделений» бригаду, сотрудникам которой закрывают наряд.

Заполнить многострочную часть документа работниками выбранного подразделения (бригады) можно с помощью кнопки «Заполнить». Далее, для каждого сотрудника проставляется его КТУ и, если необходимо, часы работы. Последнюю операцию также удобно выполнить с помощью кнопки «Заполнить».

  • рассчитать по КТУ - сумма будет распределена, исходя из величины КТУ (коэффициента трудового участия), установленном каждому сотруднику;

  • рассчитать по окладу - сумма будет распределена, исходя из величины оклада, указанного работнику при приеме на работу, но если оклад указан не был, система выдаст предупреждение;

  • рассчитать по часам - сумма распределится пропорционально отработанному времени.

На закладке "Дополнительно" необходимо указать счет и аналитику затрат. Данный счет и аналитика будет использована в проводках для всех сотрудников, которые указаны в табличной части. При необходимости указать для некоторых сотрудников другой счет и аналитику затрат, это можно сделать непосредственно в табличной части для этого сотрудника, при этом приоритетом пользуется счет и аналитика, которая введена в табличной части.

Рассчитать суммы, причитающиеся каждому работнику можно с помощью кнопки «Рассчитать», после чего будет заполнен реквизит табличной части «Результат».

Важно! Необходимо обратить внимание, что реквизит табличной части Начисления Часов заполняется исходя из данных по норме времени графика, по которому работает сотрудник (при условии, что установлен флаг Зачет отработанного времени вида расчета).

Важно! Следует также знать, что нельзя вводить одному и тому же сотруднику начисления вида "Сдельный наряд", "Бригадный наряд" и "Путевые листы" в один и тот же период. При возникновении данной ситуации система отрабатывает конфликтную ситуацию и предупреждает о ней.
