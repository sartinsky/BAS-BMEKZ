Документ "Модернизация НМА"

Документ "Модернизация НМА" предназначен для отражения в учете модернизации НМА (как правило, доработка программного обеспечения) для целей бухгалтерского и налогового учета.

В шапке документа вносятся сведения об объекте строительства, с которого списываются затраты по модернизации, счет учета затрат, а так же суммы модернизации, отдельно для каждого из видов учета.

Сумму модернизации можно получить автоматически - при нажатии кнопки "Рассчитать суммы" суммы модернизации заполнятся автоматически по данным учета на момент формирования документа. При необходимости эти суммы можно откорректировать вручную.

В табличной части "Нематериальные активы" указываются объекты НМА, которые модернизируются.

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор".

После формирования списка НМА необходимо воспользоваться кнопкой "Заполнить" для заполнения табличной части  информацией об объектах НМА по данным учета, существующим на момент введения документа а также для распределение сумм модернизации.

При автоматическом заполнении суммы модернизации распределяются равными долями по всем объектам, перечисленным в табличной части.
