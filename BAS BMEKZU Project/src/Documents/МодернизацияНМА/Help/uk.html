Документ "Модернізація НМА"

Документ "Модернізація НМА" призначений для відображення в обліку модернізації НМА (як правило, доопрацювання програмного забезпечення) для цілей бухгалтерського і податкового обліку.

У шапці документа вносяться відомості про об'єкт будівництва, з якого списуються витрати з модернізації, рахунок обліку витрат, а також суми модернізації, окремо для кожного з видів обліку.

Суму модернізації можна отримати автоматично - при натисканні кнопки "Розрахувати суми" суми модернізації заповняться автоматично за даними обліку на момент формування документа. При необхідності ці суми можна відкоригувати вручну.

У табличній частині "Нематеріальні активи" наводяться об'єкти НМА, які модернізуються.

Для більш швидкого заповнення табличної частини документа необхідно скористатися кнопкою "Підбір".

Після формування списку НМА необхідно скористатися кнопкою "Заповнити" для заповнення табличної частини інформацією про об'єкти НМА за даними обліку, що існують на момент введення документа, а також розподіл сум модернізації.

При автоматичному заповненні суми модернізації розподіляються рівними частками по всіх об'єктах, перерахованих у табличній частині.
