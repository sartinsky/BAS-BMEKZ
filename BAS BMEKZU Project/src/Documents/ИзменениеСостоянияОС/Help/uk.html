Документ "Зміна стану ОЗ"

Документ дозволяє змінити стан основного засобу в бухгалтерському обліку.

При зміні стану можна змінити і стан нарахування або не нарахування амортизації на період перебування ОЗ в цьому стані. Стан нарахування амортизації можна змінити також і для податкового обліку.

Для зміни стану нарахування амортизації по об'єктах необхідно встановити прапор "Впливає на нарахування амортизації".

Даний документ змінює значення відповідних ресурсів регістрів відомостей, що зберігають інформацію про основні засоби. У податковому обліку для ОЗ, які обліковуються пооб'єктно, здійснюються нарахування амортизації та коригуючі проводки, якщо це необхідно за станом прапора нараховувати амортизацію.

Для швидкого заповнення табличної частини однотипними об'єктами основних засобів, що мають однакові найменування, потрібно ввести в табличну частину хоча б один такий об'єкт. Потім слід скористатися кнопкою Заповнити у командній панелі табличної частини. При виборі в меню пункту По найменуванню таблична частина буде заповнена об'єктами основних засобів, мають таке ж найменування, як у введеного спочатку.
