Документ "Нормирование расходов по налогу на прибыль"

Документ предназначен для определения сумм затрат, признаваемых в НУ, по нормируемым статьям затрат.

Документ нормирует все затраты, кроме

- суточных и расходов на улучшение ОС;

- производственных затрат.

На момент проведения документа по счетам затрат должна быть отражена полная сумма затрат по нормируемым статьям.

В документе указываются базы для нормирования: облагаемая прибыль и выручка прошлого года. Базы могут быть определены автоматически по кнопке "Рассчитать".

Нормирование производится автоматически по кнопке "Заполнить". Автоматически рассчитанные суммы должны быть проверены и могут быть исправлены вручную.

Документ формирует сторнирующие проводки на сумму затрат сверх норм по счету КЗ "Корректировки нормируемых затрат". Обороты по счету учитываются при заполнении Декларации.

ВАЖНО! Предприятия, которые расходы на приобретение услуг консалтинга и инжиниринга у нерезидентов, начисление роялти нерезидентам относят на 23-й счет, нормирование таких расходов производят вручную. При наличии расходов сверх норм, должна быть введена ручная операция

Дт 23 Кт 00

В операции заполняется только Сумма н\у сторно, равная сумме сверхнормативных расходов.

Нормирование производственных затрат при помощи ручной операции должно быть выполнено до ввода документа "Нормирование расходов на прибыль".
