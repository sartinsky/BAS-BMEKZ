Документ "Реєстр ТТН (ввезення)"

Бухгалтерія зернового складу згідно даних товарно-транспортних накладних виписує за формами ЗХС-3 для однорідних по якості партій зерна реєстри накладних. В конфігурації це реалізовано за допомогою документу «Реєстр ТТН (ввезення)».  

Загальні реквізити документу

Реєстри групуються по ознакам: постачальник, власник, культура, дата, рік врожаю, вид зберігання, експрес аналізу (ст. вологості, ст. смітної домішки, ст. зернової домішки).

Автоматичне формування реєстрів ТТН можливе при  використанні кнопки «Сформувати реєстри», яка знаходиться в журналі «Реєстри ТТН» чи «ТТН». Після цього необхідно ввести дату, за яку буде сформований реєстр. Якщо константа  «Інтерактивне формування Реєстрів ТТН» встановлена у значення «Так», тоді документ «Реєстр ТТН  (ввезення)» може формуватися автоматично. Сформовані реєстри будуть розміщені у відповідному журналі.

На закладці «Основні» вказуються параметри партії, що ввозиться: постачальник, власник, склад, вид зберігання, культура, врожай, при автоматичному формуванні ці реквізити будуть заповнені. В реквізиті «Аналіз» вказується відповідний лабораторний аналіз із списку, із аналізу автоматично підставляється номер аналізу в реквізит «№ аналізу» та якісні показники «Вологість», «Смітна домішка», «Зернова домішка», ці реквізити можна вказати і вручну. В реквізитах «Вага нетто», «Залікова вага» вказується загальна то залікова вага по накладним.

На закладці «Список ТТН» відображається перелік товарно-транспортних накладних, які увійшли в цей реєстр.

На закладці «Розрахунки» розміщені дані для попереднього розрахунку послуг по доведенню зернової продукції до базисних кондицій, а саме: фізична вага прийнятої продукції, лабораторні показники, залікова вага, природній убуток, інформативно виводяться базиси для розрахунків. В групі реквізитів «Вартість послуг» відображається вартість та кількість  наданих клієнту послуг. По кнопці «Розрахувати» проводиться автоматично розрахунок вартості послуг, що надаються. Послуги по очищенню та сушінню розраховуються у тому випадку, коли будуть встановлені відповідні прапорці. Прапорці «Сушіння», «Очищення», «Сушіння актив. вентилювання» приймають те значення, яке вказане у лабораторному аналізі.

Щоб вивести документ на друк, скористайтеся кнопкою «Друк». Документ має декілька друкованих форм: «Форма ЗХС-3», «Форма ЗХС-3»(по культурам), «Розрахунок по власникам».

 

Особливості проведення документу

При проведенні документу в бухгалтерському обліку будут сформовані проводки по 023 рахунку «Матеріальні цінності на відповідальному зберіганні». В оперативному обліку записуються рухи документу по регістрах.
