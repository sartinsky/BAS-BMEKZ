Документ "Реестр ТТН (ввоз)"

Бухгалтерия зернового склада согласно данных товарно-транспортних накладных выписывает за формами ЗХС-3 для однородных по качеству партий зерна реестры накладных. В конфигурации это реализовано с помощью документа «Реестр ТТН».  

Общие реквизиты документа

Реестры группируются по признакам: поставщик, владелец, культура, дата, год урожая, вид хранения, экспресс анализу (ст. влажности, ст. сорной примеси, ст. зерновой примеси).

Автоматическое формирование реестров ТТН возможно при  использовании кнопки «Сформировать реестры», которая находится в журнале «Реестры ТТН» или «ТТН». После чего необходимо ввести дату, за которую будет сформирован реестр. Если константа  «Интерактивное формирование Реестров ТТН» установлена в значении «Да», то документ «Реестр ТТН» может формироваться автоматически. Сформированные реестры будут помещены в соответствующий журнал.

На закладке «Основные» указываются параметры ввозимой партии: поставщик, владелец, склад, вид хранения, культура, урожай, при автоматическом формировании эти реквизиты будут заполнены. В реквизите «Анализ» необходимо указать соответствующий лабораторный анализ из списка, из анализа автоматически подставляется номер анализа в реквизит «№ анализа» и качественные показатели «Влажность», «Сор. Примесь», «Зерн. Примесь», эти реквизиты можно указать и вручную. В реквизитах «Вес нетто», «Зачетный вес» указывается общий вес и зачетный вес по накладным.

На закладке «Список ТТН» отображается перечень товарно-транспортных накладных которые вошли в этот реестр.

На закладке «Расчеты» размещены данные для предварительного расчета услуг по доведению зерновой продукции до базисных кондиций, а именно: физический вес принятой продукции, лабораторные показатели, зачетный вес, натуральная убыль, информативно выводятся базисы для расчетов. В группе реквизитов «Стоимость услуг» отображается стоимость и количество выставляемых клиенту услуг. По кнопке «Рассчитать» производит автоматически расчет стоимости выполняемых услуг. Услуги по очистке и сушки рассчитаются в том случае когда будут установлены соответствующие флажки. Флажки «Сушка», «Очистка», «Сушить актив. вентелир.» принимают то значение, которое указано в лабораторном анализе.

Чтобы вывести документ на печать, воспользуйтесь кнопкой «Печать». Документ имеет несколько печатных форм:: «Форма ЗХС-3», «Расчет по владельцам»

 

Особенности проведения документа

При проведении документа в бухгалтерском учете будут сформированы проводки по 023 счету «Материальные ценности на ответственном хранении». В оперативном учете записывается движение по регистрам.
