Документ "Акт уничтожения негодных отходов"

Документ содержит информацию о количестве вывезенных за период негодных отходов.

 

С помощью кнопки "Заполнить" документ заполняется данными о количестве вывезенных отходов 3-ей категории.

 

Общие реквизиты документа

 Для этого необходимо заполнить следующие реквизиты:

·           «Склад» - внутренний склад элеватора, на котором хранится зерно;

·           «Урожай» - имеет вид года сбора урожая, данной зерновой продукции;

·           «Период» - указать период, за который вывозились отходы;

·           «Отходы» - отходы, которые необходимо распределить между владельцами.
