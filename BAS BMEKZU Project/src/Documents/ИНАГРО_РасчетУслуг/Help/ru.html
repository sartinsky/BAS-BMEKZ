Документ «Расчет услуг»

Одним из преимуществ данной системы является автоматический расчет стоимости услуг, оказанных клиенту. Для этого в программе предусмотрен документ «Расчет услуг».

Общие реквизиты документа

Для автоматического расчета необходимо в диалоговой форме документа указать такую информацию:

·           «Владелец» - определяет тип справочника «Контрагенты»;

·           «Договор» - необходимо указать документ договор по которому осуществляется хранение зерна;

·           «Культура» - элемент справочника «Номенклатура» определяет культуру по которой будут насчитываться услуги.

·           «Вид хранения» - вид хранения, на который поступает данная партия зерна;

·           «Урожай» - имеет вид года сбора урожая, данной зерновой продукции;

·           «Склад» - внутренний склад предприятия, на котором хранится зерно;

·           В реквизитах «Начало», «Конец» - указать период, за который производится расчет.

После этого нажмите кнопку «Рассчитать». При этом автоматически заполнится табличная часть документа.

На закладке «Расчет хранения» в табличной части отображается информация по которой был произведен расчет услуги хранения.

На закладке «Счета учета расчетов» необходимо указать счета учета расчетов с владельцем зерна.

На закладке «Дополнительно» заполняются дополнительные реквизиты для печатной формы документа. Для автоматического формирования счета на оплату покупателя необходимо нажать

на кнопку «Счет-фактура». Документ имеет печатную форму в виде «Акта выполненных работ» или «Расчета платы за хранение». Вид печатных форм выбирается с помощью кнопки

«Печать».

Особенности проведения документа

При проведении документа формируются проводки по взаиморасчетам с клиентом за услуги в бухгалтерском учете, отражается движение по регистрам налогового учета.
