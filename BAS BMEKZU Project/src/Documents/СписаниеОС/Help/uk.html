Документ "Списання ОЗ"

Документ «Списання ОЗ» призначений для відображення в обліку вибуття основного засобу у зв'язку з моральним і фізичним зносом, а також при ліквідації у зв'язку з надзвичайною ситуацією.

  • бухгалтерським обліком;
  • податковим обліком.

Об'єкти, що вибувають, вибираються з довідника "Основні засоби".

Для кожного об'єкта основних засобів вказується поточна балансова вартість, сума накопиченої на початок місяця амортизації, розрахункова сума амортизації, яка повинна бути донараховано за поточний місяць.

Для більш швидкого заповнення табличної частини документа необхідно скористатися кнопкою "Підбір".

Після формування списку основних засобів, що готуються до передачі необхідно скористатися кнопкою "Заповнити" для заповнення табличної частини інформацією про об'єкти основних засобів за даними обліку, сума амортизації за поточний місяць визначається розрахунковим шляхом в документі.

Для швидкого заповнення табличної частини однотипними об'єктами основних засобів, що мають однакові найменування, потрібно ввести в табличну частину хоча б один такий об'єкт. Потім слід скористатися кнопкою Заповнити у командній панелі табличної частини. При виборі в меню пункту По найменуванню таблична частина буде заповнена об'єктами основних засобів, мають таке ж найменування, як у введеного спочатку.

Відображення у бухгалтерському обліку

При списання основного засобу необхідно оформити акт про списання об'єкта основних засобів. Для цього слід натиснути на кнопку "ОЗ-3".

При списанні з обліку кожного з перечисленнных в табличній частині основних засобів для об'єктів, формуються проводки:

• за донарахуванням амортизації, якщо це необхідно;
• по списанню амортизаційних нарахувань;
• по списанню залишкової вартості об'єкта на зазначений у формі рахунок обліку витрат. 

Додатково в регістрах відомостей знімається прапор нарахування амортизації за об'єктами обліку, що перераховані у документі .

Донарахована амортизація розподіляється за витратами, аналогічно документа "Закриття місяця".

У податковому обліку списання основних засобів відображається, згідно з податковим законодавством.

В якості рахунки обліку витрат при списанні основних засобів з бухгалтерського обліку за замовчуванням пропонується рахунок 976 "Списання необоротних активів". 

Якщо ліквідація відбувається за рішенням платника податку, а не в наслідок руйнування, то для коригування ПДВ можна ввести на підставі даного документа податкову накладну.
