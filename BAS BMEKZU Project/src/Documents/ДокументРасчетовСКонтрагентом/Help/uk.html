Документ "Документ розрахунків з контрагентом (ручний облік)"

Допоміжний документ, який не робить ніяких проводок, призначений для ручного ведення обліку за документами розрахунків, тобто для організації субконто "Документи розрахунків з контрагентами" на рахунках розрахунків з контрагентами. Реквізити необхідні для заповнення:

  • Організація
  • Контрагент
  • Договір
