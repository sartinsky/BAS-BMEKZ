Документ "Документ расчетов с контрагентом (ручной учет)"

Вспомогательный документ, который не делает никаких проводок, предназначен для ведения ручного учета по документам расчетов, т.е. для организации субконто "Документы расчетов с контрагентами" на счетах расчетов с контрагентами. Реквизиты необходимые для заполнения:

  • Организация
  • Контрагент
  • Договор
