Документ «Акт списания»

Документ предназначен для отражения в программе операций списания произвольного состава и количества сырья в производство. По смыслу он подобен бухгалтерскому документу «Отчет производства за смену», только выполняет движение по подсистеме элеватор.

В шапке документа необходимо заполнить следующие реквизиты:

·           «Склад» - место хранения, с которого будет выполнено списание товароматериальных ценностей;

·            «Владелец» - владелец списываемого сырья;

·           «Договор» - документ-основание для взаиморасчетов с клиентом;

·            «Урожай» и «Вид хранения» - списать с определенного урожая и  вида хранения;

·           «Рецепт», «Продукция» - группа реквизитов, показывающая какое наименование готовой продукции планируется выпускать, согласно какого рецепта;

·           «Процент потерь» - указывается процент потерь сырья при производстве, используется при автоматическом заполнении табличной части документа.

В табличной части необходимо указать перечень товароматериальных ценностей списываемых в производство, их количество и качество.

·           «Номенклатура», «Ед» - списываемая номенклатура, единица измерения;

·            «К-во, шт», «Вес, кг» - укажите количество мешков, вес сырья;

·           «Анализ», «№ анализа», «Влажность», «Сорная примесь» - реквизиты лабораторного анализа, показатели качества;

·           «Зачетный вес» - зачетный вес партии сырья.

Для того чтобы заполнить документ с помощью подбора нажмите кнопку «Подбор». Документ можно заполнить автоматически по данным рецепта. Для этого необходимо нажать кнопку «Заполнить/Заполнить по рецепту». Для того чтобы заполнить документ по остаткам сырья нажмите кнопку «Заполнить/Заполнить по остаткам».

Для печати документа нажмите кнопку «Печать». Документ имеет несколько печатных форм: «Требование» (форма №119); «Накладная» (форма №110); «Распоряжение (форма №109)»; «Накладная форма115».

При проведении документа формируются движения по регистрам оперативного учета и проводки по счету 023 «Материальные ценности на ответственном хранении».
