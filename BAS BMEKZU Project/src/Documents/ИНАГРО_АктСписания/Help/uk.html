Документ "Акт списання"

Документ призначений для відображення в програмі операцій списання довільного складу і кількості сировини у виробництво. За змістом він подібний до бухгалтерського документу «Звіт виробництва за зміну», тільки виконує рух по підсистемі елеватор.

Загальні реквізити документа

У шапці документа необхідно заповнити такі реквізити:

·           «Склад» - місце зберігання, з якого буде виконано списання товарно-матеріальних цінностей;

·            «Власник» - власник списуваної сировини;

·           «Договір» - документ-підстава для взаєморозрахунків з клієнтом;

·            «Врожай» і «Вид зберігання» - списати з певного врожаю і виду зберігання;

·           «Рецепт», «Продукція» - група реквізитів, що показує яке найменування готової продукції планується випускати, згідно якого рецепту;

·           «Відсоток втрат» - вказується відсоток втрат сировини при виробництві, використовується при автоматичному заповненні табличної частини документа.

У табличній частині необхідно вказати перелік товарно- матеріальних цінностей, що списуються у виробництво, їх кількість і якість.

·           «Номенклатура», «Од» - списується номенклатура, одиниця виміру;

·            «К, шт», «Вага, кг» - вкажіть кількість мішків, вага сировини;

·           «Аналіз», «№ анализу», «Вологість», «Смітна домішка» - реквізити лабораторного аналізу, показники якості;

·           «Залікова вага» - залікова вага партії сировини.

Для того щоб заповнити документ за допомогою підбору натисніть кнопку «Підбір». Документ можна заповнити автоматично за даними рецепта. Для цього необхідно натиснути кнопку «Заповнити/Заповнити по рецепту».  Для того щоб заповнити документ по залишках сировини натисніть кнопку  «Заповнити/Заповнити по залишкам».

Для друку документа натисніть кнопку «Друк». Документ має декілька друкованих форм: «Вимога» (форма №119); «Накладна» (форма №110); «Розпорядження (форма №109)»; «Накладна форма115».

Особливості проведення документа

При проведенні документа формуються рухи по регістрах оперативного обліку та проводки по рахунку 023 «Матеріальні цінності на відповідальному зберіганні».
