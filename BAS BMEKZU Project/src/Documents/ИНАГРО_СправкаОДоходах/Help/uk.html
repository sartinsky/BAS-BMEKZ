Документ "Довідка про доходи"

Документ "Довідка про доходи" дозволяє формувати й редагувати для працівників організацій довідки про суми доходів та утримань за довільний період часу одного з наступних видів:

  • довідка в довільній формі - не регламентована форма, що використовується для різних цілей;
  • довідка для призначення допомоги по недієздатності - по формі, затвердженій Постановою Кабінету Міністрів України від 26.09.2001 р. № 1266;
  •  довідка для розрахунку пенсії - по формі, затвердженій Постановою правління Пенсійного фонду України від 25.11.2005 р. № 22-1;
  • довідка для оформлення субсидії - по формі, затвердженій Наказом Міністерства праці і соціальної політики України від 27.08.2004р. № 192.
