Документ "Платежное поручение"

Документ предназначен для подготовки печатной формы платежного поручения. Движений документ не формирует.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Банковский счет - текущий счет организации, с которого переводятся денежные средства.
  • Получатель - контрагент, который является получателем денежных средств.
  • Счет получателя - банковский счет получателя, на который переводятся денежные средства.
  • Сумма платежа - сумма, которая будет списана с банковского счета.

Для корректного формирования печатной формы платежного поручения должны быть правильно заполнены реквизиты:

  • Код по ЕДРПОУ плательщика, Наименование плательщика - реквизиты организации-плательщика. Редактируются в отдельном окне Реквизиты плательщика, которое открывается по гиперссылке под Банковским счетом организации.
  • МФО банка плательщика, Наименование банка плательщика - реквизиты банка, которому направляется платежное поручение. Редактируются в отдельном окне Реквизиты банка плательщика, которое открывается по гиперссылке под Банковским счетом организации.
  • Код по ЕДРПОУ получателя, Наименование получателя - реквизиты контрагента-получателя. Редактируется в отдельном окне Реквизиты получателя. которое открывается по гиперссылке под Счетом получателя.
  • МФО банка получателя, Наименование банка получателя - реквизиты банка, которому перечисляются денежные средства. Редактируется в отдельном окне Реквизиты банка получателя. которое открывается по гиперссылке под Счетом получателя.
