Документ "Выработка для расчета амортизации НМА"

Документ служит для отражения фактического объема продукции (работ) для НМА с начислением амортизации способом списания стоимости пропорционально объему продукции (работ).

Данные о фактическом объеме продукции (работ) записываются в специальный регистр накопления "Выработка НМА" и используются в дальнейшем при начислении амортизации за текущий период. 

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистра накопления. Используется только для бухгалтерского учета.
