Документ "Виробіток для розрахунку амортизації НМА"

Документ служить для відображення фактичного обсягу продукції (робіт) для НМА з нарахуванням амортизації способом списання вартості пропорційно обсягу продукції (робіт).

Дані про фактичний обсяг продукції (робіт) записуються в спеціальний регістр накопичення "Виробіток НМА" і використовуються надалі при нарахуванні амортизації за поточний період. 

Даний документ не формує ніяких проводок, він лише змінює значення відповідних ресурсів регістру накопичення. Використовується тільки для бухгалтерського обліку.
