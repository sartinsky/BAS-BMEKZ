Документ "Изменение параметров начисления амортизации"

Документ предназначен для изменения способа начисления амортизации, а также ее параметров: срока полезного использования, объема выработки и пр.

Для более быстрого заполнения табличной части документа  необходимо воспользоваться кнопкой "Подбор". Для заполнения табличной части однотипными объектами основных средств, имеющими одинаковые наименования, нужно ввести в табличную часть хотя бы один такой объект. Затем следует воспользоваться кнопкой Заполнить в командной панели табличной части. При выборе в выпадающем меню пункта По наименованию табличная часть будет заполнена объектами основных средств, имеющими такое же наименование, как у введенного первоначально.

После формирования списка ОС, необходимо воспользоваться кнопкой "Заполнить" для заполнения табличной части  информацией о параметрах амортизации ОС на момент введения документа. Новые параметры вносятся вручную.

При проведении документа новые параметры амортизаций записываются в соответствующие регистры сведений и используются документом "Закрытие месяца" при начислении амортизации начиная с месяца, следующего за месяцем изменения параметров.
