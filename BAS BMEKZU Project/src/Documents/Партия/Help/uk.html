Документ "Партія"

Допоміжний документ, який не робить ніяких проводок, призначений для ручного ведення обліку по партіях, тобто для організації субконто "Партія" на рахунках бухгалтерського обліку.

Реквізити необхідні для заповнення:

  • Організація
  • Контрагент - постачальник партії товарів чи матеріалів;
  • Договір - договір поставки партії товарів чи матеріалів;
