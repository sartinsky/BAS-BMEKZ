Документ "Реализация товаров и услуг"

Документ предназначен для отражения операций по реализации товаров, материалов, услуг, готовой продукции.

Документ можно ввести на основании документов Счет на оплату покупателю и Поступление товаров и услуг.

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Организация - организация предприятия, от имени которой оформляется реализация.
  • Контрагент - покупатель (комиссионер).
  • Договор - договор поставки (комиссии).
  • Склад - склад, с которого производится реализация.
  • Тип цен - цена товара, по которой оформляется реализация. Выбирается из нескольких допустимых для товара цен. Тип цен заполняется автоматически тем типом цен, который установлен в договоре с покупателем (комиссионером). Тип цен можно изменить по кнопке Редактировать цены и валюту документа. 
  • Валюта - валюта документа. Валюта заполняется автоматически валютой договора. 

Реализация товаров, материалов, готовой продукции

Для отражения реализации товаров, материалов или готовой продукции необходимо выбрать в поле Вид операции операцию Продажа, комиссия. После выбора операции нужно заполнить закладки:

  • Товары
  • Счета учета расчетов
  • Дополнительно

На закладке Товары указывается номенклатура, количество, цена, ставка НДС, а также счет учета, налоговые назначения и схема реализации номенклатуры.

  • Цена заполняется автоматически при указании номенклатуры. Информация для заполнения этих полей берется из регистра Цены номенклатуры.
  • Ставка НДС определяется автоматически при указании номенклатуры. Указывается ставка из номенклатурной карточки.
  • Поля Счет учета, Налоговое назначение НДС, Схема реализации заполняются автоматически при указании номенклатуры. Информация для заполнения этих полей берется из регистра Счета учета номенклатуры.

На закладке Счета учета расчетов указываются счета расчетов и авансов, а также счет учета налоговых обязательств (НДС). Все счета учета заполлняются автоматически из регистра сведений Счета учета расчетов с контрагентами.

На закладке Дополнительно указывается информация, которая будет выводиться в печатные формы: грузоотправитель и грузополучатель, место составления документа, представители сторон и пр.

Оказание услуг

Для отражения оказанных услуг необходимо выбрать в поле Вид операции операцию Продажа, комиссия. После выбора операции нужно заполнить закладки:

  • Услуги
  • Счета учета расчетов
  • Дополнительно

На закладке Услуги указывается содержание оказанной услуги, количество, цену, ставка НДС, а также Схема реализации.

  • Если оказание услуги носит постоянный характер, то рекомендуется внести наименование услуги в справочник Номенклатура. Если услуга носит разовый характер, то можно ввести указать содержание услуги в поле Содержание услуги, доп.сведения.
  • Цена заполняется автоматически, если указана номенклатура. Информация для заполнения этих полей берется из регистра Цены номенклатуры.
  • Ставка НДС определяется автоматически, если указана номенклатура. Указывается ставка из номенклатурной карточки.
  • Поле Схема реализации заполняется автоматически при указании номенклатуры. Информация для заполнения этого поля берется из регистра Счета учета номенклатуры.

На закладке Счета учета расчетов указываются счета расчетов и авансов, а также счет учета налоговых обязательств (НДС). Все счета учета заполлняются автоматически из регистра сведений Счета учета расчетов с контрагентами.

На закладке Дополнительно указывается информация, которая будет выводиться в печатные формы: место составления документа, представители сторон и пр.

Передача возвратной тары

Для отражения передачи возвратной тары необходимо выбрать в поле Вид операции операцию Продажа, комиссия. После выбора операции нужно заполнить закладки:

  • Возвратная тара
  • Счета учета расчетов
  • Дополнительно

На закладке Возвратная тара указывается номенклатура тары, количество и залоговая цена, счет учета. Поскольку в бухгалтерском учете передача тары отражается как ее продажа, указываются также налоговые назначения и схема реализации номенклатуры.

На закладке Счета учета расчетов указываются счета расчетов и авансов по таре. Счета заполняются автоматически из регистра сведений Счета учета расчетов с контрагентами.

На закладке Дополнительно указывается информация, которая будет выводиться в печатные формы: место составления документа, представители сторон и пр.

Передача товаров комиссионеру

Для отражения передачи товаров или готовой продукции комиссионеру необходимо выбрать в поле Вид операции операцию Продажа, комиссия. При этом в поле Договор должен быть указан договор вида С комиссионером. 

После выбора операции нужно заполнить закладки:

  • Товары
  • Счета учета расчето
  • Дополнительно

На закладке Товары указывается номенклатура, количество, цена, ставка НДС, а также счет учета и налоговое назначение НДС.

  • Цена заполняется автоматически при указании номенклатуры. Информация для заполнения этих полей берется из регистра Цены номенклатуры.
  • Ставка НДС определяется автоматически при указании номенклатуры. Указывается ставка из номенклатурной карточки
  • Поля Счет учета, Налоговое назначение НДС заполняются автоматически при указании номенклатуры. Информация для заполнения этих полей берется из регистра Счета учета номенклатуры.

    На закладке Счета учета расчетов указывается счет учета налоговых обязательств (НДС). Счет заполняется автоматически из регистра сведений Счета учета расчетов с контрагентами.
