Документ "ГТД по импорту"

Документ "ГТД по импорту" оформляется в том случае, если необходимо при поступлении импортных товаров отразить информацию о таможенных сборах, акцизе, НДС и пошлине, зафиксированных в грузовой таможенной декларации.

В документе указывается номер грузовой таможенной декларации ("Номер ГТД"), по которой необходимо ввести информацию о таможенных пошлинах и сборах. 


В поле "Таможня" заполняется наименование таможни, а в поле "Договор с таможней" указывается информация о депозите на таможне, по которому учитывается взимание таможенных платежей. Информация о названии таможни  и договоре (депозите на таможне) заполняется в справочниках "Контрагенты" и "Договоры контрагентов".

 

Информация о таможенных сборах (из графы 47 основного листа ГТД) заполняется на закладке "Основные".

Грузовая таможенная декларация может иметь несколько разделов, в которые группируются товары с одинаковым порядком расчета таможенных платежей (условно к разделу относятся графы 31-47 декларации, которые могут повторяться). Информация по каждому разделу ГТД заполняется на отдельной закладке в документе.

Добавление и удаление разделов ГТД производится с помощью кнопки "Разделы ГТД". В документе присутствуют две табличные части. В верхней таблице отображается сводная информация о пошлине, акцизе и НДС по разделам ГТД, по одной строке на каждый раздел. При установке курсора на одну из строк этой таблицы в дополнительной таблице "Товары по разделу.." отображается информация о товарах по данному разделу ГТД.

По каждому разделу ГТД может быть введена своя сумма пошлины, акциза  и НДС. Сумма платежей вычисляется исходя из введенной таможенной стоимости товаров и соответствующей ставки. Сумма НДС рассчитывается с учетом начисленной пошлины и акциза.

В табличную часть документа на закладке "Разделы ГТД" заполняется информация о товарах. Фактурная стоимость товаров вводится в гривнах. Перечень товаров может быть взят из документа "Поступление товаров и услуг" (по кнопке "Заполнить" выбирается документ поступления, из которого весь перечень товаров попадает в табличную часть соответствующего раздела; товары, которые не относятся к данному разделу, можно удалить, после чего повторить процедуру для следующего раздела ГТД).

По кнопке "Распределить" введенная информация о пошлине, акцизе и  НДС автоматически распределяется пропорционально фактурной стоимости товаров.

При необходимости информация о платежах по каждому товару может быть откорректирована вручную.

Документ может быть проведен по бухгалтерскому и налоговому учету. Информация о счетах учета расчетов с таможней вводятся на закладке "Счета учета расчетов". На закладке "Прочие счета учета" вводится информация о счетах учета пошлины, акциза. Если ГТД оформляется на товары комитента в реквизите "Счет затрат" нужно указать счет, на который будут отнесены суммы пошлин, акциза и сборов, уплаченные при таможенном оформлении.

При проведении документа, помимо записей (проводок) по бухгалтерскому учету, также производятся записи в регистр накопления "НДС налоговый кредит" в части НДС, уплаченного таможенным органам. Информация из этого регистра используется при построении Декларации по НДС.
