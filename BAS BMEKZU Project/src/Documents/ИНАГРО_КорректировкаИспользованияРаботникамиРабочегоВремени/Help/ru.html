Документ «Корректировка использования работниками рабочего времени»

Документ предназначен для регистрации фактического времени явок и отсутствия работников по видам использования рабочего времени.

Заполнение полей документа. В соответствующих полях документа указываются:

  • Организация (по умолчанию это организация, указанная в настройках пользователя);
  • Ответственный (по умолчанию это ответственный, указанный в настройках пользователя);
  • Месяц (по умолчанию текущий месяц) - определяет, к какому месяцу относятся данные о фактическом времени;
  • Подразделение - если указано подразделение, документ будет автоматически заполняться данными работников этого подразделения.

В табличной части документа указываются:

  • работник; если в организации используется внутреннее совместительство, дополнительно уточняется конкретное место работы - указывается приказ о приеме на работу;
  • период времени - один или несколько дней;
  • код использования рабочего времени (явка, болезнь и пр.);

Автоматическое заполнение. Меню «Заполнить» позволяет автоматически занести в табличную часть документа работников организации, отобранных по произвольным критериям, и заполнить документ по графикам работы и имеющимся данным об отклонениях (включая почасовые).
