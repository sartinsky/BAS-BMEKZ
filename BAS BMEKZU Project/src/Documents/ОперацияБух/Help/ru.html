Документ "Операция"

 

Документ предназначен для выполнения следующих операций:

  • Ввод ручной операции
  • Ввод типовой операции
  • Корректировка регистров
  • Сторнирование записей одного или нескольких документов

Ввод ручной операции

В списке операций нужно нажать кнопку Создать - Операция. Документ заполняется вручную

Ввод типовой операции

В списке операций нужно нажать кнопку Создать - Типовая операция, выбрать соответствующую операцию и нажать кнопку Заполнить.

Корректировка регистров

В списке операций нужно нажать кнопку Создать - Операция, затем нажать кнопку Еще - Выбор регистров и выбрать список регистров для корректировки.

Сторнирование

В списке операций нужно нажать кнопку Создать - Сторно документа, затем выбрать сторнируемый документ по кнопке .... Сторнируемые проводки и движения документа заполнятся автоматически.
