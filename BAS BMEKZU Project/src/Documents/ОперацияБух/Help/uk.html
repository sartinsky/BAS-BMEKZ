Документ "Операція"

 

Документ призначений для виконання наступних операцій:

  • Введення ручної операції
  • Введення типової операції
  • Коригування регістрів
  • Сторнування записів одного або кількох документів

Введення ручної операції

У списку операцій потрібно натиснути кнопку Створити - Операція. Документ заповнюється вручну

Введення типової операції

У списку операцій потрібно натиснути кнопку Створити - Типова операція, вибрати відповідну операцію і натиснути кнопку Заповнити.

Коригування регістрів

У списку операцій потрібно натиснути кнопку Створити - Операція, потім натиснути кнопку Ще - Вибір регістрів і вибрати список регістрів для коригування.

Сторнування

У списку операцій потрібно натиснути кнопку Створити - Сторно документа, потім вибрати документ, що сторнується, по кнопці .... Сторнируемые проводки та рухи документа заповняться автоматично.
