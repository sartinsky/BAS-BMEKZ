Документ «Расчеты с владельцами»

Документ является рабочим местом бухгалтера по переработке давальческого сырья. Предназначен для автоматического формирования пакета документов по фасовке отгрузке, давальческой продукции. С помощью документа сформированный пакет документов можно вносить изменения.

 

Особенности документа

Документ движений не делает, а лишь автоматически формирует следующие документы:

  • "Акт фасовки";
  • "Приказ на вывоз";
  • "Переоформление".
