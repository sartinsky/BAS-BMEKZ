Документ "Нарахування зарплати працівникам організацій"

Документ "Нарахування зарплати працівникам організацій" призначений для введення нарахувань і утримань для співробітників організації.

Заповнення документа може відбуватися автоматично за допомогою кнопки "Заповнити" командної панелі форми. При натисненні на дану кнопку всі табличні частини документа заповнюються даними по співробітниках вказаної в шапці організації та підрозділу (якщо останнє вказане). До таких даних можуть відноситься всі планові нарахування і утримання, які були вказані співробітникові при прийомі на роботу в документі "Прийом на роботу в організацію", а також дані, зафіксовані документами "Введення будь-якого розрахунку", "Відрядний наряд" і "Бригадний наряд".

Заповнити табличні частини документа можна також з допомогою підменю "Заповнити по виділених співробітниках" і "Заповнити по всіх співробітниках". Дії команд цих два підменю практично однакові - обробляються всі табличні частини документа (окрім "Працівники") по виділених (або по всіх) в табличній частині "Працівники" співробітникам.

Для кожної табличної частини існує також своя кнопка "Заповнити", яка має локальне значення, - заповнюється тільки ця конкретна таблична частина.

Розрахунок документа здійснюється при натисненні на кнопку "Розрахувати" командної панелі форми, при цьому розраховуються всі записи у всіх табличних частинах. Так само, як і у випадку з кнопкою "Заповнити", для табличних частин "Нарахування", "Внески", "ПДФО", "Утримання" і "Внески ФОП" є кнопка "Розрахувати" локального значення, що розраховує тільки свою табличну частину.

Для згаданих вище табличних частин існує також підменю "По робітникам", елементи якого дозволяють заповнити і розрахувати записи даного виду для виділеного в табличній частині "Робітники" співробітника. Також за допомогою окремого елементу підменю "По працівникові" можна виконати операцію додавання новому запису табличної частини (для всіх вказаних табличних частин, окрім "ПДФО").

Для табличних частин "Нарахування", "Внески", "ПДФО", "Утримання" і "Внески ФОП" для відображення проводок, що формуються, існує підзакладка "Проводки". Загальну картину по проводках можна побачити на закладці "Відображення в бухобліку".

На закладці "Додатково" є можливість включити або відключити видимість деяких реквізитів табличної частини "Нарахування".

Важливо! На підзакладку "Проводки" табличної частини "Нарахування" можуть потрапляти також записи, розраховані документами "Нарахування по лікарняному листу" та "Нарахування відпустки працівникам організацій". Крім того, в табличну частину "Відображення в бухобліку" потраплять записи, які були розраховані документами "Введення будь-якого розрахунку", "Відрядний наряд" та "Бригадний наряд". Для полегшення контролю коректності таких записів на закладці "Додатково" існує прапор "Відбір по нарахуваннях співробітнику", який дозволяє відфільтрувати записи табличної частини "Відображення в бухобліку".

Важливо! Якщо документом "Введення будь-якого розрахунку" були записані "нульові" записи, ці записи потраплять також на закладку "Нарахування" ("Утримання") для подальшого, остаточного, розрахунку.

Важливо! Перед початком роботи з даним документом, для коректної роботи механізмів розрахунку, необхідно заповнити актуальні на підприємстві графіки роботи, що зберігаються в довіднику "Графіки роботи", або ввести документ "Табель обліку робочого часу організації".
