Документ "Начисление зарплаты работникам организаций"

Документ "Начисление зарплаты работникам организаций" предназначен для ввода начислений и удержаний для сотрудников организации.

Заполнение документа может происходить автоматически с помощью кнопки "Заполнить" командной панели формы. При нажатии на данную кнопку все табличные части документа заполняются данными по сотрудникам указанной в шапке организации и подразделения (если последнее указано). К таким данным могут относится все плановые начисления и удержания, которые были указаны сотруднику при приеме на работу в документе "Прием на работу в организацию", а также данные, зафиксированные документами "Ввод любого расчета", "Сдельный наряд" и "Бригадный наряд".

Заполнить табличные части документа можно также с помощью подменю "Заполнить по выделенным сотрудникам" и "Заполнить по всем сотрудникам". Действия команд этих двух подменю практически одинаковы - обрабатываются все табличные части документа (кроме "Работники") по выделенным (либо по всем) в табличной части "Работники" сотрудникам.

Для каждой табличной части существует также своя кнопка "Заполнить", которая имеет локальное значение - заполняется только данная табличная часть.

Расчет документа осуществляется при нажатии на кнопку "Рассчитать" командной панели формы, при этом рассчитываются все записи во всех табличных частях. Так же, как и в случае с кнопкой "Заполнить", для табличных частей "Начисление", "Взносы", "НДФЛ", "Удержания" и "Взносы ФОТ" имеется кнопка "Рассчитать" локального значения, рассчитывающая только свою табличную часть.

Для упомянутых выше табличных частей существует также подменю "По работнику", элементы которого позволяют заполнить и рассчитать записи данного вида для выделенного в табличной части "Работники" сотрудника. Также с помощью отдельного элемента подменю "По работнику" можно выполнить операцию добавления новой записи табличной части (для всех указанных табличных частей, кроме "НДФЛ").

Для табличных частей "Начисление", "Взносы", "НДФЛ", "Удержания" и "Взносы ФОТ" для отображения формируемых проводок существует вкладка "Проводки". Общую картину по проводкам можно увидеть на закладке "Отображение в бухучете".

На закладке "Дополнительно" имеется возможность включить или отключить видимость некоторых реквизитов табличной части "Начисления".

Важно! На подзакладку "Проводки" табличной части "Начисления" могут попадать также записи, рассчитанные документами "Начисление по больничному листу" и "Начисление отпуска работникам организаций". Кроме того, в табличную часть "Отображение в бухгалтерском учете" попадут записи, которые были рассчитаны документами "Ввод любого расчета", "Сдельный наряд" и "Бригадный наряд". Для облегчения контроля корректности таких записей на закладке "Дополнительно" существует флаг "Отбор по начислениям сотруднику", который позволяет отфильтровать записи табличной части "Отображение в бухучете".

Важно! Если документом "Ввод любого расчета" были записаны "нулевые" записи, эти записи попадут также на закладку "Начисления" ("Удержания") для дальнейшего, окончательного, расчета.

Важно! Перед началом работы с данным документом, для корректной работы механизмов расчета, необходимо заполнить актуальные на предприятии графики работы, хранящиеся в справочнике "Графики работы", либо ввести документ "Табель учета рабочего времени организации".
