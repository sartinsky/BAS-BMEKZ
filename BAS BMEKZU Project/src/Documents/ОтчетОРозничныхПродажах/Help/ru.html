Документ "Отчет о розничных продажах"

Документ предназначен для отражения розничных продаж. Документ можно ввести на основании документа Инвентаризация товаров на складе.

Отражение продаж и возвратов одновременно с приемом розничной выручки (склады вида "Оптовый" или "Розничный")
Отражение продажи по ранее принятой выручке (склады с видом "Неавтоматизированная торговая точка")

При вводе документа в шапке обязательно нужно указать следующие реквизиты:

  • Склад - торговая точка, в которой отражается розничная продажа.
  • Тип цен - цена товара, по которой оформляется реализация. Выбирается из нескольких допустимых для товара цен. Тип цен заполняется автоматически тем типом цен, который задан для выбранного склада. Тип цен можно изменить или задать, если тип цен для склада не установлен, по кнопке Редактировать цены и валюту документа. 
  • Статья ДДС - статья движения денежных средств, по которому будет отражено поступление розничной выручки в кассу организации.

Отражение продаж и возвратов одновременно с приемом розничной выручки

Указывается Вид операции ККМ.

На закладке Товары указываются товары и услуги, реализованные розничному покупателю. Указывается Количество и Цена проданных товаров (с НДС).
При возврате товаров указывается отрицательное Количество. Если возвращаются товары, проданные в предыдущие смены, указывается Документ продажи - "Отчет о розничных продажах" за смену, когда товар был продан.

На закладке Платежные карты и банковские кредиты указываются оплаты покупателя, произведенных платежной картой или с привлечением банковских кредитов. Для каждого Вида оплаты указывается Сумма оплат, а также Контрагент - банк, выдавший платежную карту или кредит, и его Договор с организацией.

Отражение продажи по ранее принятой выручке

Документ может быть введен на основании документа Инвентаризация товаров на складе. 

Указывается Вид операции НТТ.

На закладке Товары указываются товары и услуги, реализованные розничному покупателю. Указывается Количество и Цена проданных товаров (с НДС). Закладка Товары может быть заполнена по инвентаризации, для этого в заголовке документа должен быть указан документ Инвентаризация товаров на складе.
