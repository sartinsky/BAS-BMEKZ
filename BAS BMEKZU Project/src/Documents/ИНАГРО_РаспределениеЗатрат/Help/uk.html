Документ "Розподіл витрат"

Документ виконує регламентну процедуру по закриттю місяця. Функції документу:

  • Розподіл та списання загальновиробничих витрат
  • Розрахунок фактичної собівартості продукції та її списання по напрямкам видатків
  • Закриття виробничих витрат

Застосування даного документу можливе як у кінці, так і в продовж звітного періоду.

Щоб мати можливість керувати процесом розподілу, є наступні налаштування для документу:

  • Управління витратами
  • Управління базою розподілу
  • Управління розподілом

Дані налаштування відкривають доступ до відповідних закладок документу.

Закладка «Відбори» документу «Розподіл витрат» призначена для виконання відбору по витратам. Передбачена можливість вибору витрат по рахунку, номенклатурній групі, статті витрат, підрозділу. На цій же закладці вибирається база розподілу. Можливий відбір бази розподілу по рахунку, підрозділу, складу, номенклатурній групі, статті витрат, номенклатурі.

Витрати включають у себе: рахунок витрат, аналітику витрат. Якщо необхідно закрити не всі витрати, а тільки частину, тоді необхідно встановити прапорець «Застосовувати коефіцієнт» та вказати його нижче.

На закладках Витрати, База розподілу, Розподіл при натисканні на кнопку «Заповнити» буде виведена інформація згідно до вказаних даних на закладці «Відбори». При необхідності користувач може внести ручні зміни в кожен рядок.

Використовуючи  цей документ для кожної операції, необхідно буде ввести документ. Якщо ж структура розподілу не складна та не змінюється у періоді, то можна скористатися документом «Розподіл витрат та розрахунок собівартості».
