Документ "Распределение затрат"

Документ выполняет регламентную процедуру по закрытию месяца. Функции документа:

  • Распределение и списание общепроизводственных затрат
  • Расчет фактической себестоимости продукции и ее списание по направлениям расхода
  • Закрытие производственных затрат

Применение данного документа возможно как в конце, так и в течение отчетного периода.

Чтобы иметь возможность управлять процессом распределения, есть следующие настройки для документа:

  • Управление затратами
  • Управление базой распределения
  • Управление распределением

Данные настройки открывают доступ к соответствующим закладкам документа.

Закладка «Отборы» документа «Распределение затрат» предназначена для выполнения отбора по затратам. Предусмотрена возможность выбора затрат по счету, номенклатурной группе, статье затрат, подразделению. На этой же закладке выбирается база распределения. Возможен отбор базы распределения по счету, подразделению, складу, номенклатурной группе, статье затрат, номенклатуре.

Затраты включают в себя: затратный счет, аналитика затрат. Если необходимо закрыть не все затраты, а только часть, необходимо установить флаг «Применять коэффициент» и указать его ниже.

На закладках Затраты, База распределения, Распределение при нажатии на кнопку «Заполнить» будет выведена информация согласно указанным данным на закладке «Отборы». При необходимости пользователь может внести ручные изменения в каждой строке.

Используя этот документ на каждую операцию, необходимо будет ввести документ. Если же структура распределения не сложная и не меняется в периоде, то можно воспользоваться документом «Распределение затрат и расчет себестоимости».
