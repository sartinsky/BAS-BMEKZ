Документ "Распределение затрат и расчет себестоимости"

Документ позволяет выполнить следующие операции:

•         Распределение и списание общепроизводственных затрат;

•         Закрытие производственных затрат;

•         Расчет фактической себестоимости продукции и корректировку стоимости по направлениям расхода.

Для каждой строки табличной части документа указывается своя настройка для отбора затрат, способ их распределения и база распределения. Распределение выполняется автоматически в момент проведения документа. Расчет выполняется последовательно по каждой строке табличной части документа.

Корректировка результата распределения не предусмотрена.

Документ рекомендуется использовать, если структура распределения не сложная и не меняется в течение периода.
