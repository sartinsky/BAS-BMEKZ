Документ "Розподіл витрат та розрахунок собівартості"

Документ дозволяє виконати наступні операції:

•         Розподіл та  списання загальновиробничих витрат;

•         Закриття виробничих витрат;

•         Розрахунок фактичної собівартості продукції та коригування вартості по напрямкам видатків.

Для кожного рядку табличної частини документу вказується своє налаштування для відбору витрат, спосіб їх розподілу та база розподілу. Розподіл виконується автоматично під час проведення документу. Розрахунок виконується послідовно для кожного рядку табличної частини документу.

Коригування результату розподілу не передбачене.

Документ рекомендується використовувати, якщо структура розподілу не складна та не змінюється у періоді.
