"Характеристики видов культур"

План видов характеристик содержит полный список качественных характеристик зерна, определяемых при прохождении зерна через лабораторию предприятия.

Общие реквизиты

Содержит следующие реквизиты для заполнения:

·           «Наименование» - сокращенное наименование качественного показателя;

·           «Полное наименование» - полное наименование качественного показателя;

·           «Код» - заполняется автоматически программой;

·           «Тип значения» - тип данных который может принимать характеристика (число, строка, перечисление);

·           «Единица измерения» - указывается единица измерения характеристики;

·           «Тип характеристики» - указывается к какой базовой характеристики принадлежит данный показатель. Может принимать следующие значения: «Влажность», «Сорная примесь», «Зерновая примесь», «Клейковина количество», «Клейковина качество», «Белок», «Натура», «Цвет», «Запах», «Тип», «Под тип», «Стекловидность», «Зараженность», «Число падения» ,  «Вид зерновой примеси», «Вид сорной примеси», «Мелкое зерно», «Сажковое зерно», «Проход сита», «Клоп черепашки», «Не базовая».

Если признак «Показывать в складской квитанции» установлен, тогда данная характеристика будет отображаться в складской квитанции. Если установить признак «Показывать в журнале лабораторных анализов», тогда данная характеристика будет отображаться в отчете «Журнал лабораторных анализов».

 

На закладке «Настройки» в соответствующих реквизитах указываются: базисные кондиции для переработки, надбавка за отклонение от базиса (эти показатели используются документом «Акт о зачистке производственного корпуса и результаты переработки зерна» форма №117). Для идентификации характеристики при выгрузке складских квитанций в программу «ЕРЗС – Реестратор ІІІ» необходимо заполнить реквизиты «Код характеристики складской квитанции (DRU)», «Код характеристики складского свидетельства (DRU)», «Тип характеристики (DRU)». Для указания порядка следования характеристики в журнале лабораторных анализов по форме 49 служит реквизит "Порядок следования в журнале ф. 49".
