Передбачена для зазначення властивостей одного додаткового реквізиту залежно від значення інших реквізитів.

Для спільних додаткових реквізитів дана настройка недоступна.

Для введення нових умов натисніть кнопку Додати умову.

Таблиця для введення умов складається з колонок:

  • виберіть Реквізит зі списку за допомогою кнопки ;
  • виберіть Вид порівняння;
  • виберіть зі списку або напишіть потрібне Значення (в залежності від обраного реквізиту спосіб зазначення значення може змінюватися).

Якщо умова не влаштовує, приберіть його з таблиці за допомогою кнопки Видалити умову.
