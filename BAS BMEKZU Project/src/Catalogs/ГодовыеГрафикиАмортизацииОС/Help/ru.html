Справочник "Годовые графики амортизации"

Справочник содержит способ распределения годовой суммы амортизации по месяцам. Для каждого месяца можно установить долю годовой амортизации, относящейся на этот месяц. Доля расчитывается относительно общей суммы долей всех месяцев, то есть для каждого месяца доля может быть больше единицы.

Годовые графики могут быть установлены для основных средств в момент принятия к учету в документе "Ввод в эксплуатацию ОС", в дальнейшем их можно изменить в документе "Изменение графиков амортизации ОС".
