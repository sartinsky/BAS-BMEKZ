Довідник "Види взаєморозрахунків"

Довідник призначений для додаткового поділу взаєморозрахунків для аналізу даних у звітах.

З допомогою цього довідника можна провести додаткову класифікацію однотипних договорів різних контрагентів.

Наприклад, для довгострокових договорів встановити вид взаєморозрахунків - довгострокові, а для разових договорів встановити вид взаєморозрахунків - разові.
