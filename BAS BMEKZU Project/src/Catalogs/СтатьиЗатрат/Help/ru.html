Справочник "Статьи затрат"

Справочник статей затрат предназначен для хранения перечня статей затрат.

Проведение документов программы будет зависеть от установок реквизитов этого справочника. Поэтому не рекомендуется менять значение реквизитов, после того как элемент справочника был использован в проведенных документах. В противном случае документы необходимо перепровести.

 

  • Вид затрат - элемент затрат:
    • Материальные;
    • Оплата труда;
    • Отчисления на социальные нужды;
    • Амортизация;
    • Прочие.
  • Счет 8 класса (бух.учет) - счет 8 класса, соответствующий элементу затрат. При использовании счета 8-го класса, при регистрации затрат по статье автоматически формируется проводка по счету 8-го класса, указанному в статье.
  • Постоянная затрата - флаг заполняется для постоянных общепроизводственных затрат. В соответствии с НП(С)БУ 16, постоянные ОПЗ между прямыми производственными затратами и себестоимостью реализации пропорционально нормальной мощности.
  • Статья налоговой декларации - использовалась до 01.01.2015 г. для заполнения Декларации о прибыли.
