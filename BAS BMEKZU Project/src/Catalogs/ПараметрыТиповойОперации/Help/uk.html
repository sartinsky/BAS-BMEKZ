Довідник "Параметри типової операції"

Довідник призначений для зберігання списку параметрів, які використовуються для заповнення документа Операція. Даний довідник підпорядкований довіднику "Типові операції". У формі елемента використовується наступні поля:

  • Ім'я параметра містить назву параметра, як воно буде відображатися на формі документа Операція
  • Тип параметра містить тип обраного значення
  • Зв'язок по власнику містить параметр власник. Наприклад для параметра Договір, можна вказати параметр Контрагент, який буде встановлюватися в якості обмеження при виборі значення Договору
  • Підказка використовується для опису параметра. Відображається або в підказці введення, або в підказці по кнопці в документі Операція для даного параметра
