Довідник "Склади (місця зберігання)"

В довіднику міститься список місць зберігання товарів. Кожен елемент довідника описує фізичне місце зберігання (ангар, майданчик, кімната тощо). 

Аналітичний облік в програмі можна вести по складах або по підприємству в цілому. Спосіб ведення партіонного обліку визначається в настройках параметрів обліку.

Довідник має багаторівневу, ієрархічну структуру. Групи довідника можна створювати самостійно.

У довідник може бути введено будь-яку кількість складів. Інформація хоча б про один склад повинна бути присутньою обов'язково. Один зі складів рекомендується вказати в якості основного складу. Зазначений склад буде використовуватися за умовчанням при введенні господарських операцій.

При введенні складу можна заповнити реквізити:

  • Відповідальна особа - матеріально-відповідальна особа. Переглянути список матеріально-відповідальних осіб обраного складу за різні періоди можна по кнопці Історія.
  • Вид складу:
    • Гуртовий - склади для гуртової торгівлі.
    • Роздрібний - автоматизована торгівельна точка. Торгівельна точка вважається автоматизованою, якщо засоби технічного забезпечення або специфіка торговельної діяльності дають змогу щоденно формувати детальний звіт про продані товари для подальшого введення в інформаційну базу.
    • Неавтоматизована торгівельна точка - роздрібна торгівельна точка вважається неавтоматизованої (НТТ), якщо щоденної реєстрації проданих товарів в ній не ведеться.
  • Тип цін - для кожного складу (роздрібного, гуртового і неавтоматизованої торгівельної точки) може бути заданий тип цін, за якими буде здійснюватися продаж в роздріб. Тип цін роздрібної торгівлі використовується для автоматичного заповнення цін в документах, що відображають продаж товарів у роздріб.
  • Підрозділ - підрозділ, якому належить склад.

Список відповідальних осіб складу за різні періоди і рахунки обліку номенклатури можна переглянути за посиланням Рахунки обліку номенклатури панелі навігації.
