Справочник "Агрегаты"

Данный справочник содержит перечень агрегатов, используемых элеватором в процессе работы. Используется в технологических документах для уточнения способа обработки сырья.

Общие реквизиты документа

Для каждого элемента справочника заполняются следующие реквизиты:

·           «Код».  Заполняется автоматически программой;

·           «Наименование». Наименование агрегата.
