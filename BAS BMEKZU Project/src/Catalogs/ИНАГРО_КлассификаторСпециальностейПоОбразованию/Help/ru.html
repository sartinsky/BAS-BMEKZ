Справочник «Классификатор специальностей по образованию»

В этом справочнике хранится список специальностей по образованию.

Справочник вызывается для ввода и редактирования информации из пункта меню "Справочники" / "Классификатор специальностей по образованию".

Справочник имеет одноуровневую структуру. Редактирование и ввод новых элементов производится непосредственно в форме списка справочника.
