Перегляд відомостей про папку файлів.

Відомості про папку

  • Заповніть Найменування папки. Поле є обов'язковим;
  • Одна папка може входити в іншу. При необхідності в полі В папці виберіть папку вищого рівня зі списку;
  • Поле Повний шлях заповнюється програмою автоматично;
  • У полі Опис напишіть детальну інформацію про папці;
  • У полі Відповідальний виберіть зі списку користувача, відповідального за папку файлів;
  • У полі Дата створення автоматично проставляються дата і час створення папки.
  • При необхідності виберіть Робочий каталог для папки файлів. У цьому випадку після редагування файли цієї папки будуть зберігатися окремо, в зазначеному робочому каталозі.

Права доступу до папки

  • Папкам можна призначати різні права доступу. Для переходу до роботи над правами доступу папки натисніть Настроїти права.

Швидкий перехід до списку файлів

  • Натисніть Відкрити список папок і файлів, для того щоб перейти в список Файли.
