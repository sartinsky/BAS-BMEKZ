Справичник "Список статей затрат, включенных в базу распределения косвенных расходов"

Каждый элемент справочника представляет собой список прямых производственных статей затрат, являющихся базой распределения общепроизводственных затрат.

Соответствующий элемент справочника указывается в регистре сведений Методы распределения косвенных расходов организаций, если указан Способ распределения Отдельные статьи прямых затрат. В этом случае при распределении общепроизводственных затрат документом Закрытие месяца в качестве базы распределения используются остатки и обороты 23-го счета по статьям затрат из списка.
