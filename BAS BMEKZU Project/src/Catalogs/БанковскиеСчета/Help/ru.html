Справочник "Банковские счета"

В справочнике "Банковские счета" хранятся банковские счета всех юридических и физических лиц: собственных и сторонних.

Справочник "Банковские счета" подчинен справочникам «Организации» и «Контрагенты».

Переход к списку банковских счетов собственной организации или контрагента производится по ссылке Все банковские счета в форме элемента соответствующего справочника. 

Для каждого контрагента или собственной организации можно выбрать т.н. «основной» банковский счет, который будет подставляться по умолчанию в платежные документы.

Редактирование информации о банковском счете производится в отдельном диалоговом окне, открываемом из формы списка справочника банковских счетов.

При этом указывается информация, характеризующая банковский счет (банк, номер счета, вид счета, валюта денежных средств), а также дополнительная информация, необходимая для корректного заполнения платежных документов по выбранному счету.
