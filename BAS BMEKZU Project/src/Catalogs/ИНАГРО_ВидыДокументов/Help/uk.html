Довідник "Види документів"

Спеціальний службовий довідник. Даний довідник служить для визначення  способів нумерації документів.

У списку відображені всі документи, які входять в інформаційну базу

 «Спосіб нумерації документів» служить для визначення способу установки номера документа. Може приймати значення:

  •  «С префиксом вида урожая»,
  •  «За реквізиту документа» ,
  •  «З префіксом дати документа»,
  •  «По реквізиту документа і з префіксом виду врожаю».

«Ім'я реквізиту номера документа» - вказувати з якого реквізиту необхідно брати значення для формування номера при обраному способі формування «За реквізиту документа».
