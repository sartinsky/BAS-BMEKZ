Справочник "Виды хранения"

Справочник содержит список видов хранения зерна на предприятии. Как правило, с помощью элементов данного справочника разделяют коммерческое хранение от некоммерческого. Например: «Обычное хранение», «Госрезерв» и «Переработка». В дальнейшем справочник используется для обеспечения возможности провести анализ по каждому виду хранения.  

Общие реквизиты документа

Справочник содержит следующие реквизиты для заполнения:

·           «Код». Заполняется автоматически программой;

·           «Наименование» вида хранения.
