Довідник "Види зберігання"

Довідник містить список можливих видів зберігання зерна на підприємстві. Зазвичай, за допомогою елементів цього довідника розділяють комерційне зберігання від  некомерційного. Наприклад: «Звичайне зберігання», «Держрезерв» та «Переробка». В подальшому довідник використовується для забезпечення можливості проводити аналіз по кожному виду зберігання.  

Загальні реквізити довідника

Довідник містить наступні реквізити для заповнення:

·           «Код». Заповнюється автоматично програмою;

·           «Найменування» виду зберігання.
