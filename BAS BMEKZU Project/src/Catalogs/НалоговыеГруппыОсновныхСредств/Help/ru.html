Справочник "Налоговые группы ОС"

Справочник содержит налоговые группы основных средств. В справочнике Основные средства для каждого объекта ОС указывается налоговая группа.

В настоящее время налоговая группа ОС не влияет на начисление амортизации и на формирование регламентированных печатных форм. Налоговая группа ОС указывается в справочных целях и для анализа данных в отчетах.
