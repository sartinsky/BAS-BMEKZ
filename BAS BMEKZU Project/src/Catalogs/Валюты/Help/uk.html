Список призначений для ведення валют, що використовуються в програмі. Необхідні для роботи валюти можна підібрати з Класифікатора валют (рекомендується) або додати вручну.

Відкрити список валют можна з розділу Довідники.

У списку виводиться:

  • Найменування валюти - повне найменування валюти.
  • Цифр. код - унікальний числовий код, прийнятий міжнародним стандартом, призначений для ідентифікації валюти при завантаженні курсів з мережі Інтернет.
  • Симв. код - прийнятий міжнародним стандартом, використовується як основне представлення валюти.
  • Курс - значення курсу на поточну дату.
  • Кратність - по умовчанню заповнене одиницею і не виводиться. Якщо курс будь-якої валюти дуже низький (точність становить менше чотирьох знаків дробової частини), то в цьому полі вказується кількість котирувальних одиниць.

Додавання валюти з класифікатора валют

  • Натисніть Створити - За класифікатором (рекомендується для коректного введення валюти).
  • Наступні поля нової валюти будуть заповнені з класифікатора автоматично: Найменування, Символьний код, Цифровий код, положення перемикача Курс валюти (буде встановлено положення завантажується з інтернету, якщо є можливість завантажувати курс цієї валюти з мережі Інтернет).

Введення валюти не з класифікатора (вручну)

  • Натисніть Створити-Нову. Також можна натиснути клавішу Ins.
  • Заповніть відомості про валюту.
