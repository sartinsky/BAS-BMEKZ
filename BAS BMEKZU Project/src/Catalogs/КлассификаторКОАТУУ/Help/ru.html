Справочник "Классификатор КОАТУУ"

В справочнике хранится Классификатор объектов административно-территориального устройства Украины. Классификатор используется для формирования Декларации акцизного налога субъектами хозяйствования розничной торговли.

Справочник может быть заполнен по кнопке "Подбор из классификатора".

Для каждой территории в справочнике должна быть указана Налоговая инспекция.
