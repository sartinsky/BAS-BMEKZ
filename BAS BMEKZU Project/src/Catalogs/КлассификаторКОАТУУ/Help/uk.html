Довідник "Класифікатор КОАТУУ"

У довіднику зберігається Класифікатор об'єктів адміністративно-територіального устрою України. Класифікатор використовується для формування Декларації акцизного податку суб'єктами господарювання роздрібної торгівлі.

Довідник може бути заповнений по кнопці "Підбір з класифікатора".

Для кожної території в довіднику повинна бути вказана Податкова інспекція.
