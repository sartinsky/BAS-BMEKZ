Справочник "Схемы налогового учета по договорам контрагентов"

Справочник "Схемы налогового учета по договорам контрагентов" позволяет описать моменты определения базы налогообложения НДС. Момент определения базы налога задается отдельно для операции приобретения и продажи и может быть выбран из следующего списка:

  • по первому событию;
  • по отгрузке;
  • по оплате;
  • не определять.


Схемы налогового учета используются при оформлении договоров контрагентов.
