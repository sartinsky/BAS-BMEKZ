Довідник "Схеми податкового обліку по договорах контрагентів"

Довідник "Схеми податкового обліку по договорах контрагентів" дозволяє описати моменти визначення бази оподаткування ПДВ. Момент визначення бази податку визначається окремо для операції придбання та продажу і може бути вибраний з наступного списку:

  • за першою подією;
  • по відвантаженню;
  • по оплаты;
  • не визначати.


Схеми податкового обліку використовуються при оформленні договорів контрагентів.
