Справочник "Происхождение зерна"

Справочник содержит список регионов с которых завозилось зерно для хранения на предприятии.

Общие реквизиты справочника

Справочник содержит следующие реквизиты для заполнения:

·           «Код». Заполняется автоматически программой;

·           «Наименование» - наименование региона.
