Справочник "Виды культур для расчета убыли при хранении"

Согласно Инструкции «О порядке ведения учета и оформления операций с зерном и продуктами его переработки на предприятиях хлебопродуктов» расчет естественной убыли проводится исходя с таблицы  «Нормы естественной убыли зерна, продуктов его переработки …». Справочник содержит перечень культур, для которых в инструкции определены нормы. Данные справочника используются при расчете естественной убыли.

Общие реквизиты документа

Для каждого элемента справочника заполняются следующие реквизиты:

·           «Код».  Заполняется автоматически программой;

·           «Наименование». Наименование вида культур.
