Довідник"Види культур"

Довідник містить перелік видів зернових культур, погрупованих по однаковим властивостям, таким як базисні показники, якісні характеристики.

Загальні реквізити довідника

Для кожного елементу довідника заповнюються наступні реквізити:

·           «Код».  Заповнюється автоматично програмою;

·           «Найменування». Найменування виду культур;

·           «ДСТУ» - назва культури по ДСТУ;

·           «Вид культури для розрахунку вартості послуг» - елемент довідника «Види культур для розрахунку вартості послуг»;

·           «Вид культури для розрахунку природного убутку» - елемент довідника «Види культур для убутку»;

·           «Непридатні відходи» - елемент довідника «Номенклатура»;

·           Група реквізитів «Базисні кондиції» - відображає поточне значення кондицій, які повинні бути у зерна в заліковій вазі;

·           «Доводить до якості, розраховувати послуги» - відображає поточне значення показників для розрахунку послуг.

В табличній частині довідника необхідно вказати список характеристик, притаманних цьому виду культури. Реквізит «Вид характеристики» - елемент плану видів характеристик «Характеристики видів культур».

Вибрані значення реквізитів «ст.вол-ті», «ст.зас-ті», «ст.зара-ня» будуть по замовчуванню автоматично проставлятися в товарно-транспортних накладних як значення експрес аналізу зерна, що ввозиться на підприємство.

·           «ст.вол-ті» - може приймати наступні значення: «Сухе», «Середня сухість», «Вологе», «Сире»;

·           «ст.зас-ті» - може приймати наступні значення: «Чисте», «Середнє забруднення», «Брудне»;

·           «ст.зара-ня» - може приймати наступні значення: «Ні», «Перша», «Друга» , «Третя».
