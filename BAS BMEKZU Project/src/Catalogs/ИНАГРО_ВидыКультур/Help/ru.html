Справочник "Виды культур"

Справочник содержит перечень видов зерновых культур, сгруппированных по одинаковым свойствам таких как  базисные показатели, качественные характеристики.

Общие реквизиты документа

Для каждого элемента справочника заполняются следующие реквизиты:

·           «Код».  Заполняется автоматически программой;

·           «Наименование». Наименование вида культур;

·           «ГОСТ» - название культуры по ГОСТу;

·           «Вид культуры для расчета стоимости услуг» - элемент справочника «Виды культур для расчета стоимости услуг»;

·           «Вид культуры для расчета естественной убыли» - элемент справочника «Виды культур для убыли»;

·           «Негодные отходы» - элемент справочника «Номенклатура»;

·           Группа реквизитов «Базисные кондиции» - отображается текущее значение кондиций, которые должны быть у зерна в зачетном весе;

·           «Доводить до качества, рассчитывать услуги» - отображается текущее значение показателей для расчета услуг.

В табличной части справочника необходимо указать список характеристик, присущих этому виду культуры. Реквизит «Вид характеристики» - элемент плана видов характеристик «Характеристики видов культур».

Выбранные значения реквизитов «ст.вл-ти», «ст.заг-ия», «ст.зара-ти» будут по умолчанию автоматически проставляться в товарно-транспортных накладных как значения экспресс анализа ввозимого на предприятие зерна.

·           «ст.вл-ти» - может принимать следующие значения: «Сухое», «Средняя сухость», «Влажное», «Сырое»;

·           «ст.заг-ия» - может принимать след\ующие значения: «Чистое», «Среднее загрязнение», «Грязное»;

·           «ст.зара-ти» - может принимать следующие значения: «Нет», «Первая», «Вторая» , «Третья».
