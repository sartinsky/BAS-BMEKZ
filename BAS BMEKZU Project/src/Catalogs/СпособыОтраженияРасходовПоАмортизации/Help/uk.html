Довідник "Способи відображення витрат по амортизації"

Довідник містить способи віднесення на витрати підприємства:

  • сум амортизації основних засобів, нематеріальних активів та малоцінних активів;
  • сум поточних ремонтів та поліпшення понад ліміту основних засобів.

Перелік способів задається у табличній частині "Способи". Кожен рядок табличної частини визначає рахунки та аналітику для віднесення на витрати, а також податкові призначення витрат. У тому випадку якщо введено кілька рядків, витрати по амортизації спочатку будуть розподілені по рядках (способах) згідно з коефіцієнтами розподілу. Коефіцієнти розподілу вказуються в кожному рядку. У разі якщо в табличній частині зазначений тільки один рядок (тобто задано лише один спосіб), коефіцієнт слід встановлювати рівним одиниці.

Напрямок віднесення на витрати сум амортизації включає Рахунок витрат і субконто цього рахунку.

Для віднесення на витрати сум поточних ремонтів та поліпшення понад ліміт ОЗ вказується Стаття витрат на поліпшення ОЗ. Рахунок витрат і субконто рахунку, за винятком субконто Стаття витрат, використовуються ті ж, що і для віднесення на витрати сум амортизації.

Для основних засобів спосіб відображення витрат по амортизації може бути встановлений в документах: "Введення в експлуатацію ОЗ", "Зміна способів відображення витрат по амортизації". Для нематеріальних активів - в документах "Прийняття до обліку НМА", "Зміна способу відображення витрат по амортизації НМА ".

Спосіб відображення витрат на ремонт та поліпшення ОЗ може бути додатково задано в документі Модернізація і ремонт ОЗ. Якщо окремий спосіб відображення в документі не вказано, використовується спосіб, раніше зазначений у документі Введення в експлуатацію ОЗ (Зміна способів відображення витрат по амортизації)
