Призначена для вибору способу створення нового файлу.

Виводиться по кнопці Insert або команди Добавить контекстного меню по правій кнопці миші на списку файлів або списку приєднаних файлів.

Способи створення нового файлу

Виберіть за допомогою перемикача:

  • З шаблону - новий файл буде створений на основі обраного шаблону;
  • З файлу на диску - новий файл буде створений з існуючого на локальному або мережевому диску файлу;
  • Зі сканера - передбачена можливість створити новий файл, отримавши графічний файл зі сканера (цей параметр доступний, якщо до комп'ютера підключено сканер). Для завантаження файлів зі сканера необхідно також настроїти сканування.

Натисніть кнопку Створити для переходу до додавання файлу.
