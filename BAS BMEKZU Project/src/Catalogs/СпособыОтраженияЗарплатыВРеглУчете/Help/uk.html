Способи відображення зарплати в регламентованому обліку

У довіднику задаються господарські операції, які можна поставити у відповідність розрахунковим величинам. при відображенні заробітної плати.

 

Господарська операція складається з однієї проводки з бухгалтерського обліку та податкового призначення для податкового обліку.

 

Також для способів відображення нарахувань вказується порядок формування відображення внесків у фонди для таких нарахувань:

  • Відображається тим же способом, що і нарахування;
  • Відображається як основне нарахування співробітника;
  • Відображається особливим способом (зазначається, яким саме). 

Додаткові параметри для проводок по внесках визначаються за даними реєстру відомостей "Статті витрат за внесками ФОП" - тут кожному податку (виду внеску) і бухгалтерського рахунку витрат ставиться у відповідність своя стаття витрат для внесків.
