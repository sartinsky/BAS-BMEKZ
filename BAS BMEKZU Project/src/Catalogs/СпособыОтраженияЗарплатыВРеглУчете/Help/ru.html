Способы отражения зарплаты в регламентированном учете

В справочнике задаются хозяйственные операции, которые можно поставить в соответствие расчетным величинам. при отражении заработной платы.

 

Хозяйственная операция состоит из одной проводки по бухгалтерскому учету и налогового назначения для налогового учета.

 

Также для способов отражения начислений указывается порядок формирования отражения взносов в фонды для таких начислений:

  • Отражается тем же способом, что и начисление;
  • Отражается как основное начисление сотрудника;
  • Отражается особым способом (указывается, каким именно). 

Дополнительные параметры для проводок по взносам определяется по данным регистра сведений "Статьи затрат по взносам ФОТ" - здесь каждому налогу (виду взноса) и бухгалтерскому счету затрат ставится в соответствие своя статья затрат для взносов.
