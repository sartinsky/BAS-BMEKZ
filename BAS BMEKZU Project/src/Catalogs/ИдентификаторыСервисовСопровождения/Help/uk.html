Призначений для зберігання і перегляду ідентифікаторів сервісів супроводу. Дані списку не редагуються, призначені для забезпечення роботи програми.

Список заповнюється автоматично при першому запуску (і кожному оновленні) конфігурації, згідно зі списком доступних для конфігурації сервісів, а також автоматично актуалізується при кожному оновленні версії конфігурації (враховуються перейменовані, додані і вилучені об'єкти метаданих).

Натисніть для перегляду докладних відомостей про ідентифікатор сервісу супроводу.
