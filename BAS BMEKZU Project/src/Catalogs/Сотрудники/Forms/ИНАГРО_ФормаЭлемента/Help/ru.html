Карточка сотрудника

В карточке сотрудника указывается организация, с которой оформляется трудовой договор.

Фамилия, имя и отчество сотрудника вносятся в одно поле, одной строкой, однако
программой они разбиваются на три строки (фамилия, имя и отчество) и хранятся отдельно.

Если сотрудник изменил фамилию, имя или отчество, то это следует отразить в специальной форме по команде "изменить ФИО". Просмотреть и отредактировать историю изменения ФИО можно в форме по команде "История ФИО".

В форме указываются основные данные сотрудника, такие как дата рождения, пол, код по ДРФО. Другие личные данные указываются в специальной форме, которая открывается из раздела "Перейти".

Для вновь внесенных в программу сотрудников доступны команды для создания документов приема на работу.

Табельный номер формируется программой автоматически при записи нового сотрудника, как следующий по порядку за последним, который уже есть в программе. При необходимости его можно отредактировать вручную.

Для сотрудников, которые уже оформлены как принятые на работу по трудовому договору соответствующим документом программы, указывается дата приема и информация о текущем месте работы. По ссылке можно посмотреть историю изменения мест работы. Для уволенных сотрудников указывается дата увольнения.

С помощью меню "Оформить документ" можно создать различные документы для этого сотрудника.

Если с сотрудником в программе регистрируется несколько трудовых договоров, т.е. для одного человека создается несколько сотрудников, то для того чтобы быстро различать таких сотрудников в списках и отчетах программы, можно задать произвольную строку, которая будет дополнять представление (название) сотрудника. Например, для совместителя можно указать "(совм.)", а если сотрудник принимается повторно после увольнения, то для прошлого сотрудника можно указать "(устар.)".

"Устаревших" сотрудников дополнительно можно и поместить "в архив", с помощью специального флага. Тогда по умолчанию этот сотрудник вообще не будет показываться в списках.
