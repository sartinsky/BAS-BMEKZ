Справочник Спецификации номенклатуры предназначен для хранения спецификаций – состава изделий.

В спецификации можно указать изготавливаемую продукцию (выходное изделие) и список материалов, необходимых для изготовления продукции. Для производства одного вида продукции может использоваться несколько спецификаций.

В заголоке спецификации указываются:

  • Наименование спецификации - используется для выбора спецификации в производственных документах. Наименование спецификации может не совпадать с наименованием производимой продукции.
  • Выходное изделие - наименование продукции.
  • Количество - количество выходных изделий, на производство которых рассчитана спецификация.

Заполняется списком используемых материалов, полуфабрикатов (из справочника Номенклатура)

Реквизиты:

  • Номенклатура – используемый материал или полуфабрикат.
  • Количество – указать количество, необходимое на выполнение производственного процесса согласно спецификации.
