Довідник "Зовнішнє обладнання"

Спеціальний службовий довідник. Служить для визначення списку зовнішнього обладнання, підключеного до інформаційної бази (Електронні ваги, Web камери)

Закладка «Драйвер» визначає параметри необхідні для з'єднання зовнішнього обладнання

  • «Найменування» - ім'я комп'ютера до якого безпосередньо підключено зовнішнє обладнання;
  • «Вид» - реквізит, який визначає вид зовнішнього обладнання. Може приймати значення «Ваги» і «Камера»;
  • «Файл драйвера» - визначає місце розташування та ім'я файлу-драйвера зовнішнього обладнання;
  • «Файл обробки» - визначає місце розташування і ім'я зовнішньої обробки для зчитування показників з зовнішнього обладнання.

 

Закладка «Налаштування портів» містить значення необхідні для правильної побудови обробкою запиту до драйвера зовнішнього обладнання. Дана закладка заповнюється відповідно специфікації, що поставляється разом із зовнішнім обладнанням.
