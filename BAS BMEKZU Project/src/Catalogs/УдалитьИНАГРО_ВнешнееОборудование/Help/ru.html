Справочник "Внешнее оборудование"

Специальный служебный справочник. Служит для определения списка внешнего оборудования , подключённого к информационной базе (Электронные весы, Web камеры )

Закладка «Драйвер» определяет  параметры необходимые для соединения внешнего оборудования

  • «Наименование» - имя комп’ютера к которому непосредственно подключено внешнее оборудование;
  • «Вид»- реквизит, который определяет вид внешнего оборудования. Может принимать значения «Весы» и «Камера» ;
  • «Файл драйвера» - определяет месторасположение и имя файла –драйвера внешнего оборудования;
  • «Файл обработки» - определяет месторасположения и имя внешней обработки для считывания показателей с внешнего оборудования.

 

Закладка «Настройка портов» содержит значения необходимые для правильного построения обработкой запроса к драйверу внешнего оборудования . Даная закладка заполняется соответственно спецификации, поставляемой вместе с внешним оборудованием.
