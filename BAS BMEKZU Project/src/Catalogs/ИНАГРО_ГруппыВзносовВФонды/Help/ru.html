Справочник «Группы взносов в фонды»

Справочник используется в карточке сотрудников организации для определения взносов выбранному работнику и предназначен для хранения наименования групп взносов в фонды. Справочник вызывается для ввода и редактирования информации из пункта меню "Справочники" / "Группы взносов в фонды".

Справочник имеет одноуровневую структуру. Редактирование и ввод новых элементов производится непосредственно в форме списка справочника.
