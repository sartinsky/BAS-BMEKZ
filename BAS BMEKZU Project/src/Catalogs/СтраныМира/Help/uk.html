Призначений для зберігання інформації про країни світу, або регіонах або частинах країн. Дані довідника використовуються, наприклад, при вказівці країни адреси контактної інформації, країни походження товару.

Відкривається по команді Країни світу розділу Довідники. У списку можна зберігати тільки кілька необхідних для роботи країн.

Додавання країни з класифікатора

  • Натисніть Додати з класифікатора.

Введення вручну

У деяких випадках необхідно ввести якусь область, або співдружність країн, наприклад, Євросоюз.

  • Не рекомендується вводити вручну країни, наявні в класифікаторі, оскільки ручне введення збільшує можливість помилок.
  • Для того щоб додати нову країну в список країн, виконайте команду Ще - Додати вручну.
  • Заповніть необхідні поля.
