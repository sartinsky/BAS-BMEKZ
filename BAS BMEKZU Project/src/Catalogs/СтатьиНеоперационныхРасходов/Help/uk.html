Довідник "Статті неопераційних витрат"

Довідник "Статті неопераційних витрат" призначений для ведення аналітичного обліку на рахунках неопераційних витрат. Такими рахунками є:

  • рахунок 85 "Інші витрати" для організацій, що використовують тільки 8-й клас рахунків витрат;
  • рахунки 95-99 для організацій, що застосовують 9-й клас.
