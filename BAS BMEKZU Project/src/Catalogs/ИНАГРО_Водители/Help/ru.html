Справочник "Водители"

Вспомогательный справочник, содержащий список водителей, ввозивших или вывозивших что-либо через проходную элеватора. Служит для автоматического заполнения реквизитов товаротранспортных накладных: «Автопредприятие», «Номер транспортного средства».  

 

Общие реквизиты документа

Содержит следующие реквизиты для заполнения:

·           «Код». Заполняется автоматически программой;

·           «Наименование» - фамилия, имя и отчество водителя;

·           «Автопредприятие» - элемент справочника «Контрагенты»;

·           «Номер ТС» - поле для ввода номера транспортного средства;

·           «Транспорт» - элемент справочника «Транспортные средства».
