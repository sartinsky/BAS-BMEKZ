Довідник "Водії"

Допоміжний довідник, що містить список водіїв, які ввозили чи вивозили що-небудь за прохідну елеватора. Слугує для автоматичного заповнення реквізитів товаротранспортних накладних: «Автопідприємство», «Номер транспортного засобу».  

 

Загальні реквізити довідника

Довідник містить наступні реквізити для заповнення:

·           «Код». Заповнюється автоматично програмою;

·           «Найменування» - прізвище, им'я та по батькові водія;

·           «Автопідприємство» - елемент довідника «Контрагенти»;

·           «Номер ТЗ» - поле для вводу номеру транспортного засобу;

·           «Транспорт» - елемент довідника «Транспортні засоби».
