Справочник "Коды номенклатуры для НН"

В этом справочнике хранится информация о кодах номенклатуры для заполнения налоговых накладных.  Справочник подчинен справочнику "Номенклатура". Данные этого справочника используются при заполнении документов "Налоговая накладная" и "Приложение 2 к налоговой накладной".
