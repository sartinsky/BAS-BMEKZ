Обов'язково вказувати номер ВМД для импортых товарів?

Як вводиться інформація про коди УКТ ЗЕД та номери ВМД для імпортних товарів, для яких ведеться облік за номерами ВМД?

У формі редагування вибирається код УКТЗЕД з довідника Класифікатор УКТЗЕД та номер ВМД з довідника Номери вантажних митних декларацій.

Як вводиться інформація про коди УКТ ЗЕД для вітчизняних підакцизних товарів?

У формі редагування вибирається код УКТЗЕД з довідника Класифікатор УКТЗЕД. Номер ВМД не заповнюється.

Чи обов'язково вказувати номер ВМД для импортых товарів?

В період до набрання чинності вимоги вказувати номер та дату ВМД у податкових документах можна як вказувати, так і не вказувати номер ВМД для імпортних товарів.

  • Якщо при придбанні товарів не зазначається номер ВМД, в інформаційній базі ведеться облік залишків товарів в розрізі кодів УКТЗЕД. Товари надходять і вибувають по порожньому номеру ВМД. 
  • Якщо при придбанні товарів зазначається номер ВМД, ведеться облік залишків товарів в розрізі кодів УКТЗЕД та номерів ВМД. При вибутті товарів автоматично визначаються номера ВМД. Бухгалтеру підприємства відомі залишки імпортних товарів на будь-який момент часу; покупцю за його запитом можуть бути повідомлені номера ВМД поставлених йому товарів (форма Реєстр кодів УКТЗЕД та номерів ВМД документа Реалізація товарів і послуг). В друкованих формах податкових документів наявність номерів ВМД регулюється настройкою облікової політики
