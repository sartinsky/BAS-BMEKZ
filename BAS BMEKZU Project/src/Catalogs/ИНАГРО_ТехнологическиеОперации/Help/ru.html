Справочник "Технологические операции"

Справочник технологических операций предназначен для хранения перечня используемых на предприятии технологических операций. Данные для каждой технологической операции могут хранится в трех регистрах сведений:

  • Виды работ ручной труд;
  • Виды работ сельхозтехники;
  • Виды работ автотранспорта.

В поле "Технологический цикл" следует выбрать технологический цикл из справочника "Технологические циклы", частью которого является данная технологическая операция.

Для корректной работы системы рекомендуется заполнить указанные регистры данными, принятыми на предприятии.
