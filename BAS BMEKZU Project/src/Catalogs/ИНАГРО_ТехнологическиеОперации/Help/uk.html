Довідник "Технологічні операції"

Довідник технологічних операцій призначений для зберігання переліку використовуваних на підприємстві технологічних операцій. Дані для кожної технологічної операції можуть зберігатися в трьох регістрах відомостей:

  • Види робіт ручна праця;
  • Види робіт сільгосптехніки;
  • Види робіт автотранспорту.

В полі "Технологічний цикл" слід обрати технологічний цикл із довідника "Технологічні цикли", частиною якого є дана технологічна операція.

Для коректної роботи системи рекомендується заповнити вказані регістри даними, прийнятими на підприємстві.
