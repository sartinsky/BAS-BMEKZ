Справочник «Причины нетрудоспособности»

Справочник содержит перечень причин нетрудоспособности. Доступен из меню "Кадры" - "Параметры начислений".

Данные справочника используются в документе "Начисление по больничному листу" при расчете больничных.

Для ввода новой причины нетрудоспособности необходимо ввести следующие данные:

  • Наименование;
  • Количество месяцев, за которые будут браться начисления для расчета;
  • Число дней больничного, оплачиваемых за счет организации;
  • Вид расчета оплаты больничного за счет фонда социального страхования;
  • Вид расчета оплаты больничного за счет организации.
