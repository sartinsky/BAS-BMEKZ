Довідник «Причини непрацездатності»

Довідник містить перелік причин непрацездатності. Доступний з меню "Кадри" - "Параметри нарахувань".

Дані довідника використовуються в документі "Нарахування по лікарняному листу" при розрахунку лікарняних.

Для введення нової причини непрацездатності необхідно ввести наступні дані:

  • Найменування;
  • Кількість місяців, за які братимуться нарахування для розрахунку;
  • Число днів лікарняного, оплачуваних за рахунок організації;
  • Вид розрахунку оплати лікарняного за рахунок фонду соціального страхування;
  • Вид розрахунку оплати лікарняного за рахунок організації.
