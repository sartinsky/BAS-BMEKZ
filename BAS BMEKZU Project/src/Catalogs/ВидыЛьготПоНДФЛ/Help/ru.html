Виды льгот по НДФЛ

Справочник содержит список налоговых социальных льгот, которые могут быть применены при расчете НДФЛ. Код каждого элемента соответствует номеру статьи, которая описывает эту льготу в Налоговом кодексе.

"Код Отчетности" соответствует требованиям регламентированной формы 1ДФ.

"Размер" - указывает весовую долю применения льготы.
