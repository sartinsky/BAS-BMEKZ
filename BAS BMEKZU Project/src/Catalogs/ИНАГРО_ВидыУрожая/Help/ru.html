Справочник "Виды урожая"

Справочник содержит список возможных видов урожая. Фактически, каждый элемент справочника определяет год сбора урожая зерна.

Общие реквизиты справочника

При вводе нового элемента справочника необходимо заполнить следующие реквизиты:

  • «Код» - присваивается автоматически;
  • «Наименование» - внутреннее наименование вида;
  • «Год» - используется для определения года партии урожая.
