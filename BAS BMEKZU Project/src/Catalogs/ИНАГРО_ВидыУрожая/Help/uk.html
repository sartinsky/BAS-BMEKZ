Довідник "Види врожаю"

Довідник містить список можливих видів врожаю. Фактично, кожний елемент довідника визначає рік збору врожаю зерна.

Загальні реквізити довідника

Під час введення нового елементу довідника необхідно заповнити наступні реквізити:

  • «Код» - заповнюється автоматично;
  • «Найменування» - внутрішнє найменування виду;
  • «Рік» - використовується для визначення року партії врожаю.
