Призначена для вибору формату вкладень при відправленні друкованих форм електронною поштою.

Викликається по кнопці Відправити поштою.

  • Вкажіть за допомогою прапорців, в якому форматі (або декількох форматах) потрібно відправити документ.
  • Для зменшення розміру вкладень друковану форму перед відправленням рекомендується упакувати в архів (формат .zip).
  • При необхідності увімкніть прапорець Транслітерувати імена файлів. У цьому випадку збережені файли матимуть імена, що містять тільки латинські букви і цифри, для можливості перенесення між різними операційними системами. Наприклад, файл Звіт.mxl буде збережений з ім'ям Zvit.mxl.
  • Натисніть кнопку Вибрати. Настройки будуть збережені.
