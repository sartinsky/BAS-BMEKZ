Призначена для введення або зміни пароля.

В цілях інформаційної безпеки рекомендується вводити паролі, не залишати це поле порожнім.

Введення пароля

  • При зміні пароля для забезпечення безпеки введіть Старий пароль.
  • Напишіть Новий пароль.
  • В полі Підтвердження напишіть новий пароль повторно.
  • З метою забезпечення безпеки символи паролів, що вводяться, приховуються. Для того щоб бачити введені символи пароля, увімкніть прапорець Показувати новий пароль.
  • Натисніть Встановити пароль.

Автоматичне створення пароля

  • Якщо самостійно створити пароль складно, натисніть кнопку Створити пароль.
    • Програма пропонує новий пароль. Запам'ятайте його або скопіюйте в буфер обміну, потім напишіть або вставте в поле Новий пароль.

Рекомендації по створенню пароля для входу в програму

Добрий пароль повинен:

  • бути малозрозумілим виразом.
  • складатися не менше ніж з восьми символів.
  • включати в себе:
    • заголовні латинські літери
    • прописні латинські літери
    • цифри
    • символи (підкреслення, дужки і т. д.)

Небажано, щоб пароль збігався з ім'ям користувача, повністю складався з цифр, містив зрозумілі слова, містив чергуються групи символів.

  • Приклади добрих паролів:
    "nj7{jhjibq*Gfhjkm,
    F5"njnGhkmNj;t{HI.
  • Приклади невдалих паролів:
    Іванов,
    qwerty,
    12345678,
    123123123.
