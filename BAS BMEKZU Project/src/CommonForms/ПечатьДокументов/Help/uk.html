Призначена для перегляду, редагування і друку сформованих друкованих форм, а також для відправки друкованої форми по електронній пошті і збереження в файл.

Викликається з документів і списків документів програми по команді Друк.

Передбачені наступні можливості:

  • Друк документів - загальна друкована форма для всіх документів;
  • Друк документа зі штампом візуалізації електронного підпису;
  • Друк комплекту;
  • Настроюваний комплект.

Виведення на принтер

  • Натисніть Друк для виведення потрібної кількості копій сформованої друкованої форми на принтер.

Кількість копій

  • Вкажіть кількість Копій. Вибрана кількість копій запам'ятовується. Для комплектів документів кількість копій залежить від настройок комплекту.

Редагування друкованої форми

  • З допомогою кнопки Редагування можна включити режим редагування цієї друкованої форми. У цьому випадку друкована форма буде зайнята і інші користувачі не зможуть її редагувати. Повторним натисканням кнопки можна відключити режим редагування.

Збереження друкованих форм

  • По кнопці Зберегти є можливість зберегти друковані форми в потрібних форматах в папку на комп'ютері або приєднати до документа. Виберіть за допомогою перемикача спосіб збереження документа і потрібні формати для збереження.

Надсилання електронною поштою

Сформовані друковані форми документа можна Відправити в якості вкладень в електронному листі.

  • Перед відправкою необхідно вказати за допомогою прапорців, в якому форматі (або декількох форматах) потрібно відправити документ.
  • Для зменшення розміру вкладень друковану форму перед відправкою рекомендується упакувати в архів (формат .zip). Натисніть кнопку Вибрати. Настройки будуть збережені.
  • Заповніть поля повідомлення Кому, Тема, напишіть зміст листа. Натисніть кнопку Надіслати.
  • При відправці комплекту документів для кожної друкованої форми буде сформовано окреме вкладення. Якщо друкується комплект друкованих форм відразу для декількох документів зі списку, то рекомендується розбивати документи за контрагентами. У цьому випадку зручно відправляти весь комплект одному контрагенту, наприклад, якщо в комплект входить дві друковані форми, то в першому файлі будуть всі рахунки, у другому всі накладні.

Перехід до потрібного документу

  • Зі списку документів можна вибрати кілька документів для друку.
  • У цьому випадку буде виведено кілька друкованих форм. Для швидкого переходу до потрібної друкованої форми натисніть Перейти до документа (клавішу F7).
  • Із запропонованого списку виберіть потрібний документ, натисніть ОК.

Редагування макета

  • Натисніть Ще - Змінити макет, для того щоб перейти до редагування макета друкованої форми. Друкована форма стає неактивною. Після завершення редагування друкована форма зміниться автоматично.

Перехід до списку макетів друкованих форм

  • Виконайте команду меню Ще - Перейти до макетів друкованих форм, для того щоб відкрити список макетів.

Друк документа зі штампом візуалізації електронного підпису

Для додавання штампа візуалізації електронного підпису в друковану форму обраного документа:

  • У списку документів виберіть документ і натисніть кнопку Друк.
  • За допомогою кнопки Зберегти приєднайте друковану форму до документа, вибравши формат збереження - Табличний(.mxl).
  • У командній панелі документа перейдіть по гіперпосиланню Приєднані файли, виберіть доданий табличний документ і підпишіть його електронним підписом.
    • у контекстному меню виберіть Електронний підпис та шифрування - Підписати, в полі Сертифікат виберіть потрібний сертифікат електронного підпису, введіть пароль, натисніть Підписати.
    • у веб-клієнті на запит розширення для роботи з криптографією натисніть Так;
  • По завершенні підпису, залишаючись на цьому ж табличному документі, натисніть Друк - Зі штампом електронного підпису.

Друк комплекту

  • Програма дозволяє роздрукувати комплект різних документів. Друковані форми комплекту виводяться на окремих сторінках, при цьому можна переглянути конкретну форму, вибираючи з списку друкованих форм зліва потрібну. Можна змінити кількість Копій комплекту.

Настроюваний комплект

У програмі передбачена можливість вибіркового друку комплекту документів.

  • У вікні Комплект, що настроюється зліва знаходиться панель настройки кількості копій, яка складається з стовпців:
    • - в цьому стовпці за допомогою прапорців проводиться вибір для друку,
    • Друкована форма;
    • кількість Копій.
  • Для того щоб настроїти друк комплекту, натисніть Ще - Настройка комплекту. З допомогою команд контекстного меню настройки можна Встановити або Зняти всі прапорці, поміняти порядок друкованих форм в комплекті (Перемістити вгору/вниз) або Скинути налаштування комплекту.
  • Виберіть потрібну друковану форму з комплекту для перегляду. Увімкніть прапорець для друку цієї друкованої форми. Проставте необхідну кількість копій для кожної друкованої форми. Настройки будуть збережені.
  • Крім цього, можна змінити кількість Копій комплекту.
  • Для зручності перегляду друкованої форми можна відключити панель настройки кількості копій з допомогою кнопки Показати/Приховати настройку кількості копій.

Показники виділених комірок

  • За допомогою кнопки можна підрахувати суму, середнє, мінімум і максимум. Вказується також загальна кількість виділених комірок і кількість числових комірок. Розрахунки проводяться тільки для числових комірок.
  • Виділіть потрібні комірки з числами.
  • Натисніть , для того щоб підрахувати Показники виділених комірок.
