Настроюються для механізмів, які працюють з веб-сервісами.

При необхідності вкажіть адреси проксі-серверів для окремих протоколів та адреси інтернет-серверів, для яких повинен використовуватися прямий доступ.

По умовчанню увімкнуто прапорець Використовувати проксі-сервер по умовчанню для всіх протоколів.

Основні параметри настройки (Адреса сервера і Порт) по умовчанню будуть також використані в якості додаткових параметрів.

Настройка додаткових параметрів

  • Для того щоб вказати адреси проксі-серверів для окремих протоколів, вимкніть прапорець Один проксі-сервер для всіх протоколів;
  • У відповідному полі - HTTP, HTTPS, FTP - відкоригуйте ім'я проксі-сервера, який буде використовуватися для кожного протоколу;
  • При необхідності вкажіть Порт.
  • У списку Не використовувати проксі-сервер для адрес за допомогою кнопки Додати вкажіть адреси, до яких буде здійснюватися прямий доступ.
  • Для підтвердження введених параметрів натисніть ОК.
