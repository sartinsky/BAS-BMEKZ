Призначена для настройки параметрів робочого каталогу.

Робочий каталог - папка на Вашому персональному комп'ютері, в якій тимчасово зберігаються файли, отримані з програми для перегляду або редагування.

Відкривається за посиланням Настройка робочого каталогу в персональних настройках програми (закладка Робота з файлами).

Розташування робочого каталогу

  • У полі Розташування вказується шлях до основного робочого каталогу на Вашому персональному комп'ютері.
  • Встановлюється програмою автоматично, але може бути змінений при необхідності. Для цього Виберіть іншу папку на комп'ютері. При виборі іншого шляху до основного робочого каталогу треба вибирати порожній каталог (не містить файлів та підкаталогів).
  • Для того щоб повернутися до початкових настройок програми, натисніть кнопку Відновити значення по умовчанню.

Максимальний розмір

  • У полі Максимальний розмір (Мб) встановіть межу сумарного обсягу всіх файлів в основному робочому каталозі (по умовчанню100 Мб). Якщо встановлений рівним 0, то обсяг файлів не обмежений. Розмір вказується в мегабайтах (Мб).

Запит програми на вилучення файлів

  • Якщо встановлений прапорець Підтверджувати при вилученні з робочого каталогу, то перед вилученням кожного файлу буде видаватися запит на вилучення файлу. По умовчанню прапорець знятий.

Вилучення файлів після редагування

  • Якщо встановлено прапорець Вилучати файли при завершенні редагування, то після перенесення в програму файл вилучається на вашому персональному комп'ютері. Використовуйте цю настройку для економії місця на жорсткому диску або в цілях підвищення безпеки. По умовчанню прапорець знятий.

Поточний стан

  • Програма показує Поточний стан основного робочого каталогу - кількість Файлів, які містяться в ньому і їх Загальний обсяг (Мб).

Перегляд списку файлів в основному робочому каталозі

  • Натисніть Відкрити список файлів, для того щоб переглянути список файлів, які зараз розміщені в основному робочому каталозі.

Очищення основного робочого каталогу

  • Натисніть Очистити робочий каталог, для того щоб вилучити всі файли в основному робочому каталозі, крім зайнятих Вами для редагування. Для підтвердження очищення натисніть Так.
