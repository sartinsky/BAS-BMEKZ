Призначена для переходу до близьких за змістом варіантів звітів.

Відкривається по команді Інші звіти з підменю Варіанти звіту.

У списку у вигляді посилань виводяться близькі за змістом варіанти звітів, якщо вони є в програмі.

Відкривається список інших варіантів звітів даного розділу. Для переходу до потрібного варіанту звіту, натисніть на відповідне посилання.

У нижній частині вікна по умовчанню включений прапорець Закривати це вікно після переходу до звіту, для того щоб вікно не заважало роботі зі звітом.
