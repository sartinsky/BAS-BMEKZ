План видів розрахунку «Середній заробіток»

Містить список нарахувань, які використовуються при розрахунку нарахувань по середньому працівників організації. 

Даний план видів розрахунку є розширюваним, користувач не може додавати свої види розрахунків.

Складові частини заробітку, по-різному враховуються при обчисленні середнього:

- По заробітку для розрахунку лікарняного - використовується тільки в розрахунку середньої для лікарняних;

- За річної премії;  за основним заробітком; по преміях, нарахованих у фіксованих сумах; по преміях нарахованих пропорційно відпрацьованому часу - використовуються для розрахунку відпускних і розрахунку за середнім (відрядження і т.п.).
