План видов расчета «Средний заработок»

Содержит список начислений, используемых при расчете начислений по среднему работников организации. 

Данный план видов расчета является нерасширяемым, пользователь не может добавлять свои виды расчетов.

Составные части заработка, по-разному учитываемые при исчислении среднего:

- По заработку для расчета больничного - используется только в расчета средней для больничных;

- По годовой премии;  по основному заработку; по премиям, начисленным в фиксированных суммах; по премиям начисленным пропорционально отработанному времени - используются для расчета отпускных и расчета по среднему (командировки и т.п.).
